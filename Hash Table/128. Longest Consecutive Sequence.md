# **LeetCode Longest Consecutive Sequence**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires finding the length of the longest consecutive elements sequence in an unsorted array of integers. We want to identify sequences where each number is exactly one greater than the previous number. The goal is to efficiently determine the longest such sequence in terms of length. A naive approach would involve sorting the array, leading to a potentially suboptimal solution, especially with large datasets.

### **🚀 Approach**  
To achieve the optimal solution, a HashSet can be utilized to store unique values from the array. This approach allows for O(1) average time complexity checks for the presence of numbers. The main steps of the algorithm are:
1. Insert all numbers from the array into a HashSet for quick access.
2. For each number in the array, if it is the beginning of a consecutive sequence (i.e., it's not preceded by another number), count how long the sequence goes by checking consecutive numbers until we reach the end of that sequence.
3. Track the maximum length found during these checks.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) - We traverse the array and perform constant-time operations for each element.  
- **Space Complexity**: O(n) - Storage for the HashSet.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        unordered_set<int> numSet(nums.begin(), nums.end());
        int longestStreak = 0;

        for (int num : numSet) {
            // Only try to build a sequence if num is the start of a sequence
            if (numSet.find(num - 1) == numSet.end()) {
                int currentNum = num;
                int currentStreak = 1;

                while (numSet.find(currentNum + 1) != numSet.end()) {
                    currentNum++;
                    currentStreak++;
                }
                longestStreak = max(longestStreak, currentStreak);
            }
        }
        return longestStreak;
    }
};  
```

        int longestStreak = 0;

        for (int num : numSet) {
            if (numSet.find(num - 1) == numSet.end()) {
