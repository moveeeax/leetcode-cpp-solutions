# **LeetCode Minimum Window Substring**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to find the smallest substring within a given string `s` that contains all the characters of another string `t`. We need to ensure that we return the minimum length substring that meets this criteria. The challenge lies in effectively managing the character counts and the window size while scanning through the string.

### **🚀 Approach**  
1. **Sliding Window Technique**: Use two pointers (start and end) to create a window in the string `s` that can expand and contract. The left pointer (`start`) indicates the beginning of the current window, and the right pointer (`end`) expands the window by moving to the right.
  
2. **Character Count Management**: We'll maintain a hashmap (or array) to count the occurrences of characters in `t` and a separate count for the current window in `s`. 

3. **Checking Valid Window**: When the character counts in the current window fulfill the requirements of `t`, we check if the current window is smaller than any previously recorded valid window. 

4. **Contracting the Window**: If a valid window is found, we try to reduce it by moving the `start` pointer to the right, while ensuring the condition (all characters in `t` are still present) is still satisfied.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the length of string `s`. Each character in the string is processed a constant number of times.
- **Space Complexity**: O(m), where m is the size of character set in `t`. We use a hashmap to count characters.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    string minWindow(string s, string t) {
        if(s.length() < t.length()) return "";

        unordered_map<char, int> charCountT, charCountWindow;
        for(char c : t) charCountT[c]++;
        
        int start = 0, end = 0, minLen = INT_MAX, minStart = 0, required = t.length(), formed = 0;
        
        while(end < s.length()) {
            char cEnd = s[end];
            charCountWindow[cEnd]++;
            
            if(charCountT.count(cEnd) && charCountWindow[cEnd] <= charCountT[cEnd]) {
                formed++;
            }
            
            while(start <= end && formed == required) {
                if(end - start + 1 < minLen) {
                    minLen = end - start + 1;
                    minStart = start;
                }
                
                char cStart = s[start];
                charCountWindow[cStart]--;
                if(charCountT.count(cStart) && charCountWindow[cStart] < charCountT[cStart]) {
                    formed--;
                }
                start++;
            }
            end++;
        }
        
        return minLen == INT_MAX ? "" : s.substr(minStart, minLen);
    }
};  
```
