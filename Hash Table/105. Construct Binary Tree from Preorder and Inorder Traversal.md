# **Construct Binary Tree from Preorder and Inorder Traversal**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to construct a binary tree given the preorder and inorder traversal lists. Understanding the properties of these traversals is key to solving the problem.  
- **Preorder Traversal**: In this traversal, the root node is processed before its children. The first element in the preorder traversal list is always the root of the binary tree.
- **Inorder Traversal**: In this traversal, the nodes are processed in the left-root-right order. By locating the root in the inorder list, we can easily determine the left and right subtrees.

### **🚀 Approach**  
1. **Identify the Root**: The first element of the preorder list is the root of the current subtree.
2. **Locate the Root in Inorder**: Find the root in the inorder list to determine the elements that belong to the left and right subtrees. 
3. **Recursively Build Subtrees**: Using the indices from the inorder array, recursively build the left and right subtrees using the respective elements from the preorder array. This recursive process will continue until all elements are placed correctly in the tree.
4. **Base Case**: If there are no elements left in the subtree (i.e., the start index exceeds the end index), return `nullptr`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the binary tree. Each node is processed once.
- **Space Complexity**: O(n), which includes space for the tree nodes and potentially the space used in the recursive call stack.

```cpp
class ConstructBinaryTree {
public:
    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        unordered_map<int, int> inorder_map;
        for (int i = 0; i < inorder.size(); ++i) {
            inorder_map[inorder[i]] = i; // Create a map of inorder values to their indices
        }
        return buildTreeHelper(preorder, 0, preorder.size() - 1, inorder_map, 0);
    }

private:
    TreeNode* buildTreeHelper(vector<int>& preorder, int preStart, int preEnd, 
                               unordered_map<int, int>& inorder_map, int inStart) {
        if (preStart > preEnd) {
            return nullptr;  // Base case: no elements to construct the tree
        }
        
        int rootVal = preorder[preStart];  // The root value from preorder
        TreeNode* root = new TreeNode(rootVal);
        
        int inRootIndex = inorder_map[rootVal];  // Find the root index in inorder
        int leftSubtreeSize = inRootIndex - inStart;  // Size of left subtree

        // Recursively build left and right subtrees
        root->left = buildTreeHelper(preorder, preStart + 1, preStart + leftSubtreeSize, 
                                      inorder_map, inStart);
        root->right = buildTreeHelper(preorder, preStart + leftSubtreeSize + 1, preEnd, 
                                      inorder_map, inRootIndex + 1);
        return root;
    }
};
```
