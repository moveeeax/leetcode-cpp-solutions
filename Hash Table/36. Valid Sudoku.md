# **LeetCode Valid Sudoku**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to determine if a given 9x9 Sudoku board is valid. A valid Sudoku board adheres to certain constraints where:
- Each row must contain the digits 1-9 without repetition.
- Each column must also contain the digits 1-9 without repetition.
- Each of the nine 3x3 sub-boxes that compose the grid must also contain the digits 1-9 without repetition.

Given this setup, we need to check these conditions while traversing through the board.

### **🚀 Approach**  
To solve the problem, we can use a set data structure to keep track of seen numbers in each row, column, and 3x3 sub-box. We will perform the following steps:
1. Traverse through each cell of the Sudoku board.
2. For each number in the cell (from '1' to '9'), check if it has already been seen:
   - Maintain three sets: one for rows, one for columns, and one for sub-boxes.
   - Convert the indices of the sub-boxes using the formula: `(row / 3) * 3 + (col / 3)`.
3. If we encounter a number that has already been recorded in its respective set, the Sudoku configuration is invalid, and we can return false immediately.
4. If all cells are valid, we will return true at the end.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(1) - Since the board size is fixed at 81 cells, our checks will take constant time.
- **Space Complexity**: O(1) - We use a fixed number of sets for tracking seen numbers (at most 27 sets).

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool isValidSudoku(vector<vector<char>>& board) {
        unordered_set<string> seen;
        for (int i = 0; i < 9; i++) {
            for (int j = 0; j < 9; j++) {
                if (board[i][j] != '.') {
                    char num = board[i][j];
                    string rowKey = "row" + to_string(i) + "-" + num;
                    string colKey = "col" + to_string(j) + "-" + num;
                    string boxKey = "box" + to_string(i / 3) + "-" + to_string(j / 3) + "-" + num;
                    
                    if (seen.count(rowKey) || seen.count(colKey) || seen.count(boxKey)) {
                        return false;
                    }
                    seen.insert(rowKey);
                    seen.insert(colKey);
                    seen.insert(boxKey);
                }
            }
        }
        return true;
    }
};
```

#include <iostream>
#include <vector>
#include <string>
#include <unordered_set>

using namespace std;

class Solution {
public:
    bool isValidSudoku(vector<vector<char>>& board) {
        unordered_set<string> seen;
        for (int i = 0; i < 9; i++) {
            for (int j = 0; j < 9; j++) {
                if (board[i][j] != '.') {
                    char num = board[i][j];
                    string rowKey = "row" + to_string(i) + "-" + num;
                    string colKey = "col" + to_string(j) + "-" + num;
                    string boxKey = "box" + to_string(i / 3) + "-" + to_string(j / 3) + "-" + num;
                    
                    if (seen.count(rowKey) || seen.count(colKey) || seen.count(boxKey)) {
                        return false;
                    }
                    seen.insert(rowKey);
                    seen.insert(colKey);
                    seen.insert(boxKey);
                }
            }
        }
        return true;
    }
};

int main() {
    Solution solution;
    vector<vector<char>> board = {
        {'5', '3', '.', '.', '7', '.', '.', '.', '.'},
        {'6', '.', '.', '1', '9', '5', '.', '.', '.'},
        {'.', '9', '8', '.', '.', '.', '.', '6', '.'},
        {'8', '.', '.', '.', '6', '.', '.', '.', '3'},
        {'4', '.', '.', '8', '.', '3', '.', '.', '1'},
        {'7', '.', '.', '.', '2', '.', '.', '.', '6'},
        {'.', '6', '.', '.', '.', '.', '2', '8', '.'},
        {'.', '.', '.', '4', '1', '9', '.', '.', '5'},
        {'.', '.', '.', '.', '8', '.', '.', '7', '9'}
    };
    
    cout << (solution.isValidSudoku(board) ? "true" : "false") << endl;

    return 0;
}
