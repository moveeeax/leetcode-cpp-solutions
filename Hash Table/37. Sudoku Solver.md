# **Sudoku Solver**  

## **1ÔłŹ‚É£ Problem Understanding**  
### **ūüďĆ Intuition**  
The goal of this problem is to fill an incomplete Sudoku board such that each row, column, and the nine 3x3 sub-boxes contain all of the digits from 1 to 9 without repetition. The board is represented as a 9x9 grid where empty cells are indicated by the character '.'. We need to implement a backtracking algorithm to explore possible placements of numbers in the empty cells and ensure that the Sudoku rules are adhered to.

### **ūüöÄ Approach**  
1. **Backtracking**:
   - Start by scanning the board for the next empty cell (denoted by '.').
   - For each empty cell, try placing numbers from 1 to 9 and check if placing the number violates Sudoku rules (i.e., the number is not already present in the current row, column, or the corresponding 3x3 sub-box).
   - If a number can be placed, recursively attempt to solve the rest of the board.
   - If placing a number leads to a valid complete board, we can return true. If it leads to a conflict, undo (backtrack) and try the next number.
   
2. **Validation**:
   - Create a helper function to check if placing a specific number at a specific (row, column) position is valid according to Sudoku rules.

3. **Base Case**:
   - If all cells are filled correctly, the board is solved.

### **‚ŹĪÔłŹ Complexity Analysis**  
- **Time Complexity**: O(9^(N^2)), where N is the number of empty cells. In practice, it's more efficient due to early stopping.
- **Space Complexity**: O(1), as we are modifying the board in place without using additional data structures that scale with input size.

class Solution {
public:
    void solveSudoku(vector<vector<char>>& board) {
        solve(board);
    }

private:
    bool solve(vector<vector<char>>& board) {
        for (int i = 0; i < 9; ++i) {
            for (int j = 0; j < 9; ++j) {
                if (board[i][j] == '.') {
                    for (char c = '1'; c <= '9'; ++c) {
                        if (isValid(board, i, j, c)) {
                            board[i][j] = c; // place the number
                            if (solve(board)) {
                                return true; // recursively solve 
                            }
                            board[i][j] = '.'; // backtrack
                        }
                    }
                    return false; // no valid number found for this cell
                }
            }
        }
        return true; // the board is solved
    }

    bool isValid(vector<vector<char>>& board, int row, int col, char c) {
        for (int i = 0; i < 9; ++i) {
            if (board[row][i] == c) return false; // check row
            if (board[i][col] == c) return false; // check column
            if (board[3 * (row / 3) + i / 3][3 * (col / 3) + i % 3] == c) return false; // check 3x3 sub-box
        }
        return true;
    }
};
