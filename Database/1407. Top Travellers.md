# **[Top Travellers](https://leetcode.com/problems/top-travellers/description/)**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to identify the top travellers based on the total revenue they generated for a specific time period using several queries. Each traveller has a unique ID, and we need to collect and sum the total revenue generated by each traveller from their respective trips. The final output should return the top K travellers based on the cumulative revenue they generated across all trips.

### **🚀 Approach**  
To solve the problem, we can use the following steps:

1. **Data Collection**: We will iterate through the trips to sum up the revenue for each traveller. A dictionary (or hashmap) will be effective here to maintain the total revenue associated with each traveller ID.
  
2. **Sorting**: Once we have the total revenue per traveller, we will sort these travellers in descending order based on their revenue. In cases where the revenue is the same, sorting will then consider the traveller ID in ascending order.

3. **Selecting Top Travellers**: After sorting, we will extract the top K travellers. If we have fewer than K travellers, we will return them all.

4. **Formatting Output**: The final step entails formatting the output into the required structure as specified in the prompt.

This approach will efficiently aggregate and retrieve the desired data within the constraints.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(N log N) due to the sorting step, where N is the number of unique travellers. 
- **Space Complexity**: O(U) where U is the number of unique traveller IDs stored in our map.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<string>> topTravellers(vector<vector<string>>& trips, int K) {
        map<string, long long> revenueMap;
        
        // Gather revenue for each traveller
        for (const auto& trip : trips) {
            string travellerId = trip[0];
            long long revenue = stoll(trip[2]);
            revenueMap[travellerId] += revenue;
        }
        
        vector<pair<long long, string>> revenueList;
        
        // Create a list of (revenue, travellerId)
        for (const auto& entry : revenueMap) {
            revenueList.emplace_back(entry.second, entry.first);
        }
        
        // Sort the revenue list: by revenue descending, then travellerId ascending
        sort(revenueList.begin(), revenueList.end(), [](const auto& a, const auto& b) {
            if (a.first != b.first) return a.first > b.first; // compare by revenue
            return a.second < b.second; // compare by travellerId
        });
        
        vector<vector<string>> result;
        
        // Collect the top K travellers
        for (int i = 0; i < min(K, (int)revenueList.size()); ++i) {
            result.push_back({revenueList[i].second, to_string(revenueList[i].first)});
        }
        
        return result;
    }
};
```
