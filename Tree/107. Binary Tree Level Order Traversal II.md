# **Binary Tree Level Order Traversal II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The goal of this problem is to traverse a binary tree level by level, but instead of presenting the results in the traditional top-to-bottom manner, we need to return the node values from the bottom level to the top level. This is also commonly known as reverse level order traversal.  

To achieve this, we can utilize a queue to facilitate the level order traversal and then store the results in a stack or a vector that we will subsequently reverse before returning it as our final output.  

### **🚀 Approach**  
1. **Use a Queue**: Implement a queue to facilitate the breadth-first search (BFS) traversal of the tree. We first enqueue the root node and then iteratively dequeue the nodes for processing.
2. **Process Levels**: While traversing, for each level of the tree, gather the node values into a temporary vector. Once the nodes for that level are collected, push this vector to a main result container (which is a vector of vectors).
3. **Reverse the Result**: Once we've traversed the entire tree, we reverse the main result vector to achieve the desired bottom-to-top order.
4. **Return the Result**: Finally, we return the reversed vector as the output.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the binary tree. Each node is processed exactly once during the level order traversal.
- **Space Complexity**: O(n) due to the additional space used to store the result and the queue.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<int>> levelOrderBottom(TreeNode* root) {
        vector<vector<int>> result;
        if (!root) return result;
        
        queue<TreeNode*> q;
        q.push(root);
        
        while (!q.empty()) {
            int levelSize = q.size();
            vector<int> levelNodes;
            for (int i = 0; i < levelSize; i++) {
                TreeNode* node = q.front(); q.pop();
                levelNodes.push_back(node->val);
                if (node->left) q.push(node->left);
                if (node->right) q.push(node->right);
            }
            result.push_back(levelNodes);
        }
        
        reverse(result.begin(), result.end());
        return result;
    }
};  
```

```cpp
#include <iostream>
#include <vector>
#include <queue>
using namespace std;

struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
};

class Solution {
public:
    vector<vector<int>> levelOrderBottom(TreeNode* root) {
        vector<vector<int>> result;
        if (!root) return result;
        
        queue<TreeNode*> q;
        q.push(root);
        
        while (!q.empty()) {
            int levelSize = q.size();
            vector<int> levelNodes;
            for (int i = 0; i < levelSize; i++) {
                TreeNode* node = q.front(); q.pop();
                levelNodes.push_back(node->val);
                if (node->left) q.push(node->left);
                if (node->right) q.push(node->right);
            }
            result.push_back(levelNodes);
        }
        
        reverse(result.begin(), result.end());
        return result;
    }
};

int main() {
    Solution sol;
    TreeNode* root = new TreeNode(3);
    root->left = new TreeNode(9);
    root->right = new TreeNode(20);
    root->right->left = new TreeNode(15);
    root->right->right = new TreeNode(7);

    vector<vector<int>> result = sol.levelOrderBottom(root);
    for (const vector<int>& level : result) {
        for (int val : level) {
            cout << val << " ";
        }
        cout << endl;
    }

    // Clean up memory
    delete root->right->right;
    delete root->right->left;
    delete root->right;
    delete root->left;
    delete root;

    return 0;
}
```
