# **Construct Binary Tree from Inorder and Postorder Traversal**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to construct a binary tree given its inorder and postorder traversal sequences. Understand that in these sequences:
- **Inorder Traversal**: This represents the order of nodes where the left child is visited first, then the node, and finally the right child.
- **Postorder Traversal**: This represents the order where the left child is visited first, then the right child, and finally the node itself.

Given these two orders, we can use the properties of binary trees to reconstruct the original tree by identifying which node appears where in the traversals.

### **🚀 Approach**  
1. **Identify the root**: The last element in postorder traversal is always the root of the tree.
2. **Find the root position in inorder**: Using the root identified from postorder, find its index in the inorder array which will help in splitting the inorder sequence into left and right subtrees.
3. **Recursion**: 
   - The elements to the left of the identified root index in inorder belong to the left subtree.
   - The elements to the right belong to the right subtree.
   - Recursion continues until all nodes are processed.

By leveraging these properties and using a recursive approach, we can build the binary tree.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes because we are effectively visiting each node once.
- **Space Complexity**: O(n), for the recursion stack and storage of the tree nodes.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    TreeNode* buildTree(vector<int>& inorder, vector<int>& postorder) {
        if (inorder.empty() || postorder.empty()) return nullptr;
        unordered_map<int, int> inorderMap;
        for (int i = 0; i < inorder.size(); ++i) {
            inorderMap[inorder[i]] = i;
        }
        return helper(postorder, 0, postorder.size() - 1, inorderMap, 0, inorder.size() - 1);
    }
    
    TreeNode* helper(vector<int>& postorder, int postStart, int postEnd, unordered_map<int, int>& inorderMap, int inStart, int inEnd) {
        if (postStart > postEnd || inStart > inEnd) return nullptr;
        
        int rootValue = postorder[postEnd];
        TreeNode* root = new TreeNode(rootValue);
        int inRootIndex = inorderMap[rootValue];
        int leftTreeSize = inRootIndex - inStart;

        root->left = helper(postorder, postStart, postStart + leftTreeSize - 1, inorderMap, inStart, inRootIndex - 1);
        root->right = helper(postorder, postStart + leftTreeSize, postEnd - 1, inorderMap, inRootIndex + 1, inEnd);

        return root;
    }
};
```
