# **Convert Sorted List to Binary Search Tree**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires transforming a singly linked list that is sorted into a height-balanced binary search tree (BST). A height-balanced BST is defined as a tree in which the depth of the two subtrees of every node never differs by more than one. The linked list is given in such a way that if we could manipulate it (like converting it to an array), we could use the properties of binary search to efficiently find a middle element to be the root of the tree.

### **🚀 Approach**  
1. **Finding the Middle Node**: To efficiently build the BST, we can use the "slow and fast pointer" approach to find the middle element of the linked list. The slow pointer moves one step at a time, while the fast pointer moves two steps at a time. When the fast pointer reaches the end of the list, the slow pointer will be at the median element, making it the root of the BST.
  
2. **Recursion**: Once we have the middle node:
   - The middle node becomes the root of the BST.
   - The nodes before the middle node will form the left subtree.
   - The nodes after the middle node will form the right subtree.
   - We will recursively repeat this process for left and right subtrees until we exhaust the linked list.

3. **Base Case**: When the left pointer surpasses the right pointer in our recursive method, we have no nodes to add, hence return null.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the linked list. Each node is processed exactly once.
- **Space Complexity**: O(log n) for the recursion stack in the average-case scenario for a balanced tree. In the worst-case scenario (unbalanced), it could go up to O(n).

## **2️⃣ LeetCode Submission**  
```cpp
struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(nullptr) {}
};

struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    TreeNode* sortedListToBST(ListNode* head) {
        if (!head) return nullptr;

        ListNode* mid = findMiddle(head);
        TreeNode* root = new TreeNode(mid->val);
        
        if (head == mid) return root; // base case when there's only one node

        root->left = sortedListToBST(head); // build left subtree
        root->right = sortedListToBST(mid->next); // build right subtree
        
        return root;
    }

private:
    ListNode* findMiddle(ListNode* head) {
        ListNode* prev = nullptr;
        ListNode* slow = head;
        ListNode* fast = head;

        while (fast && fast->next) {
            prev = slow;
            slow = slow->next;
            fast = fast->next->next;
        }

        if (prev) prev->next = nullptr; // detach left half from mid

        return slow; // the middle node
    }
};
```
