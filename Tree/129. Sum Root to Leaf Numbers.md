# **Sum Root to Leaf Numbers**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to find the total sum of all numbers formed by root-to-leaf paths in a binary tree. Each path represents a number where the digits are formed by traversing the tree from the root to the leaves. Specifically, if we go from the root with value `1` to a left child `2` and then to a left child `3`, it represents the number `123`. The goal is to sum all such formed numbers for all paths in the tree.

### **🚀 Approach**  
To solve the problem, we can use a depth-first search (DFS) approach with backtracking. We will traverse the tree recursively, maintaining the current sum as we go deeper into the tree. At each node, we will build the number by multiplying the current sum by 10 (to shift the previous digits left) and adding the node's value. When we reach a leaf node (a node without children), we add the current sum to a total sum. After exploring a node, we backtrack to continue exploring other paths.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(N), where N is the number of nodes in the binary tree, because we visit each node exactly once.  
- **Space Complexity**: O(H), where H is the height of the tree, because of the recursion stack in the worst-case scenario (which occurs in the case of a skewed tree).

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int sumNumbers(TreeNode* root) {
        return dfs(root, 0);
    }

    int dfs(TreeNode* node, int currentSum) {
        if (!node) return 0;

        currentSum = currentSum * 10 + node->val;

        // If it's a leaf node, return the current sum
        if (!node->left && !node->right) {
            return currentSum;
        }

        // Recursively sum up the values from left and right subtrees
        return dfs(node->left, currentSum) + dfs(node->right, currentSum);
    }
};
```

```cpp
#include <iostream>

struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    int sumNumbers(TreeNode* root) {
        return dfs(root, 0);
    }

    int dfs(TreeNode* node, int currentSum) {
        if (!node) return 0;

        currentSum = currentSum * 10 + node->val;

        if (!node->left && !node->right) {
            return currentSum;
        }

        return dfs(node->left, currentSum) + dfs(node->right, currentSum);
    }
};

int main() {
    // Create a sample tree: 
    //      1
    //     / \
    //    2   3
    TreeNode* root = new TreeNode(1);
    root->left = new TreeNode(2);
    root->right = new TreeNode(3);

    Solution solution;
    int result = solution.sumNumbers(root);
    
    std::cout << "Sum of root to leaf numbers: " << result << std::endl; // Expected output: 25

    // Clean up memory
    delete root->left;
    delete root->right;
    delete root;

    return 0;
}
```
