# **Valid Palindrome**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to determine if a given string is a palindrome, meaning it reads the same forwards as backward, considering only alphanumeric characters and ignoring cases. A straightforward approach would be to remove all non-alphanumeric characters from the string and then check if the modified string is equal to its reverse.

### **🚀 Approach**  
1. **Filter Characters**: Iterate through the string and collect only the alphanumeric characters, converting them to lowercase.
2. **Check Palindrome**: Compare the filtered string to its reverse for equality.
3. **Return Result**: If they are the same, return `true`, indicating the string is a palindrome; otherwise, return `false`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the length of the input string, since we traverse the string a couple of times (once for filtering and once for checking).
- **Space Complexity**: O(n), for storing the filtered characters.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool isPalindrome(string s) {
        string filtered;
        for (char c : s) {
            if (isalnum(c)) {
                filtered += tolower(c);
            }
        }
        string reversed = string(filtered.rbegin(), filtered.rend());
        return filtered == reversed;
    }
};
```

            if (isalnum(c)) {
                filtered += tolower(c);
            }
        }
        string reversed = string(filtered.rbegin(), filtered.rend());
        return filtered == reversed;
