# **Find the Index of the First Occurrence in a String**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires finding the index of the first occurrence of a given substring (`needle`) within a larger string (`haystack`). If the substring is not found, we need to return -1. The most straightforward approach could be a brute-force solution, which would involve checking each possible starting index in `haystack` to see if the substring matches. However, this would be inefficient for large strings.

### **🚀 Approach**  
A more efficient approach is to utilize the built-in string function in C++, which can find substrings. We can use the `find()` method of the `std::string` class, which is implemented under the hood for optimized substring search. This function takes the starting position and returns the index of the first occurrence of the substring, or `std::string::npos` if it is not found.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the length of the haystack. The `find()` function performs the substring search in linear time on average.
- **Space Complexity**: O(1), since we are using a constant amount of space.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int strStr(string haystack, string needle) {
        return haystack.find(needle);
    }
};
```

#include <iostream>

    int strStr(std::string haystack, std::string needle) {

    }

    Solution solution;

    needle = "sad";
    
    int index = solution.strStr(haystack, needle);
    std::cout << index << std::endl; // Expected output: 0

    // Additional test case
    haystack = "leetcode";
    needle = "leeto";

    return 0;
}
