# **Remove Duplicates from Sorted Array II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to manage duplicates in a sorted array. Specifically, we are allowed to keep at most two occurrences of the same number while removing additional duplicates. The sorted nature of the array simplifies our task because all duplicates are clustered together. A key insight is to maintain a pointer that tracks the position into which to copy the valid numbers (up to two occurrences).

### **🚀 Approach**  
1. **Pointer Initialization**: Set a pointer `j` to 0, which will track where the next valid element should be placed in the array.  
2. **Iteration**: Use a loop to iterate through the array with an index `i`.  
3. **Count Occurrences**: Keep a count of occurrences of the current element. If the occurrence of a number reaches more than two, skip adding it to the result.  
4. **Copying Valid Elements**: If the count of an element is less than or equal to two, copy it to the array position tracking by `j` and increment `j`.  
5. **Output the Result**: At the end of the loop, `j` will point to the length of the modified array containing unique elements with allowed duplicates.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of elements in the array, since we're traversing the array once.  
- **Space Complexity**: O(1), since we are using the same array for output without needing additional storage that scales with input size.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        if (nums.empty()) return 0;
        
        int j = 0; // Pointer for the place to put valid elements
        for (int i = 0; i < nums.size(); i++) {
            // Allow at most two occurrences
            if (j < 2 || nums[i] != nums[j - 2]) {
                nums[j++] = nums[i]; // Place the element in the output position
            }
        }
        return j; // j is the new length of the modified array
    }
};
```

#include <iostream>
#include <vector>
using namespace std;

class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        if (nums.empty()) return 0;
        
        int j = 0; // Pointer for the place to put valid elements
        for (int i = 0; i < nums.size(); i++) {
            // Allow at most two occurrences
            if (j < 2 || nums[i] != nums[j - 2]) {
                nums[j++] = nums[i]; // Place the element in the output position
            }
        }
        return j; // j is the new length of the modified array
    }
};

int main() {
    Solution sol;
    vector<int> nums = {1, 1, 1, 2, 2, 3};
    int newLength = sol.removeDuplicates(nums);
    
    cout << "New length: " << newLength << endl;
    cout << "Modified array: ";
    for (int i = 0; i < newLength; i++) {
        cout << nums[i] << " ";
    }
    cout << endl;
    return 0;
}
