# **Partition List**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to rearrange a linked list based on a given value `x`. The goal is to partition the list so that all nodes with values less than `x` come before nodes with values greater than or equal to `x`. The original relative order of the nodes should be preserved. For example, given the linked list `1 -> 4 -> 3 -> 2 -> 5 -> 2` and `x = 3`, the output should be `1 -> 2 -> 2 -> 4 -> 3 -> 5`.  

To solve this problem, we can utilize two separate lists: one for nodes less than `x` and another for nodes that are greater than or equal to `x`. After traversing the list and distributing the nodes into these two lists, we will connect them at the end.  

### **🚀 Approach**  
1. **Define Two Dummy Nodes**: We will create two dummy nodes, `less_head` for nodes less than `x` and `greater_head` for nodes greater than or equal to `x`.

2. **Initialize Pointers**: Two pointers, `less` and `greater`, will be initialized to point at `less_head` and `greater_head`, respectively. This will help us build both lists.

3. **Iterate Through the Original List**: We will traverse the original linked list, and for each node:
   - If the node's value is less than `x`, we will append it to the `less` list.
   - Otherwise, we will append it to the `greater` list.

4. **Connect the Two Lists**: After the traversal, we will connect the end of the `less` list to the beginning of the `greater` list.

5. **Terminate the List**: It’s crucial to ensure that the last node of the `greater` list points to `nullptr` to signify the end of the list.

6. **Return the New List**: Finally, we will return the head of the new partitioned list, which is the next node of `less_head`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where `n` is the number of nodes in the list since we are traversing the list only once.  
- **Space Complexity**: O(1), since we are not using any extra space that grows with the input size, just a few pointers.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    ListNode* partition(ListNode* head, int x) {
        ListNode less_head(0), greater_head(0);
        ListNode *less = &less_head, *greater = &greater_head;

        while (head) {
            if (head->val < x) {
                less->next = head;
                less = less->next;
            } else {
                greater->next = head;
                greater = greater->next;
            }
            head = head->next;
        }

        greater->next = nullptr; // Ensure the last node points to nullptr
        less->next = greater_head.next; // Connect the two lists

        return less_head.next; // Return the merged list starting from the next of dummy node
    }
};
```

```cpp
#include <iostream>

struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(nullptr) {}
};

class Solution {
public:
    ListNode* partition(ListNode* head, int x) {
        ListNode less_head(0), greater_head(0);
        ListNode *less = &less_head, *greater = &greater_head;

        while (head) {
            if (head->val < x) {
                less->next = head;
                less = less->next;
            } else {
                greater->next = head;
                greater = greater->next;
            }
            head = head->next;
        }

        greater->next = nullptr; // Ensure the last node points to nullptr
        less->next = greater_head.next; // Connect the two lists

        return less_head.next; // Return the merged list starting from the next of dummy node
    }
};

int main() {
    // Create example linked list: 1 -> 4 -> 3 -> 2 -> 5 -> 2
    ListNode* head = new ListNode(1);
    head->next = new ListNode(4);
    head->next->next = new ListNode(3);
    head->next->next->next = new ListNode(2);
    head->next->next->next->next = new ListNode(5);
    head->next->next->next->next->next = new ListNode(2);

    int x = 3;
    Solution solution;
    ListNode* new_head = solution.partition(head, x);

    // Output the result
    while (new_head) {
        std::cout << new_head->val << " ";
        new_head = new_head->next;
    }
    return 0;
}
```
