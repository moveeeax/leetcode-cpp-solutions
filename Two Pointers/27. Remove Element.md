# **LeetCode Remove Element**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to modify an array in-place to remove all instances of a specified value. The key objective is to not use extra space for another copy of the array, making the solution more memory efficient. The elements that are not part of the specified value should remain in the array, and the new length of this modified array should be returned.

### **🚀 Approach**  
1. **Two-Pointer Technique**: We can maintain two pointers, `i` and `j`. Pointer `j` will iterate through the array while pointer `i` will track the position of the next valid element (i.e., not equal to the given `val`).
2. **Iteration**:
   - If `nums[j]` is not equal to `val`, we will assign `nums[j]` to `nums[i]` and increment `i`.
   - This way, all elements that are not equal to `val` will be shifted to the front of the array, and the size of valid elements will be tracked by `i` at the end of the iteration.
3. **Return Value**: Finally, return `i` which represents the new length of the array after removing the specified value.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of elements in the array. We check each element once.
- **Space Complexity**: O(1), as we are only using a constant amount of extra space regardless of the input size.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int i = 0; // Pointer to place next valid element
        for (int j = 0; j < nums.size(); j++) {
            if (nums[j] != val) {
                nums[i] = nums[j]; // Shift valid element to the front
                i++; // Increment the count of valid elements
            }
        }
        return i; // Return new length of the array
    }
};
```

```cpp
#include <iostream>
#include <vector>

class Solution {
public:
    int removeElement(std::vector<int>& nums, int val) {
        int i = 0; // Pointer to place next valid element
        for (int j = 0; j < nums.size(); j++) {
            if (nums[j] != val) {
                nums[i] = nums[j]; // Shift valid element to the front
                i++; // Increment the count of valid elements
            }
        }
        return i; // Return new length of the array
    }
};

int main() {
    Solution solution;
    std::vector<int> nums = {3, 2, 2, 3};
    int val = 3;
    int new_length = solution.removeElement(nums, val);

    std::cout << "New length: " << new_length << std::endl;
    std::cout << "Modified array: ";
    for (int i = 0; i < new_length; i++) {
        std::cout << nums[i] << " ";
    }
    std::cout << std::endl;

    return 0;
}
```
