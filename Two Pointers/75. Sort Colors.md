# **LeetCode Sort Colors**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to sort an array containing only three distinct integers, 0, 1, and 2, which represent three colors (Red, White, and Blue). The goal is to arrange these colors in the correct order, ensuring that the algorithm runs in linear time without using any additional space.

To achieve this, we can utilize the **Dutch National Flag problem** algorithm, which allows us to maintain a three-way partition of the array. We'll use three pointers:
- One pointer for the "0s" (Red).
- One pointer for the "1s" (White).
- One pointer for the "2s" (Blue).

### **🚀 Approach**  
1. **Initialization**: Start with three pointers:
   - `low` to track the position of the next "0".
   - `mid` to track the current element being examined.
   - `high` to track the position of the last "2".
   
2. **Loop through the Array**: Use a while loop to iterate through the array with the `mid` pointer:
   - If `nums[mid]` is 0: Swap it with `nums[low]`, and increment both `low` and `mid`.
   - If `nums[mid]` is 1: Just increment `mid`, as it is already in the correct position.
   - If `nums[mid]` is 2: Swap it with `nums[high]` and decrement `high` (do not increment `mid` since we need to re-evaluate the swapped value).

3. **Termination**: The loop continues until `mid` exceeds `high`, ensuring all elements are sorted.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) as we traverse the list just once.
- **Space Complexity**: O(1) since we are sorting the array in place without using additional storage.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    void sortColors(vector<int>& nums) {
        int low = 0, mid = 0, high = nums.size() - 1;
        
        while (mid <= high) {
            if (nums[mid] == 0) {
                swap(nums[low++], nums[mid++]);
            } else if (nums[mid] == 1) {
                mid++;
            } else {
                swap(nums[mid], nums[high--]);
            }
        }
    }
};
```
