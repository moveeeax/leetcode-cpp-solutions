# **LeetCode Convert Sorted Array to Binary Search Tree**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The task is to convert a sorted array into a height-balanced binary search tree (BST). A height-balanced BST is defined as a binary tree where the depth of the two subtrees of every node never differs by more than one. Given a sorted array, we can achieve this by recursively choosing the middle element of the array as the root node, which will ensure balance among the left and right subtrees.

### **🚀 Approach**  
1. **Recursive Function**: We need a helper function to form the BST from the given array segment. This function will take two parameters: the start and end indices of the array segment.
2. **Base Case**: The base case for the recursion is when the start index exceeds the end index, in which case we need to return `nullptr`, indicating there are no elements to form a node.
3. **Choosing the Middle Element**: The middle element of the current segment of the array will be the root of the BST (or subtree). The index can be calculated as `mid = (start + end) / 2`.
4. **Recursive Calls**: Once the root is established, the function will recursively build the left subtree using the left half of the array and the right subtree using the right half.
5. **Returning the Root**: Repeat this process until all elements are inserted into the tree. 

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of elements in the array, as we touch each node exactly once.
- **Space Complexity**: O(log n) for the recursive stack space in the case of a balanced tree.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    TreeNode* sortedArrayToBST(vector<int>& nums) {
        return constructBST(nums, 0, nums.size() - 1);
    }
    
    TreeNode* constructBST(const vector<int>& nums, int start, int end) {
        if (start > end) return nullptr;
        int mid = (start + end) / 2;
        TreeNode* node = new TreeNode(nums[mid]);
        node->left = constructBST(nums, start, mid - 1);
        node->right = constructBST(nums, mid + 1, end);
        return node;
    }
};
```
