# **Climbing Stairs**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem at hand is about calculating the number of distinct ways to climb to the top of a staircase with `n` steps. At every step, you can either take 1 step or 2 steps. This is a classic dynamic programming problem where the challenge is to determine how many unique combinations of steps can lead to the top.

### **🚀 Approach**  
1. **Dynamic Programming Concept**: We can utilize a bottom-up dynamic programming approach where we iterate from the base cases up to `n`. We will maintain an array `dp` where `dp[i]` represents the number of distinct ways to reach the `i-th` step.
  
2. **Base Cases**: 
   - To reach the 1st step (`dp[1]`), there's only 1 way: a single step.
   - To reach the 2nd step (`dp[2]`), there are 2 ways: either two single steps or one double step.

3. **Recursion Relation**: 
   - For any step `i` greater than 2, the number of distinct ways to get there can be derived from the previous two steps: 
     - `dp[i] = dp[i-1] + dp[i-2]`. 
   This formula works because to reach step `i`, you could have either come from step `i-1` or step `i-2`.

4. **Iterative Calculation**: Loop through from step 3 to `n`, updating our `dp` array using the relation mentioned.

5. **Space Optimization**: Instead of maintaining an entire array, we can reduce space complexity to O(1) by storing only the last two computed values at any step since that suffices to compute the next number of ways.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), since we compute the number of distinct ways for all steps up to `n`.  
- **Space Complexity**: O(1), as we use only a constant space for storing two variables.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int climbStairs(int n) {
        if (n <= 2) {
            return n;
        }
        
        int first = 1, second = 2, current;
        for (int i = 3; i <= n; ++i) {
            current = first + second;
            first = second;
            second = current;
        }
        return second;
    }
};
```
