# **LeetCode Multiply Strings**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to multiply two non-negative integers represented as strings. Our goal is to return the product as a string. The strings avoid using any built-in conversion methods for the multiplication, meaning we must implement the multiplication algorithm from scratch, similar to how traditional multiplication is performed by hand.

### **🚀 Approach**  
1. **Initial Checks**: If either string is "0", we return "0" immediately, since multiplying by zero yields zero.
2. **Prepare the Result Array**: The maximum length of the result of two numbers `num1` and `num2` will be `len(num1) + len(num2)`. We initialize a result array with zeros of this length.
3. **Reverse Iteration**: We process each digit starting from the least significant digit (rightmost) towards the most significant (leftmost), similar to hand multiplication:
   - For each digit in `num1` and `num2`, multiply the digits and add to the corresponding index in the result array.
   - Handle carry-over by adjusting adjacent indices in the result array.
4. **Convert Result to String**: After all the multiplications are done, we convert the result array to a string, skipping leading zeros.
5. **Return the Result**: Finally, we return the formatted result string.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n * m) where n and m are the lengths of `num1` and `num2` respectively, because we effectively perform a multiplication for each digit in both numbers.
- **Space Complexity**: O(n + m) for the result array.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    string multiply(string num1, string num2) {
        if (num1 == "0" || num2 == "0") return "0";

        int len1 = num1.size(), len2 = num2.size();
        vector<int> result(len1 + len2, 0);

        for (int i = len1 - 1; i >= 0; --i) {
            for (int j = len2 - 1; j >= 0; --j) {
                int mul = (num1[i] - '0') * (num2[j] - '0');
                int sum = mul + result[i + j + 1];
                result[i + j + 1] = sum % 10;      // store remainder
                result[i + j] += sum / 10;         // carry
            }
        }

        stringBuilder;
        for (int num : result) {
            if (!(builder.empty() && num == 0)) {
                builder.push_back(num + '0'); // convert back to char
            }
        }
        
        return builder.empty() ? "0" : builder;
    }
};
```
