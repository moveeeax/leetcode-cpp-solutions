# **Gray Code**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to generate the sequence of Gray codes for a given number of bits `n`. A Gray code is a binary numeral system where two successive values differ in only one bit. Understanding how Gray codes are constructed can help us devise an efficient solution. For any integer `n`, the sequence starts from 0 and each subsequent value can be derived from the previous one by flipping a single bit.

### **🚀 Approach**  
1. **Recursive Relation**: The sequence can be generated using a recursive approach. The Gray code for `n` bits can be formed by taking the Gray code for `n-1` bits, prefixing it with `0`, and then appending a mirrored version (in reverse) of the same sequence prefixed with `1`.
   
   - For example:
     - If `n=1`: 
       - Gray code = `[0]` -> `0`
       - Mirror = `[1]` -> `1`
       - Result = `[0, 1]`
     - If `n=2`: 
       - Gray code = `[00, 01]`
       - Mirror = `[11, 10]`
       - Result = `[00, 01, 11, 10]`
   
2. **Bitwise Calculation**: Alternatively, a Gray code can also be computed for a number `i` from `0` to `2^n - 1` using the formula `i ^ (i >> 1)`. This efficiently transforms the binary representation to its Gray code form.

3. **Looping to Generate the Sequence**: We can use a loop from `0` to `2^n`, applying the bitwise formula to generate each Gray code.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(2^n) since we generate `2^n` values for `n` bits.
- **Space Complexity**: O(2^n) for storing the sequence of Gray codes.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<int> grayCode(int n) {
        int total = 1 << n; // 2^n
        vector<int> result;
        for (int i = 0; i < total; ++i) {
            result.push_back(i ^ (i >> 1));
        }
        return result;
    }
};
```

```cpp
#include <iostream>
#include <vector>
using namespace std;

class Solution {
public:
    vector<int> grayCode(int n) {
        int total = 1 << n; // 2^n
        vector<int> result;
        for (int i = 0; i < total; ++i) {
            result.push_back(i ^ (i >> 1));
        }
        return result;
    }
};

int main() {
    Solution solution;
    int n;
    cout << "Enter the number of bits: ";
    cin >> n;
    
    vector<int> grayCodes = solution.grayCode(n);
    cout << "Gray code sequence for " << n << " bits: ";
    for (int code : grayCodes) {
        cout << code << " ";
    }
    cout << endl;

    return 0;
}
```
