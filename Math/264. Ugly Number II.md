# **[Ugly Number II](https://leetcode.com/problems/ugly-number-ii/description/)**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem is asking for the nth ugly number, where an ugly number is defined as a positive number whose prime factors only include 2, 3, and 5. This means we need to generate a sequence of such numbers in increasing order and find the number at the nth position. 

### **🚀 Approach**  
To solve the problem optimally, we can use a dynamic programming approach by utilizing multiple pointers. We will keep track of the next potential ugly numbers generated by multiplying the smallest ugly numbers found so far with 2, 3, and 5. We maintain pointers for the last used ugly numbers for each of these multipliers.

1. **Initialization**: Start with an array to hold the ugly numbers and initialize the first element as 1 (the first ugly number).
2. **Pointers**: Maintain three pointers (for 2, 3, and 5) that will point to the position of the last calculated ugly number that corresponds to each of the prime factors.
3. **Generate Ugly Numbers**:
   - At each step, calculate the next ugly number by taking the minimum of the numbers obtained by multiplying the numbers at the pointers with 2, 3, and 5.
   - Store the new ugly number in the array.
   - Move the pointer(s) from where the next ugly number was derived.
4. **Repeat** until we have generated the first n ugly numbers.

This approach ensures that we efficiently find the nth ugly number without generating all numbers up to that point.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) - We compute the first n ugly numbers in a single pass through the numbers.
- **Space Complexity**: O(n) - We store the ugly numbers in an array of size n.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int nthUglyNumber(int n) {
        vector<int> ugly(n);
        ugly[0] = 1;
        
        int i2 = 0, i3 = 0, i5 = 0;
        int next2 = 2, next3 = 3, next5 = 5;
        
        for (int i = 1; i < n; ++i) {
            int nextUgly = min(next2, min(next3, next5));
            ugly[i] = nextUgly;
            
            if (nextUgly == next2) {
                i2++;
                next2 = ugly[i2] * 2;
            }
            if (nextUgly == next3) {
                i3++;
                next3 = ugly[i3] * 3;
            }
            if (nextUgly == next5) {
                i5++;
                next5 = ugly[i5] * 5;
            }
        }
        
        return ugly[n - 1];
    }
};  
```
