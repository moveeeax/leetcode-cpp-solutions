# **LeetCode Sqrt(X)**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The task is to find the integer square root of a non-negative integer `x`. The integer square root is defined as the largest integer `n` such that `n^2 <= x`. For instance, if `x` is 8, the integer square root is 2 because `2^2 = 4` is less than 8, but `3^2 = 9` exceeds 8. The solution needs to be efficient enough to handle inputs up to `2^31 - 1`, which makes a simple iterative approach impractical due to time constraints.

### **🚀 Approach**  
The optimal approach to solve this problem is to use the **Binary Search** algorithm. By using this technique, we can effectively narrow down our potential candidates for the integer square root:

1. **Initialize** two pointers, `left` at 0 and `right` at `x`.
2. **While** `left` is less than or equal to `right`, calculate the midpoint `mid`.
3. **Check** if `mid^2` equals `x`. If it does, we have found our integer square root.
4. If `mid^2` is less than `x`, move `left` to `mid + 1`.
5. If `mid^2` is greater than `x`, adjust `right` to `mid - 1`.
6. The integer square root will be stored as we track the greatest value of `mid` for which `mid^2` is less than or equal to `x`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(log x) because we are halving the search space in each iteration.
- **Space Complexity**: O(1) since we are only using a few variables for pointers and the result.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int mySqrt(int x) {
        if (x < 2) return x;
        int left = 0, right = x;
        int result = 0;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (mid * mid == x) {
                return mid;
            } else if (mid * mid < x) {
                result = mid;  // Update the result to the largest mid found
                left = mid + 1; // Move right
            } else {
                right = mid - 1; // Move left
            }
        }
        
        return result;
    }
};
```
