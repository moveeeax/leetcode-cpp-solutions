# **LeetCode Divide Two Integers**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
We are tasked with dividing two integers, `dividend` and `divisor`, without using the division operator. The goal is to find the quotient of the two numbers. A significant aspect of this problem is to handle edge cases, particularly when either the `dividend` or the `divisor` is negative, and when the `dividend` equals INT_MIN and the `divisor` equals -1, leading to overflow issues.

### **🚀 Approach**  
1. **Determine Signs**: First, we need to determine whether the result should be negative or positive based on the input signs. This can be achieved by checking if one (and only one) of the numbers is negative.
2. **Use Bit Manipulation**: Instead of performing straightforward division operations, we can use subtraction—or more optimally, bit shifts—to approach the problem. We repeatedly subtract the `divisor` from `dividend` until what's left is less than `divisor`. Each successful subtraction corresponds to a power of two that can be used in the quotient.
3. **Iterative Subtraction/Bit Shifting**: By utilizing bit manipulation (left shifts), we can find the multiples of the `divisor` that fit into the `dividend`, which allows us to build the quotient efficiently.
4. **Handle Edge Cases**: Specifically, we need to manage overflow scenarios and return the correct outputs when limits are reached.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(log n), where n is the absolute value of the `dividend`. This is due to the number of operations being proportional to the number of bits in the dividend.
- **Space Complexity**: O(1), since we are using a constant amount of space for variables.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int divide(int dividend, int divisor) {
        // Handle edge cases immediately
        if (dividend == INT_MIN && divisor == -1) return INT_MAX;
        if (dividend == 0) return 0;
        
        // Determine the sign of the result
        bool isNegative = (dividend < 0) ^ (divisor < 0);
        
        long long absDividend = abs(dividend);
        long long absDivisor = abs(divisor);
        long long quotient = 0;
        
        // Perform division using bit manipulation
        while (absDividend >= absDivisor) {
            long long temp = absDivisor, multiple = 1;
            while (absDividend >= (temp << 1)) {
                temp <<= 1; // Double the divisor
                multiple <<= 1; // Track how many times we doubled
            }
            absDividend -= temp;
            quotient += multiple;
        }
        
        return isNegative ? -quotient : quotient;
    }
};  
```
