# **Flatten Binary Tree to Linked List**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem asks us to take a binary tree and flatten it into a specific linked list format. The desired format dictates that we should rearrange the tree so that each node's left child points to `nullptr`, and the right child points to the next node in a preorder traversal of the tree. Essentially, we want to convert the structure of the binary tree into a linear list represented through right pointers only.

### **🚀 Approach**  
To achieve this, we employ a method called preorder traversal, which visits nodes in the order: root, left, right. We will follow these key steps during our implementation:

1. **Recursive Flattening**: We will create a recursive function that flattens the left and right subtrees.
2. **Rearrangement of Pointers**:
   - After flattening the left and right subtrees, we will rearrange the pointers such that the left subtree comes first followed by the right subtree.
   - Set the left child of the node to `nullptr`.
   - Link the right child of the node to the head of the flattened left subtree.
   - Find the end of the flattened left subtree and link it to the head of the flattened right subtree.

3. **Base Case for Recursion**: If the current node is `nullptr`, we simply return.

This recursive approach allows us to handle each subtree individually, making it easier to manage pointer assignments.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) where n is the number of nodes in the binary tree since we visit each node exactly once.
- **Space Complexity**: O(h) where h is the height of the tree due to the recursion stack in the case of a skewed tree. In the average case, given a balanced tree, it would be O(log n).

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    void flatten(TreeNode* root) {
        if (!root) return;
        
        flatten(root->left);
        flatten(root->right);
        
        TreeNode* tempRight = root->right; // Store the original right subtree
        root->right = root->left; // Move the left subtree to the right
        root->left = nullptr; // Set left child to nullptr
        
        // Move to the end of the new right subtree
        TreeNode* current = root;
        while (current->right) {
            current = current->right;
        }
        
        current->right = tempRight; // Attach the original right subtree
    }
};
```

```cpp
#include <iostream>
using namespace std;

// Definition for a binary tree node.
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    void flatten(TreeNode* root) {
        if (!root) return;

        flatten(root->left);
        flatten(root->right);
        
        TreeNode* tempRight = root->right; // Store the original right subtree
        root->right = root->left; // Move the left subtree to the right
        root->left = nullptr; // Set left child to nullptr
        
        // Move to the end of the new right subtree
        TreeNode* current = root;
        while (current->right) {
            current = current->right;
        }

        current->right = tempRight; // Attach the original right subtree
    }
};

void printFlattenedTree(TreeNode* root) {
    while (root) {
        cout << root->val << " ";
        root = root->right; // Only follow right pointers
    }
}

TreeNode* buildTree() {
    TreeNode* root = new TreeNode(1);
    root->left = new TreeNode(2);
    root->right = new TreeNode(5);
    root->left->left = new TreeNode(3);
    root->left->right = new TreeNode(4);
    root->right->right = new TreeNode(6);
    return root;
}

int main() {
    TreeNode* root = buildTree();
    Solution sol;
    sol.flatten(root);
    printFlattenedTree(root);
    return 0;
}
```
