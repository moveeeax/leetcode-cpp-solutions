# **Longest Valid Parentheses**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to find the length of the longest substring of well-formed parentheses in a given string containing only the characters '(' and ')'. A well-formed parentheses string is one where each opening bracket has a corresponding closing bracket in the correct order. 

We can use a dynamic programming approach to achieve this. We will maintain an array that keeps track of the lengths of valid parentheses ending at each index. When we encounter a closing parenthesis, we check if there is a matching opening parenthesis earlier in the string, and we calculate the length of valid parentheses based on previously calculated lengths.

### **🚀 Approach**  
1. **Initialize an array `dp`:** The array `dp` will store the lengths of the longest valid parentheses ending at each index.
  
2. **Iterate through the string:** Start from the second character (index 1) to the end:
   - If we encounter a closing parenthesis `')'`:
     - Check if the character before it is an opening parenthesis `'('`. If yes, then `dp[i] = dp[i-2] + 2`.
     - If the character before it is a closing parenthesis:
       - Check if there is a corresponding opening parenthesis two positions back. Compute the total length of valid parentheses by considering both valid lengths before this pair and the valid parentheses that follow.

3. **Maintain the maximum length found:** Every time we update the `dp` array, we keep track of the overall maximum length.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the length of the input string, as we scan through the string once.
- **Space Complexity**: O(n), due to the additional `dp` array used for storing lengths.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int longestValidParentheses(string s) {
        int maxLength = 0;
        vector<int> dp(s.size(), 0);
        
        for (int i = 1; i < s.size(); i++) {
            if (s[i] == ')') {
                // Check for valid parentheses ending with ')'
                if (s[i-1] == '(') {
                    dp[i] = (i >= 2) ? dp[i-2] + 2 : 2;
                } else if (i - dp[i-1] > 0 && s[i - dp[i-1] - 1] == '(') {
                    dp[i] = dp[i-1] + ((i - dp[i-1] >= 2) ? dp[i - dp[i-1] - 2] : 0) + 2;
                }
                maxLength = max(maxLength, dp[i]);
            }
        }
        return maxLength;
    }
};
```
