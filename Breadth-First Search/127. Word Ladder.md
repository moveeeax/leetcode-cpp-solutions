# **Word Ladder**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The "Word Ladder" problem requires transforming a given start word into an end word through a series of intermediate words, where each transformation changes exactly one letter. The catch is that every intermediate word must exist in a given dictionary. The goal is to find the shortest transformation sequence from the start word to the end word, if such a sequence exists.

### **🚀 Approach**  
To solve this problem, we can use a breadth-first search (BFS) approach because it naturally finds the shortest path in an unweighted graph. Here’s a breakdown of the steps:

1. **Initialization**: Start by creating a queue that will hold pairs of the current word and the number of steps taken to reach it.

2. **Set Up for BFS**: Use a set to store the dictionary for O(1) lookup time. Initialize the starting word and check if it’s equal to the end word.

3. **BFS Procedure**:
    - Dequeue the first element from the queue.
    - For each character in the current word, try changing it to every letter from 'a' to 'z'.
    - For each new word generated, check if it exists in the dictionary.
    - If the new word equals the end word, return the number of steps taken plus one.
    - If it exists in the dictionary, enqueue the new word and remove it from the dictionary to prevent cycles.
   
4. **Completion**: If the queue is exhausted without finding the end word, return 0, indicating no transformation is possible.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(N * M * 26), where N is the number of words in the dictionary, and M is the length of each word. We generate at most 26 variations of each word and check them against the dictionary.
- **Space Complexity**: O(N) for storing the dictionary and the queue.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int ladderLength(string beginWord, string endWord, vector<string>& wordList) {
        unordered_set<string> dict(wordList.begin(), wordList.end());
        
        if (dict.find(endWord) == dict.end()) {
            return 0; // End word is not in the dictionary
        }

        queue<pair<string, int>> q;
        q.push({beginWord, 1}); // Pair of current word and steps
       
        while (!q.empty()) {
            auto [currentWord, steps] = q.front();
            q.pop();
            
            if (currentWord == endWord) {
                return steps; // Found the end word
            }
            
            for (int i = 0; i < currentWord.length(); i++) {
                char originalChar = currentWord[i];
                
                for (char c = 'a'; c <= 'z'; c++) {
                    if (c == originalChar) continue; // Skip the same letter
                    currentWord[i] = c;
                    
                    if (dict.find(currentWord) != dict.end()) {
                        q.push({currentWord, steps + 1});
                        dict.erase(currentWord); // Prevent re-visiting
                    }
                }
                currentWord[i] = originalChar; // Restore original char
            }
        }
        
        return 0; // No transformation found
    }
};
```
