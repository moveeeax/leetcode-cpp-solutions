# **LeetCode Clone Graph**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to clone an undirected graph. Each node of the graph can be represented as an object in a programming language, and the relationships (edges) between the nodes indicate which nodes are connected. The main goal is to create a deep copy of the graph such that all nodes and their connections are replicated correctly.

### **🚀 Approach**  
To solve this problem, we can utilize a depth-first search (DFS) approach. Here's the breakdown of our strategy:

1. **Node Representation**: 
   - We are given a node structure that contains an integer label and a list of neighboring nodes.
  
2. **HashMap for Cloning**:
   - To keep track of already cloned nodes, we will use a hashmap (or unordered map in C++) that maps original nodes to their corresponding cloned nodes.

3. **DFS Function**:
   - We will implement a recursive DFS function that will:
     - Check if the current node has already been cloned. If it has, return the cloned node.
     - If it hasn’t been cloned, create a new node and store it in the hashmap.
     - Recursively call the DFS for all neighbors of the current node and add those cloned neighbors to the cloned node.

4. **Handle Null Input**: 
   - If the input graph (the starting node) is null, we should return null immediately.

The implementation will focus on efficiently cloning the graph while ensuring that we do not encounter cycles or duplicate processing.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(V + E), where V is the number of vertices (nodes) and E is the number of edges. We visit each node and each edge once.
- **Space Complexity**: O(V), for storing the cloned nodes in the hashmap and the recursion stack.

class Node {
public:
    int val;
    vector<Node*> neighbors;
    Node(int _val) {
        val = _val;
        neighbors = vector<Node*>();
    }
};

class Solution {
public:
    Node* cloneGraph(Node* node) {
        if (!node) return nullptr; // Handle null input
        unordered_map<Node*, Node*> cloned; // Map to store cloned nodes
        return dfs(node, cloned);
    }

    Node* dfs(Node* node, unordered_map<Node*, Node*>& cloned) {
        if (cloned.count(node)) return cloned[node]; // Return already cloned node
        cloned[node] = new Node(node->val); // Clone current node
        for (Node* neighbor : node->neighbors) {
            cloned[node]->neighbors.push_back(dfs(neighbor, cloned)); // Recursively clone neighbors
        }
        return cloned[node];
    }
};
