# **Populating Next Right Pointers in Each Node II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
In this problem, we need to populate the next right pointers for each node in a binary tree. Each node should point to its immediate right neighbor, and if there is no such neighbor, the pointer should be set to `nullptr`. This is a variation of the first problem (Populating Next Right Pointers in Each Node) which handles a perfect binary tree. Here, the tree can be incomplete, which means that we need to traverse level by level to establish the correct connections.

### **🚀 Approach**  
To solve this problem, we can employ a level order traversal approach using a queue or iterative methods that utilize the concept of the `next` pointers. We can use the existing `next` pointers to traverse nodes in the same level. 

1. **Initialize** with the root of the tree and set its next pointer to `nullptr`.
2. **Use a loop** to traverse each level of the tree:
   - For each node in the current level, connect it to its immediate right neighbor using the `next` pointers from nodes already processed.
   - Check the left and right children of the current node and add them to the list for the next level.
3. **Repeat** until all levels are processed.

The complexity of this algorithm will effectively be O(n) since we will be visiting each node exactly once.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n)  where n is the number of nodes in the tree.  
- **Space Complexity**: O(1) if we consider the pointers as part of the tree structure (not counting any extra space used for the queue).

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    void connect(Node* root) {
        if (!root) return;
        
        Node* head = root; // The first node of the next level
        Node* prev = nullptr; // The previous node on the next level
        Node* cur = nullptr; // Current node in the current level
        
        while (head) {
            cur = head; // Start at the beginning of the level
            head = nullptr; // Reset the head for the next level
            
            while (cur) {
                // Process the left child
                if (cur->left) {
                    if (head == nullptr) {
                        head = cur->left; // Set head for the next level
                    }
                    if (prev) {
                        prev->next = cur->left; // Connect previous node to current
                    }
                    prev = cur->left; // Move prev to current left child
                }
                
                // Process the right child
                if (cur->right) {
                    if (head == nullptr) {
                        head = cur->right; // Set head for the next level
                    }
                    if (prev) {
                        prev->next = cur->right; // Connect previous node to current
                    }
                    prev = cur->right; // Move prev to current right child
                }
                
                cur = cur->next; // Move to the next node in the current level
            }
        }
    }
};
```

#include <iostream>
#include <queue>

using namespace std;

struct Node {
    int val;
    Node *left;
    Node *right;
    Node *next;
    
    Node(int x) : val(x), left(NULL), right(NULL), next(NULL) {}
};

class Solution {
public:
    void connect(Node* root) {
        if (!root) return;
        
        Node* head = root;
        Node* prev = nullptr;
        Node* cur = nullptr;
        
        while (head) {
            cur = head;
            head = nullptr;
            
            while (cur) {
                if (cur->left) {
                    if (head == nullptr) {
                        head = cur->left;
                    }
                    if (prev) {
                        prev->next = cur->left;
                    }
                    prev = cur->left;
                }
                
                if (cur->right) {
                    if (head == nullptr) {
                        head = cur->right;
                    }
                    if (prev) {
                        prev->next = cur->right;
                    }
                    prev = cur->right;
                }
                
                cur = cur->next;
            }
        }
    }
};

int main() {
    // Example usage to create a tree and connect nodes
    Solution solution;
    Node* root = new Node(1);
    root->left = new Node(2);
    root->right = new Node(3);
    root->left->left = new Node(4);
    root->left->right = new Node(5);
    root->right->right = new Node(6);

    solution.connect(root);

    // Output the next right pointers for verification
    cout << (root->left->next ? root->left->next->val : -1) << endl; // Should connect to 3
    cout << (root->left->left->next ? root->left->left->next->val : -1) << endl; // Should connect to 5
    cout << (root->left->right->next ? root->left->right->next->val : -1) << endl; // Should connect to 6

    return 0;
}
