# **Minimum Depth of Binary Tree**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The objective is to find the minimum depth of a binary tree, which is defined as the number of nodes along the shortest path from the root node down to the nearest leaf node. A leaf is a node that does not have any children (both left and right children are null).

Given the structure of a binary tree and a few edge cases (like an empty tree), we can use a breadth-first search (BFS) approach to ensure we find the shortest path efficiently. The BFS will allow us to traverse the tree level by level, marking the first leaf node we encounter as the minimum depth.

### **🚀 Approach**  
1. **Breadth-First Search (BFS)**: This approach uses a queue to explore the tree level by level.
2. **Starting Point**: Begin with the root node and add it to the queue. If the root is null (indicating an empty tree), return 0.
3. **Level Order Traversal**: For each level, track the number of nodes and process each node. If a node is a leaf (both left and right child are null), return the current depth (level).
4. **Incremental Depth**: Each time we finish processing one level, we increment the depth counter.
5. **Continue Until Leaf**: This continues until we find the first leaf, providing the minimum depth.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) because we may need to visit all nodes in the worst case.  
- **Space Complexity**: O(n) as we are using a queue to store nodes, which could potentially hold all nodes in the case of a complete binary tree.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int minDepth(TreeNode* root) {
        if (!root) return 0; // if the tree is empty, return 0

        queue<TreeNode*> q;
        q.push(root);
        int depth = 1; // initialize depth

        while (!q.empty()) {
            int size = q.size();
            for (int i = 0; i < size; ++i) {
                TreeNode* node = q.front();
                q.pop();
                
                // Check if we reached a leaf node
                if (!node->left && !node->right) {
                    return depth; // return current depth when a leaf node is found
                }
                
                // Add child nodes to the queue if they exist
                if (node->left) q.push(node->left);
                if (node->right) q.push(node->right);
            }
            depth++; // Increment the depth level
        }
        return depth; // Should not reach here
    }
};
```

#include <iostream>
#include <queue>

// Definition for a binary tree node.
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
};

class Solution {
public:
    int minDepth(TreeNode* root) {
        if (!root) return 0; // if the tree is empty, return 0

        queue<TreeNode*> q;
        q.push(root);
        int depth = 1; // initialize depth

        while (!q.empty()) {
            int size = q.size();
            for (int i = 0; i < size; ++i) {
                TreeNode* node = q.front();
                q.pop();
                
                // Check if we reached a leaf node
                if (!node->left && !node->right) {
                    return depth; // return current depth when a leaf node is found
                }
                
                // Add child nodes to the queue if they exist
                if (node->left) q.push(node->left);
                if (node->right) q.push(node->right);
            }
            depth++; // Increment the depth level
        }
        return depth; // Should not reach here
    }
};

int main() {
    Solution solution;
    
    // Example of creating a binary tree
    TreeNode* root = new TreeNode(1);
    root->left = new TreeNode(2);
    root->right = new TreeNode(3);
    root->left->right = new TreeNode(4);
    
    // Computing minimum depth
    int minDepth = solution.minDepth(root);
    std::cout << "Minimum Depth: " << minDepth << std::endl; // Expected output: 2

    return 0;
}
