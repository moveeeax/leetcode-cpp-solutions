# **LeetCode Subsets II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to find all possible subsets (the power set) of a given integer array that may contain duplicates. Each subset should be unique, meaning that if two subsets are exactly the same, we include it only once in the final result.  

To solve this, we can leverage a backtracking approach, which allows us to build subsets incrementally and explore all potential combinations. Additionally, we need to handle duplicate elements carefully to avoid generating the same subset multiple times.

### **🚀 Approach**  
1. **Sorting**: First, we sort the input array. This will help us easily identify duplicate elements and skip them while backtracking.
2. **Backtracking Function**: We create a recursive function that constructs the current subset and tracks the path and the starting index for the next elements. This function also maintains a list to accumulate our results.
3. **Skip Duplicates**: While iterating through the sorted array, we skip over duplicate numbers to prevent generating the same subset multiple times. If the current element is the same as the previous one and it is not the starting point for that recursive level, we skip it.
4. **Base Case**: When we reach the end of the list, we add the current subset to the result list.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(2^n), where n is the number of elements in the array. Each element can either be included or excluded from a subset, leading to exponential growth in possible subsets.
- **Space Complexity**: O(n), for the recursion stack during the backtracking process and to store each subset.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<int>> subsetsWithDup(vector<int>& nums) {
        vector<vector<int>> result;
        vector<int> subset;
        sort(nums.begin(), nums.end());
        backtrack(result, subset, nums, 0);
        return result;
    }
    
    void backtrack(vector<vector<int>>& result, vector<int>& subset, vector<int>& nums, int start) {
        result.push_back(subset); // add the current subset to the result
        
        for (int i = start; i < nums.size(); ++i) {
            // If the current element is the same as the previous one,
            // we skip it to avoid duplicates in the same subsets level
            if (i > start && nums[i] == nums[i - 1]) continue; 
            subset.push_back(nums[i]); // include nums[i] in the current subset
            backtrack(result, subset, nums, i + 1); // move to the next element
            subset.pop_back(); // backtrack
        }
    }
};
```

#include <vector>
#include <iostream>
#include <algorithm>

class Solution {
public:
    std::vector<std::vector<int>> subsetsWithDup(std::vector<int>& nums) {
        std::vector<std::vector<int>> result;
        std::vector<int> subset;
        std::sort(nums.begin(), nums.end());
        backtrack(result, subset, nums, 0);
        return result;
    }
    
    void backtrack(std::vector<std::vector<int>>& result, std::vector<int>& subset, std::vector<int>& nums, int start) {
        result.push_back(subset); // add the current subset to the result
        
        for (int i = start; i < nums.size(); ++i) {
            if (i > start && nums[i] == nums[i - 1]) continue; // skip duplicates
            subset.push_back(nums[i]); // include nums[i] in the current subset
            backtrack(result, subset, nums, i + 1); // move to the next element
            subset.pop_back(); // backtrack
        }
    }
};

int main() {
    std::vector<int> nums = {1, 2, 2}; // Example input
    Solution solution;
    std::vector<std::vector<int>> result = solution.subsetsWithDup(nums);

    for (const auto& subset : result) {
        std::cout << "[ ";
        for (int num : subset) {
            std::cout << num << " ";
        }
        std::cout << "]\n";
    }
    return 0;
}
