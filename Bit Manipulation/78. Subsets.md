# **LeetCode Subsets**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The goal of this problem is to generate all possible subsets (the power set) from a given set of integers. For a given list of unique integers, we need to find every possible subset, including the empty subset as well as the entire set itself. The order of the resulting subsets does not matter, and each subset should be returned in any order.

### **🚀 Approach**  
To solve the problem of generating all subsets, we can use a backtracking approach. The idea is to explore each decision point for including or excluding each number from the current subset. 

1. **Backtracking Function**: We define a recursive function that takes the current index in the original array and the current subset formed up to that index.
2. **Base Case**: Once we reach the end of the list, we add the current subset to our results.
3. **Include/Exclude**: For each element at the current index, we have two choices:
   - Include the current element in the subset and move to the next index.
   - Exclude the current element and move to the next index.
4. **Construction of Results**: At each call, we add the current subset to our results before returning.

This approach guarantees that we explore all possible combinations of elements.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(2^n), where n is the number of elements in the input list. This is because for each element, we have two choices - to include it or exclude it.
- **Space Complexity**: O(n), which is the space used for the recursion stack. Additionally, we use space for storing all subsets in the results.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<int>> subsets(vector<int>& nums) {
        vector<vector<int>> result;
        vector<int> currentSubset;
        backtrack(nums, 0, currentSubset, result);
        return result;
    }
    
    void backtrack(vector<int>& nums, int index, vector<int>& currentSubset, vector<vector<int>>& result) {
        // Add the current subset to the results
        result.push_back(currentSubset);
        
        // Explore further elements
        for (int i = index; i < nums.size(); i++) {
            // Include the number in the current subset
            currentSubset.push_back(nums[i]);
            // Move to the next index
            backtrack(nums, i + 1, currentSubset, result);
            // Exclude the number from the current subset (backtrack)
            currentSubset.pop_back();
        }
    }
};
```

```cpp
#include <iostream>
#include <vector>
using namespace std;

class Solution {
public:
    vector<vector<int>> subsets(vector<int>& nums) {
        vector<vector<int>> result;
        vector<int> currentSubset;
        backtrack(nums, 0, currentSubset, result);
        return result;
    }
    
    void backtrack(vector<int>& nums, int index, vector<int>& currentSubset, vector<vector<int>>& result) {
        result.push_back(currentSubset);
        
        for (int i = index; i < nums.size(); i++) {
            currentSubset.push_back(nums[i]);
            backtrack(nums, i + 1, currentSubset, result);
            currentSubset.pop_back();
        }
    }
};

int main() {
    Solution solution;
    vector<int> nums = {1, 2, 3}; // Example input
    vector<vector<int>> subsets = solution.subsets(nums);
    
    for (const auto& subset : subsets) {
        cout << "{ ";
        for (int num : subset) {
            cout << num << " ";
        }
        cout << "}" << endl;
    }
    
    return 0;
}
```
