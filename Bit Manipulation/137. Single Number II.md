# **LeetCode Single Number II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to find the number that appears exactly once in an array, where every other number appears exactly three times. To solve this, we can't simply use a hash map due to the potential space complexity. Instead, we can use bit manipulation to keep track of the bits of the numbers.  

### **🚀 Approach**  
To achieve this efficiently:
1. Utilize two variables, `ones` and `twos`, to keep track of the bits seen once and twice, respectively.
2. Iterate through each number in the array.
3. Update `twos` with the intersection of `ones` and the current number.
4. Update `ones` with the XOR of the current number and numbers that have appeared once but not yet twice.
5. Reset `ones` and `twos` based on the current numbers, ensuring only those that appear once remain in `ones`.
6. The final value of `ones` will give us the number that appears once.

This bit manipulation approach allows us to find the solution in linear time with constant space.  

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) - We only loop through the array once.  
- **Space Complexity**: O(1) - We use a fixed number of variables regardless of input size.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ones = 0, twos = 0;
        for (int num : nums) {
            ones = (ones ^ num) & ~twos;
            twos = (twos ^ num) & ~ones;
        }
        return ones;
    }
};  
```
