# **LeetCode Single Number**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem statement requires us to find the single number in an array where every other number appears exactly twice, while one number appears only once. This situation calls for a solution that efficiently identifies the unique number without needing additional space proportional to the input size, considering both time and space constraints.

### **🚀 Approach**  
To solve the problem, we can utilize the properties of XOR (exclusive OR) operation in binary. The key observations are:
- A number XOR'd with itself results in 0 (i.e., `x ^ x = 0`).
- A number XOR'd with 0 remains unchanged (i.e., `x ^ 0 = x`).
- XOR operation is commutative (the order of operations does not matter) and associative (grouping of numbers does not change the result).

By applying the XOR operation iteratively across all numbers in the array, all numbers that appear twice will cancel each other out, leaving only the single number.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of elements in the array, as we only need to iterate through the array once.
- **Space Complexity**: O(1), since no extra space is used proportional to the input size.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int uniqueNumber = 0;
        for (int num : nums) {
            uniqueNumber ^= num;
        }
        return uniqueNumber;
    }
};
```
