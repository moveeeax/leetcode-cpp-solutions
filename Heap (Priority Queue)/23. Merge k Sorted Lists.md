# **LeetCode Merge K Sorted Lists**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires merging `k` sorted linked lists into a single sorted linked list. Each individual list is already sorted, and our goal is to create a new merged list that maintains this sorted order. 

We can visualize the process of merging these lists in several ways, like comparing the head nodes of each list and choosing the smallest one to add to our resulting merged list. Given that we have `k` lists, a naive approach would be to sequentially merge two lists at a time, but that would be inefficient.  

### **🚀 Approach**  
To efficiently merge `k` sorted lists, we can take advantage of a min-heap (priority queue). The steps of the approach are as follows:

1. **Heap Initialization**: Create a min-heap that will store the head nodes of each of the `k` linked lists.
2. **Push Heads into Heap**: Insert the head of each linked list into the heap.
3. **Merge Process**: 
   - Continuously extract the node with the smallest value from the heap.
   - Add this node to the resulting merged linked list.
   - If this node has a next node, push that next node back into the heap.
4. **Termination**: Repeat the process until the heap is empty, ensuring the merged list is fully constructed in sorted order.

This approach ensures that we are always merging the smallest available node from the heads of the lists, maintaining efficient time complexity.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(N log k), where `N` is the total number of nodes across all lists and `k` is the number of lists. The log factor arises because of the operations involving the min-heap.
- **Space Complexity**: O(k), where `k` is the number of lists, due to the storage of pointers in the min-heap.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    ListNode* mergeKLists(vector<ListNode*>& lists) {
        priority_queue<ListNode*, vector<ListNode*>, greater<ListNode*>> minHeap;
        
        for (auto list : lists) {
            if (list) {
                minHeap.push(list);
            }
        }

        ListNode dummy(0);
        ListNode* tail = &dummy;

        while (!minHeap.empty()) {
            ListNode* smallestNode = minHeap.top();
            minHeap.pop();
            tail->next = smallestNode;
            tail = tail->next;

            if (smallestNode->next) {
                minHeap.push(smallestNode->next);
            }
        }
        
        return dummy.next;
    }
};
```
