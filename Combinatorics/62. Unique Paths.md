# **LeetCode Unique Paths**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem consists of finding the number of unique paths in a grid from the top-left corner to the bottom-right corner. You can only move down or right at any point in time. The complexity arises from ensuring that you are counting all possible paths without retracing any steps and efficiently navigating the grid.

To clarify, if you have a grid of size `m x n`, the only valid moves allowed are from position `(x, y)` to `(x+1, y)` (down) or `(x, y+1)` (right). Thus, the algorithm must explore all valid paths without exceeding grid boundaries.

### **🚀 Approach**  
To solve this problem, a dynamic programming approach is optimal. We can use a 2D vector (`dp`) to store the number of ways to reach each cell `(i, j)` in the grid. 

1. **Initialization**:  
   - Set `dp[0][0] = 1` since there is one way to be at the starting position (by starting there).
   - The first row and first column will be initialized to `1` because the only way to reach any cell in the first row or column is by moving right or down exclusively from the starting cell.

2. **Filling the DP Table**:  
   For each cell `(i, j)`, the total number of unique paths to that cell is the sum of the paths from the cell directly above it `(i-1, j)` and the cell directly to the left of it `(i, j-1)`. Hence:
   \[
   dp[i][j] = dp[i-1][j] + dp[i][j-1]
   \]

3. **Final Result**:  
   The value in the bottom-right cell `dp[m-1][n-1]` will give the total number of unique paths from the top-left to bottom-right corner.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(m * n), where `m` is the number of rows and `n` is the number of columns in the grid because we iterate through each cell exactly once.  
- **Space Complexity**: O(m * n) for the `dp` array to hold the number of paths for each cell. We could optimize this to O(n) by storing only the current and previous rows.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int uniquePaths(int m, int n) {
        vector<vector<int>> dp(m, vector<int>(n, 1));
        
        for (int i = 1; i < m; ++i) {
            for (int j = 1; j < n; ++j) {
                dp[i][j] = dp[i-1][j] + dp[i][j-1];
            }
        }
        
        return dp[m-1][n-1];
    }
};
```
