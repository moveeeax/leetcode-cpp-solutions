# **LeetCode Substring with Concatenation of All Words**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem asks us to find the starting indices of all the concatenated substrings formed by a list of words in a given string. Each word in the list can only be used once and must appear in the substring exactly as it appears in the word list. To solve this, we will need to check each substring in the main string and see if it matches a specific length and contains the exact words from the list.

### **🚀 Approach**  
1. **Word Count**: First, we count the total number of words in the list and the total length of each concatenated substring formed by these words.
2. **Sliding Window**: We will use a sliding window technique where we check segments of the main string of the size equal to the length of the concatenated words.
3. **Hash Map for Counting**: We will create a hash map to keep count of the words in the list, which helps us determine if the segment contains all words exactly.
4. **Iterate Over Possible Starts**: We will iterate through the string and check all segments starting at different indices.
5. **Validation Check**: For each segment, we will validate it against the words in the hash map.
6. **Store Indices**: If the segment matches the criteria, we will store the starting index.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n * m), where n is the length of the string and m is the number of words multiplied by the length of each word. This is because we may need to examine each segment multiple times.
- **Space Complexity**: O(m), where m is the number of unique words due to the hash map used for counting.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<int> findSubstring(string s, vector<string>& words) {
        vector<int> result;
        if (s.empty() || words.empty()) return result;

        int wordLength = words[0].size();
        int wordCount = words.size();
        int totalLength = wordLength * wordCount;
        unordered_map<string, int> wordMap;

        for (const string& word : words) {
            wordMap[word]++;
        }

        for (int i = 0; i <= s.size() - totalLength; i++) {
            unordered_map<string, int> seenWords;
            int j = 0;
            while (j < wordCount) {
                string word = s.substr(i + j * wordLength, wordLength);
                if (wordMap.count(word) == 0) break;
                seenWords[word]++;
                if (seenWords[word] > wordMap[word]) break;
                j++;
            }
            if (j == wordCount) result.push_back(i);
        }

        return result;
    }
};
```

        if (s.empty() || words.empty()) return result;

        int wordLength = words[0].size();
