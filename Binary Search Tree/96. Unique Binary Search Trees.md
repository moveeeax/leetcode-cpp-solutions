# **Unique Binary Search Trees**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem revolves around finding the number of distinct binary search trees (BSTs) that can be formed using `n` distinct integers. Let's delve into why this is a combinatorial problem and how understanding the properties of BSTs can help us derive the solution.

A binary search tree is defined by its left and right subtrees which must also be BSTs. A key observation is that for each integer `k` chosen as the root from `1` to `n`, the numbers less than `k` will form the left subtree, and the numbers greater than `k` will form the right subtree. 

The number of distinct trees that can be formed with `k` as the root is the product of the number of distinct trees that can be formed from the left subtree and the right subtree. Specifically:
- If `k` is the root, the left subtree has `k-1` nodes which can be arranged in `dp[k-1]` ways.
- The right subtree has `n-k` nodes which can be arranged in `dp[n-k]` ways.

Thus, the total number of BSTs can be computed using:
\[ dp[n] = \sum_{k=1}^{n} dp[k-1] \times dp[n-k] \]

### **🚀 Approach**  
1. Define a dynamic programming array `dp` where `dp[i]` will represent the number of unique BSTs that can be formed with `i` nodes.
2. Initialize `dp[0]` and `dp[1]` as `1`, because there is one unique BST for zero nodes (the empty tree) and one unique BST for one node.
3. Iterate from `2` to `n`, for every `i`, calculate `dp[i]` using the formula mentioned above, by considering each `k` from `1` to `i` as the root.
4. The final answer will be `dp[n]`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n^2), because for each node count `i`, we might perform up to `i` operations to fill the `dp` table.
- **Space Complexity**: O(n), because we need to store the BST counts in the `dp` array.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int numTrees(int n) {
        vector<int> dp(n + 1, 0);
        dp[0] = dp[1] = 1; // Base cases: 1 BST for 0 or 1 node
        
        for (int i = 2; i <= n; i++) {
            for (int k = 1; k <= i; k++) {
                dp[i] += dp[k - 1] * dp[i - k]; // Calculate dp[i]
            }
        }
        
        return dp[n]; // Return the total number of unique BSTs for n nodes
    }
};  
```
