# **Recover Binary Search Tree**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem involves recovering a binary search tree (BST) where two nodes have been swapped, leading to an incorrect BST. A valid BST is characterized by the fact that for any given node, all values in its left subtree are smaller, and all values in its right subtree are larger. When two nodes are swapped, this condition is violated in specific places.

### **🚀 Approach**  
To solve the problem, we can employ an in-order traversal of the tree, during which we can determine the nodes that are out of order (i.e., the nodes that violate the BST properties). By storing these nodes and subsequently reassigning their values, we can restore the original BST.

The steps are as follows:
1. Perform an in-order traversal to identify nodes where the left child has a value greater than the current node, which indicates a violation.
2. We'll keep track of the first and second swapped nodes.
3. If we find a second violation during the traversal, we will update the second node.
4. At the end of the in-order traversal, swap the values of the identified nodes to repair the BST.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the tree, because we visit each node once.
- **Space Complexity**: O(h), where h is the height of the tree, due to the space used by the recursion stack during in-order traversal.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    TreeNode *first = nullptr, *second = nullptr, *prev = nullptr;

    void inorder(TreeNode* root) {
        if (!root) return;

        inorder(root->left);

        if (prev && prev->val > root->val) {
            if (!first) first = prev;
            second = root;
        }
        prev = root;

        inorder(root->right);
    }

    void recoverTree(TreeNode* root) {
        inorder(root);
        if (first && second) {
            swap(first->val, second->val);
        }
    }
};
```

```cpp
#include <iostream>
using namespace std;

struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
};

class Solution {
public:
    TreeNode *first = nullptr, *second = nullptr, *prev = nullptr;

    void inorder(TreeNode* root) {
        if (!root) return;

        inorder(root->left);

        if (prev && prev->val > root->val) {
            if (!first) first = prev;
            second = root;
        }
        prev = root;

        inorder(root->right);
    }

    void recoverTree(TreeNode* root) {
        inorder(root);
        if (first && second) {
            swap(first->val, second->val);
        }
    }
};

int main() {
    TreeNode* root = new TreeNode(1);
    root->right = new TreeNode(3);
    root->right->left = new TreeNode(2);

    Solution sol;
    sol.recoverTree(root);

    cout << root->val << " " << root->right->val << " " << root->right->left->val << endl; // Expected BST output should be 2 1 3
    return 0;
}
```
