# **LeetCode Generate Parentheses**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires generating all combinations of valid parentheses for a given number `n`, which represents the count of pairs of parentheses. A valid combination means that at no point in the string should there be more closing parentheses than opening ones.

For instance, with `n = 3`, we should generate the following valid combinations:
1. `((()))`
2. `(()())`
3. `(())()`
4. `()(())`
5. `()()()`

### **🚀 Approach**  
To solve this problem, we can use backtracking. Our approach will involve recursively building the combinations of parentheses while ensuring that the conditions for validity are maintained:
1. We will keep track of the number of open and closed parentheses used so far.
2. Initially, both counts are set to 0.
3. We can add an open parenthesis `(` if the count of open parentheses is less than `n`.
4. We can add a closed parenthesis `)` if the count of closed parentheses is less than the count of open parentheses.
5. Once we have used `n` pairs of parentheses, we store the current combination as a valid parenthesis string.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(4^n / √n) - This is due to the nature of combinations. The number of valid parentheses strings can be calculated using Catalan numbers, which grows exponentially with `n`.
- **Space Complexity**: O(n) - For the recursion stack and storing combinations.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    void backtrack(string current, int open, int close, vector<string>& result, int n) {
        if (current.size() == 2 * n) {
            result.push_back(current);
            return;
        }
        if (open < n) {
            backtrack(current + "(", open + 1, close, result, n);
        }
        if (close < open) {
            backtrack(current + ")", open, close + 1, result, n);
        }
    }
    
    vector<string> generateParenthesis(int n) {
        vector<string> result;
        backtrack("", 0, 0, result, n);
        return result;
    }
};
```

#include <iostream>

#include <string>
using namespace std;

class Solution {

    void backtrack(string current, int open, int close, vector<string>& result, int n) {
        if (current.size() == 2 * n) {
            result.push_back(current);
            return;
        }
        if (open < n) {
            backtrack(current + "(", open + 1, close, result, n);
        }
        if (close < open) {
            backtrack(current + ")", open, close + 1, result, n);
        }
    }
    
    vector<string> generateParenthesis(int n) {
        vector<string> result;
        backtrack("", 0, 0, result, n);
        return result;
    }

int main() {
    Solution solution;
    int n;
    cout << "Enter the number of pairs of parentheses: ";

    cout << "Generated parentheses combinations:" << endl;

    }

    return 0;
}
