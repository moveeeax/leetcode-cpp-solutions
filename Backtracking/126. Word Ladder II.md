# **LeetCode Word Ladder II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem involves finding all the shortest transformation sequences from a `beginWord` to an `endWord`, where only one letter can be changed at a time, and each transformed word must exist in a given list of valid words (word list). This forms a graph-like structure where each word represents a node, and an edge exists between two nodes if their corresponding words differ by just one letter.

Given the nature of the problem, we need to perform a breadth-first search (BFS) to explore all possible transformations and then backtrack to build all valid paths from `beginWord` to `endWord`.

### **🚀 Approach**  
1. **BFS for Shortest Paths**: First, we'll perform a BFS starting from `beginWord` to find all reachable words and the shortest transformation steps to `endWord`. We'll maintain a level map to understand the depth of each word.
  
2. **Backtracking to Find All Paths**: After BFS has populated the level map, we'll backtrack from `endWord` to `beginWord` using a depth-first search (DFS). During backtracking, we'll only explore words that form valid one-letter transformations and are at the current level.

3. **Result Construction**: Construct the result as we backtrack and add valid paths to a list.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(N * M^2), where N is the number of words in the word list and M is the length of each word (due to the need to check each character for every word).
- **Space Complexity**: O(N * M) for storing the adjacency list and additional structures used for BFS and backtracking.

## **2️⃣ LeetCode Submission**  
```cpp
class WordLadder {
public:
    vector<vector<string>> findLadders(string beginWord, string endWord, vector<string>& wordList) {
        unordered_set<string> wordSet(wordList.begin(), wordList.end());
        vector<vector<string>> result;
        unordered_map<string, vector<string>> graph;
        unordered_map<string, int> level;

        if (wordSet.find(endWord) == wordSet.end()) return result;

        // Step 1: BFS to build the graph
        queue<string> q;
        q.push(beginWord);
        level[beginWord] = 0;
        while (!q.empty()) {
            int size = q.size();
            bool foundEnd = false;
            unordered_set<string> seen;
            for (int i = 0; i < size; i++) {
                string word = q.front();
                q.pop();
                int currLevel = level[word];
                for (int j = 0; j < word.size(); j++) {
                    char originalChar = word[j];
                    for (char c = 'a'; c <= 'z'; c++) {
                        if (c == originalChar) continue;
                        word[j] = c;
                        if (wordSet.find(word) != wordSet.end()) {
                            // New word found for transformation
                            graph[word].push_back(q.front());
                            if (level.find(word) == level.end()) {
                                level[word] = currLevel + 1;
                                seen.insert(word);
                                if (word == endWord) foundEnd = true;
                                q.push(word);
                            }
                        }
                    }
                    word[j] = originalChar; // Restore original character
                }
            }
            for (const string &word : seen) {
                wordSet.erase(word); // Avoid revisiting words
            }
            if (foundEnd) break;
        }

        // Step 2: Backtrack to find all paths
        vector<string> path;
        backtrack(beginWord, endWord, graph, level, path, result);
        return result;
    }

private:
    void backtrack(string& currWord, string& endWord, unordered_map<string, vector<string>>& graph, unordered_map<string, int>& level, vector<string>& path, vector<vector<string>>& result) {
        path.push_back(currWord);
        if (currWord == endWord) {
            result.push_back(path);
        } else {
            for (const string& nextWord : graph[currWord]) {
                if (level[nextWord] == level[currWord] + 1) {
                    backtrack(nextWord, endWord, graph, level, path, result);
                }
            }
        }
        path.pop_back();
    }
};  
```
