# **Path Sum II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
Given a binary tree, our task is to find all root-to-leaf paths where the sum of the node values equals a given target sum. A root-to-leaf path is defined as a path starting from the root node down to any leaf node (a node with no children). 

To solve this problem, we can traverse the tree using Depth-First Search (DFS) to explore each path from the root to the leaves. At each node, we will keep track of the current path and the remaining sum needed to reach the target.

### **🚀 Approach**  
1. **Depth-First Search (DFS)**: Use a recursive function to traverse the tree. At each node, we'll subtract the node's value from the current sum and update the path.
2. **Base Case**: If we reach a leaf node (both left and right children are null) and the current sum equals the required sum, we add the current path to our results.
3. **Backtracking**: After exploring both left and right subtrees, we'll backtrack by removing the last node from the path when we unwind the recursion.
4. **Store Results**: Use a vector to store valid paths that match the required sum condition.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(N), where N is the number of nodes in the tree. In the worst case, we have to check every node once.
- **Space Complexity**: O(H), where H is the height of the tree. The maximum depth of the recursion stack corresponds to the height of the tree.

## **2️⃣ LeetCode Submission**

class Solution {
public:
    vector<vector<int>> result;

    void dfs(TreeNode* node, int sum, vector<int>& path) {
        if (!node) return;

        path.push_back(node->val);
        sum -= node->val;

        if (!node->left && !node->right && sum == 0) {
            result.push_back(path);
        } else {
            dfs(node->left, sum, path);
            dfs(node->right, sum, path);
        }
        
        path.pop_back();
    }

    vector<vector<int>> pathSum(TreeNode* root, int targetSum) {
        vector<int> path;
        dfs(root, targetSum, path);
        return result;
    }
};
```

#include <iostream>
#include <vector>
using namespace std;

struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
};

class Solution {
public:
    vector<vector<int>> result;

    void dfs(TreeNode* node, int sum, vector<int>& path) {
        if (!node) return;

        path.push_back(node->val);
        sum -= node->val;

        if (!node->left && !node->right && sum == 0) {
            result.push_back(path);
        } else {
            dfs(node->left, sum, path);
            dfs(node->right, sum, path);
        }
        
        path.pop_back();
    }

    vector<vector<int>> pathSum(TreeNode* root, int targetSum) {
        vector<int> path;
        dfs(root, targetSum, path);
        return result;
    }
};

int main() {
    Solution solution;
    
    // Constructing a sample tree
    TreeNode* root = new TreeNode(5);
    root->left = new TreeNode(4);
    root->right = new TreeNode(8);
    root->left->left = new TreeNode(11);
    root->left->left->left = new TreeNode(7);
    root->left->left->right = new TreeNode(2);
    root->right->left = new TreeNode(13);
    root->right->right = new TreeNode(4);
    root->right->right->right = new TreeNode(1);
    
    int targetSum = 22;
    vector<vector<int>> result = solution.pathSum(root, targetSum);
    
    // Output results
    for (const auto& path : result) {
        for (int val : path) {
            cout << val << " ";
        }
        cout << endl;
    }

    // Clean up memory
    delete root->left->left->right;
    delete root->left->left->left;
    delete root->left->left;
    delete root->left;
    delete root->right->right->right;
    delete root->right->right;
    delete root->right->left;
    delete root->right;
    delete root;

    return 0;
}
```
