# **[Remove Invalid Parentheses](https://leetcode.com/problems/remove-invalid-parentheses/description/)**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem involves removing the minimum number of invalid parentheses from a given input string to make it a valid parentheses string. A valid parentheses string is one where:
1. Every opening parenthesis has a corresponding closing parenthesis.
2. The parentheses are closed in the correct order.

For example, the string "(_))" can be turned into "()" by removing one invalid parenthesis. The challenge is to find all possible valid strings that can be generated by removing the necessary parentheses.

### **🚀 Approach**  
1. **Counting Invalid Parentheses**: First, we traverse the string to count the number of unmatched opening and closing parentheses. This helps determine how many parentheses need to be removed to make the string valid.
  
2. **Breadth-First Search (BFS)**: We can use BFS to explore all possible combinations of strings formed by removing invalid parentheses. Starting with the original string, we systematically remove one parenthesis at a time and check if the resulting string is valid.
  
3. **Validation Check**: We'll maintain a function to check if a given string has valid parentheses. This will be used to validate each string generated during the BFS process.

4. **Avoiding Duplicates**: Since different sequences of removals can lead to the same string, we need to store results in a set to avoid duplicates.

5. **Result Collection**: Once valid strings are identified, they will be collected and returned as a list.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(2^n) in the worst case, where n is the length of the string. This is due to the combinatorial nature of the problem as we explore all possible states.
- **Space Complexity**: O(n) for storing the queue and the resulting set of valid strings.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<string> removeInvalidParentheses(string s) {
        int left = 0, right = 0;
        for (char c : s) {
            if (c == '(') left++;
            else if (c == ')') {
                if (left > 0) left--;
                else right++;
            }
        }

        unordered_set<string> result;
        queue<string> q;
        q.push(s);
        bool found = false;
        
        while (!q.empty()) {
            s = q.front();
            q.pop();
            if (isValid(s)) {
                result.insert(s);
                found = true;
            }
            if (found) continue;

            for (int i = 0; i < s.size(); i++) {
                if (s[i] == '(' || s[i] == ')') {
                    string next = s.substr(0, i) + s.substr(i + 1);
                    q.push(next);
                }
            }
        }
        
        return vector<string>(result.begin(), result.end());
    }

    bool isValid(const string& s) {
        int balance = 0;
        for (char c : s) {
            if (c == '(') balance++;
            else if (c == ')') balance--;
            if (balance < 0) return false;
        }
        return balance == 0;
    }
};  
```
