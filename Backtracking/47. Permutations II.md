# **LeetCode Permutations II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to generate all unique permutations of a list of integers which may contain duplicates. A naive approach would involve using standard permutation generation techniques, but this would lead to duplicates in cases where the input array contains repeated elements. Thus, a more refined approach is necessary to ensure we only capture distinct permutations.

### **🚀 Approach**  
To solve the problem, we will use a backtracking algorithm along with a frequency count to track the usage of elements:

1. **Sort the Array**: Start by sorting the input array. This arrangement of elements will help to easily skip duplicates.
  
2. **Backtracking Function**: 
   - Define a recursive function that builds the current permutation.
   - Maintain a boolean array to keep track of which elements are being used, and another list to store the current permutation.
  
3. **Base Case**: If the current permutation is of the same length as the input, store it in the results as a valid permutation.

4. **Recursive Exploration**: Loop through the numbers. For each number:
   - Skip duplicates (only process the first occurrence of a number that is the same as previous).
   - Mark the number as used, add it to the current permutation, and recurse.
   - After recursion, backtrack by unmarking the number and removing it from the current permutation.

5. **Return the Results**: Finally, return the list of unique permutations accumulated during the backtracking process.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n * n!), where n is the size of the `nums` array as we are generating all permutations.
- **Space Complexity**: O(n), for storing the current permutation and a boolean array to track used elements.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<int>> permuteUnique(vector<int>& nums) {
        vector<vector<int>> result;
        vector<int> current;
        vector<bool> used(nums.size(), false);
        sort(nums.begin(), nums.end());
        backtrack(nums, current, used, result);
        return result;
    }
    
    void backtrack(vector<int>& nums, vector<int>& current, vector<bool>& used, vector<vector<int>>& result) {
        if (current.size() == nums.size()) {
            result.push_back(current);
            return;
        }
        
        for (int i = 0; i < nums.size(); ++i) {
            if (used[i]) continue;
            if (i > 0 && nums[i] == nums[i - 1] && !used[i - 1]) continue;
            used[i] = true;
            current.push_back(nums[i]);
            backtrack(nums, current, used, result);
            used[i] = false;
            current.pop_back();
        }
    }
};
```

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

class Solution {
public:
    vector<vector<int>> permuteUnique(vector<int>& nums) {
        vector<vector<int>> result;
        vector<int> current;
        vector<bool> used(nums.size(), false);
        sort(nums.begin(), nums.end());
        backtrack(nums, current, used, result);
        return result;
    }
    
    void backtrack(vector<int>& nums, vector<int>& current, vector<bool>& used, vector<vector<int>>& result) {
        if (current.size() == nums.size()) {
            result.push_back(current);
            return;
        }
        
        for (int i = 0; i < nums.size(); ++i) {
            if (used[i]) continue;
            if (i > 0 && nums[i] == nums[i - 1] && !used[i - 1]) continue;
            used[i] = true;
            current.push_back(nums[i]);
            backtrack(nums, current, used, result);
            used[i] = false;
            current.pop_back();
        }
    }
};

int main() {
    Solution solution;
    vector<int> nums = {1, 1, 2};
    vector<vector<int>> result = solution.permuteUnique(nums);
    
    for (const auto& perm : result) {
        for (int num : perm) {
            cout << num << " ";
        }
        cout << endl;
    }
    
    return 0;
}
