# **LeetCode Combinations**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires generating all possible combinations of a specified length `k` from numbers in the range `1` to `n`. This is a classic combinatorial problem where we need to explore all possible subsets of size `k` from a larger set. The combinations should be represented in ascending order, and each combination should be displayed on a new line.

### **🚀 Approach**  
1. **Backtracking**: The most effective way to generate combinations is through backtracking, where we progressively build potential solutions. 
2. **Recursive Function**: We'll employ a recursive function that maintains a current combination and builds upon it:
   - If the size of the current combination equals `k`, we store it.
   - If the size is smaller than `k`, we loop through the remaining numbers, adding each one to the current combination and recursively calling the function to continue building.
3. **Base Condition**: The recursion must terminate appropriately when either a solution is found or when we've exhausted possible candidates.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(C(n, k)), where C(n, k) is the number of combinations, since we potentially generate each combination once.
- **Space Complexity**: O(k) for the current combination stack space during recursion.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    void backtrack(int n, int k, int start, vector<int>& current, vector<vector<int>>& results) {
        if (current.size() == k) {
            results.push_back(current);
            return;
        }
        
        for (int i = start; i <= n; ++i) {
            current.push_back(i);
            backtrack(n, k, i + 1, current, results);
            current.pop_back();
        }
    }
    
    vector<vector<int>> combine(int n, int k) {
        vector<vector<int>> results;
        vector<int> current;
        backtrack(n, k, 1, current, results);
        return results;
    }
};
```

```cpp
#include <iostream>
#include <vector>
using namespace std;

class Solution {
public:
    void backtrack(int n, int k, int start, vector<int>& current, vector<vector<int>>& results) {
        if (current.size() == k) {
            results.push_back(current);
            return;
        }
        
        for (int i = start; i <= n; ++i) {
            current.push_back(i);
            backtrack(n, k, i + 1, current, results);
            current.pop_back();
        }
    }
    
    vector<vector<int>> combine(int n, int k) {
        vector<vector<int>> results;
        vector<int> current;
        backtrack(n, k, 1, current, results);
        return results;
    }
};

int main() {
    Solution solution;
    int n = 4, k = 2; // Example input
    vector<vector<int>> result = solution.combine(n, k);
    
    for (const auto& combination : result) {
        for (int num : combination) {
            cout << num << " ";
        }
        cout << endl;
    }
    return 0;
}
```
