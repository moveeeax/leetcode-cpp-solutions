# **N-Queens II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to determine the number of unique solutions that can be formed by placing N queens on an N x N chessboard such that no two queens threaten each other. Queens can attack each other horizontally, vertically, and diagonally. The challenge lies in efficiently counting all distinct arrangements without explicitly constructing and storing all valid configurations.

### **🚀 Approach**  
To solve this problem, we can use a backtracking approach. The core idea is to attempt to place queens row by row, ensuring at each step that no two queens can attack each other. We utilize three arrays (or boolean vectors) to track the constraints:
1. `columns`: This array keeps track of which columns are already occupied by queens.
2. `diag1`: This array keeps track of the major diagonals (difference between row index and column index).
3. `diag2`: This array maintains the minor diagonals (sum of row index and column index).

Using these three arrays, we can effectively prevent placing queens in threatening positions and thus maximize our efficiency. When placing a queen in a particular row, we iterate over each column and check if the column and both diagonals are free for placement.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(N!), due to the nature of the backtracking approach exploring each possible configuration.
- **Space Complexity**: O(N), for the storage of the three boolean arrays used for tracking placements.

## **2️⃣ LeetCode Submission**  
```cpp
class NQueensII {
public:
    int totalNQueens(int n) {
        int count = 0;
        vector<bool> columns(n, false);
        vector<bool> diag1(2 * n, false);
        vector<bool> diag2(2 * n, false);
        backtrack(0, n, columns, diag1, diag2, count);
        return count;
    }

private:
    void backtrack(int row, int n, vector<bool>& columns, vector<bool>& diag1, vector<bool>& diag2, int& count) {
        if (row == n) {
            count++;
            return;
        }
        for (int col = 0; col < n; col++) {
            if (!columns[col] && !diag1[row - col + n] && !diag2[row + col]) {
                columns[col] = diag1[row - col + n] = diag2[row + col] = true;
                backtrack(row + 1, n, columns, diag1, diag2, count);
                columns[col] = diag1[row - col + n] = diag2[row + col] = false;
            }
        }
    }
};
```
