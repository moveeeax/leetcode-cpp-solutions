# **LeetCode Valid Number**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem is to determine whether a given string is a valid numerical representation according to specific rules. A valid number can be an integer or a floating-point number, and it may also include an optional exponent part, which typically consists of the letter 'e' or 'E'.

### **🚀 Approach**  
To solve this problem, we can use a finite state machine (FSM) that transitions between states based on the characters encountered in the input string. The potential states include:
- **Initial**: Start state where we can encounter a sign, digits, or a decimal point.
- **Integer**: This state is where digits are read.
- **Decimal**: This state indicates we are reading a decimal point followed possibly by digits.
- **Exponent**: This indicates we've started reading an exponent part after an 'e' or 'E', where we can have an optional sign followed by digits.

The main idea is to iterate through each character of the string and transition between these defined states. We'll keep flags to ensure we don't end up in invalid configurations, such as having multiple decimal points or an exponent without a preceding number.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the length of the string, since we traverse each character at most once.  
- **Space Complexity**: O(1), as we're using a constant amount of space to track states.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool isNumber(string s) {
        int n = s.size();
        int i = 0;
        
        // Skip leading spaces
        while (i < n && s[i] == ' ') i++;
        
        // Check optional sign
        if (i < n && (s[i] == '+' || s[i] == '-')) i++;
        
        bool hasNum = false;
        bool hasDot = false;
        bool hasE = false;
        
        while (i < n) {
            if (isdigit(s[i])) {
                hasNum = true;
            } else if (s[i] == '.') {
                if (hasDot || hasE) return false; // Two dots or dot after e
                hasDot = true;
            } else if (s[i] == 'e' || s[i] == 'E') {
                if (hasE || !hasNum) return false; // Invalid if e appears more than once or before a number
                hasE = true;
                // Check optional sign after e
                i++;
                if (i < n && (s[i] == '+' || s[i] == '-')) i++;
                if (i >= n || !isdigit(s[i])) return false; // After e must have number
            } else {
                return false; // Invalid character
            }
            i++;
        }
        
        // Location must have passed through at least one digit
        return hasNum;
    }
};
```
