# **LeetCode Simplify Path**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to simplify a Unix-style path, which may include redundant `.` and `..` segments, along with multiple slashes. The goal is to return a canonical path that represents the same location in the file system in a simplest form.  

To clarify:
- `.` refers to the current directory and can be ignored.
- `..` refers to the parent directory, effectively removing the last valid directory from the simplified path.
- Redundant slashes (e.g., `//`) should be collapsed into single slashes.
- The resulting path should always start with a `/` and will not end with a `/` unless it's the root directory.

### **🚀 Approach**  
The most efficient way to handle this is to utilize a stack data structure:
1. Split the input path using `/` as the delimiter.
2. Iterate through each component of the split path:
   - If it's an empty string or a `.` (current directory), skip it.
   - If it's `..` (parent directory), pop from the stack if there’s a valid directory to pop.
   - Otherwise, push the component onto the stack as it represents a valid directory.
3. After processing all components, we can construct the simplified path from the contents of the stack. Join the items in the stack with `/` and prepend it with a `/`.
4. If the stack is empty, return `/` indicating the root directory.  

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of characters in the input path (processing each character).
- **Space Complexity**: O(n), in the worst case where all directories are valid and there are no `..` to pop off.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    string simplifyPath(string path) {
        vector<string> stack;
        stringstream ss(path);
        string token;

        while (getline(ss, token, '/')) {
            if (token == "" || token == ".") continue; // skip empty parts and current dir
            if (token == "..") {
                if (!stack.empty()) stack.pop_back(); // go to parent dir if exists
            } else {
                stack.push_back(token); // valid directory
            }
        }
        
        if (stack.empty()) return "/"; // if nothing in stack, return root
        
        string result;
        for (const string& dir : stack) {
            result += "/" + dir; // build the simplified path
        }
        return result;
    }
};
```

class Solution {

        vector<string> stack;
        stringstream ss(path);
        string token;

        while (getline(ss, token, '/')) {
            if (token == "" || token == ".") continue; // skip empty parts and current dir
            if (token == "..") {
                if (!stack.empty()) stack.pop_back(); // go to parent dir if exists
            } else {

        }

int main() {
    Solution solution;
