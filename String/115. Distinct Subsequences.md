# **LeetCode Distinct Subsequences**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to determine the number of distinct subsequences of a string `s` that match a string `t`. A subsequence is derived by deleting some or none of the characters from `s` while maintaining the order of the remaining characters.

To clarify, for two strings:
- **s**: "rabbbit"  
- **t**: "rabbit"  
The distinct subsequences of `s` that equal `t` are few, and we need to count them.

### **🚀 Approach**  
We'll use dynamic programming to solve this problem. The idea is to maintain a 2D dp array where `dp[i][j]` will store the number of distinct subsequences of `s[0..i-1]` that match `t[0..j-1]`. 

1. **Base Case**: `dp[0][0] = 1`, as there is one way to match two empty strings.
2. **Initialization of the First Row**: If `t` is empty (`j = 0`), then `dp[i][0] = 1` for all `i`, reflecting that an empty `t` can be matched by any prefix of `s` (by deleting all characters).
3. **Filling the DP Table**:
   - If the current characters of `s` and `t` match (i.e., `s[i-1] == t[j-1]`), then:
     - Include this match: add `dp[i-1][j-1]` (ways to form the subsequence without these two characters).
     - Exclude this match: add `dp[i-1][j]` (ways to match `t` with the first `i-1` characters of `s`).
   - If the characters don't match, we just take the value from the left: `dp[i][j] = dp[i-1][j]`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(m * n), where m is the length of `s` and n is the length of `t`.
- **Space Complexity**: O(m * n) due to the dp array usage.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int numDistinct(string s, string t) {
        int m = s.size(), n = t.size();
        vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));

        // Base case
        dp[0][0] = 1;

        // Fill in dp table
        for (int i = 0; i <= m; ++i) {
            dp[i][0] = 1; // An empty t can be formed by any prefix of s
        }

        for (int i = 1; i <= m; ++i) {
            for (int j = 1; j <= n; ++j) {
                if (s[i - 1] == t[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1] + dp[i - 1][j];
                } else {
                    dp[i][j] = dp[i - 1][j];
                }
            }
        }

        return dp[m][n];
    }
};
```
