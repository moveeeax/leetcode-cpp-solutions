# **LeetCode Palindrome Partitioning**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem at hand is to find all possible ways to partition a given string such that every substring is a palindrome. A palindrome is a string that reads the same forwards and backwards. The key challenge lies in efficiently generating all possible partitions, filtering them to return only those where each substring is a palindrome. 

### **🚀 Approach**  
To solve this problem, we can use a backtracking approach combined with a helper function to check if a substring is a palindrome:

1. **Backtracking**:
   - Utilize a recursive function to generate partitions.
   - At each position in the string, iterate through all possible ending indices for substrings, adding those that are palindromic to the current partition.
   - If a substring is valid, recursively call the function on the remaining substring.
   - When we reach the end of the string, we add the current valid partition to the result list.

2. **Palindrome Check**:
   - Implement a helper function to verify if a given substring is a palindrome.
   - This can be done by comparing characters from the start and end of the substring moving towards the center.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n * 2^n) since in the worst case we have to consider 2^n possible partitions, and checking each substring takes O(n).
- **Space Complexity**: O(n) for the call stack due to recursion, and O(n) for the current partition being built.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<string>> partition(string s) {
        vector<vector<string>> result;
        vector<string> current;
        backtrack(s, 0, current, result);
        return result;
    }

private:
    void backtrack(const string& s, int start, vector<string>& current, vector<vector<string>>& result) {
        if (start >= s.length()) {
            result.push_back(current);
            return;
        }
        
        for (int i = start; i < s.length(); i++) {
            if (isPalindrome(s, start, i)) {
                current.push_back(s.substr(start, i - start + 1));
                backtrack(s, i + 1, current, result);
                current.pop_back();
            }
        }
    }

    bool isPalindrome(const string& s, int left, int right) {
        while (left < right) {
            if (s[left++] != s[right--]) {
                return false;
            }
        }
        return true;
    }
};  
```

#include <iostream>
#include <vector>
#include <string>
using namespace std;

class Solution {
public:
    vector<vector<string>> partition(string s) {
        vector<vector<string>> result;
        vector<string> current;
        backtrack(s, 0, current, result);
        return result;
    }

private:
    void backtrack(const string& s, int start, vector<string>& current, vector<vector<string>>& result) {
        if (start >= s.length()) {
            result.push_back(current);
            return;
        }
        
        for (int i = start; i < s.length(); i++) {
            if (isPalindrome(s, start, i)) {
                current.push_back(s.substr(start, i - start + 1));
                backtrack(s, i + 1, current, result);
                current.pop_back();
            }
        }
    }

    bool isPalindrome(const string& s, int left, int right) {
        while (left < right) {
            if (s[left++] != s[right--]) {
                return false;
            }
        }
        return true;
    }
};

int main() {
    Solution solution;
    string input = "aab";
    vector<vector<string>> partitions = solution.partition(input);
    
    for (const auto& partition : partitions) {
        cout << "[ ";
        for (const auto& str : partition) {
            cout << "\"" << str << "\" ";
        }
        cout << "]" << endl;
    }
    
    return 0;
}
