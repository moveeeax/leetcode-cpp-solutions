# **LeetCode Add Binary**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The task is to perform binary addition of two given binary strings, `a` and `b`, and return their sum as a binary string. Since both `a` and `b` can vary in length, and might not fit within standard integer types for straightforward addition, we need to simulate the binary addition manually. Binary addition rules are simple: 0 + 0 = 0, 0 + 1 = 1, 1 + 0 = 1, and 1 + 1 = 0 (with a carry of 1). 

### **🚀 Approach**  
1. **Initialization**: Set up variables to hold the indices of `a` and `b`, a carry variable initialized to 0, and a string to build the result.
2. **Iterate from the end**: We will process both strings from the last character (least significant bit) towards the first, performing addition until both strings are fully processed.
3. **Bitwise Addition**:
    - For each corresponding bit, convert the characters to integers (0 or 1), sum them along with the carry.
    - Determine the current bit of the result (using modulo operation) and update the carry (using integer division).
    - Prepend the result (to build the binary string correctly since we are iterating backwards).
4. **Final Carry**: After processing both strings, check if there is any remaining carry to prepend to the result.
5. **Return the Result**: Convert the list or string of bits back to a binary string and return it.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(max(n, m)), where n and m are the lengths of the two binary strings. We iterate through both strings once.
- **Space Complexity**: O(max(n, m)), as we build the result string based on the maximum length of the two inputs.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    string addBinary(string a, string b) {
        string result;
        int i = a.size() - 1, j = b.size() - 1, carry = 0;

        while (i >= 0 || j >= 0 || carry) {
            if (i >= 0) carry += a[i--] - '0'; // Convert char to int
            if (j >= 0) carry += b[j--] - '0'; // Convert char to int
            
            result.insert(result.begin(), (carry % 2) + '0'); // Append current bit
            carry /= 2; // Update carry
        }

        return result;
    }
};  
```
