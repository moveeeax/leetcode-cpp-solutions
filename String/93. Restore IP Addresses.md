# **LeetCode Restore IP Addresses**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The task requires generating all possible valid IP addresses from a given string of digits. An IP address consists of four octets (segments), each ranging from 0 to 255. To solve this, we must explore every combination of splitting the input string into these four segments while ensuring that they meet the criteria of being valid IP address segments.

### **🚀 Approach**  
1. **Backtracking**: Implement a backtracking approach where we try to split the string into four parts. At each step, we will consider the next segment and check if it's valid.
2. **Validation**: Each selected segment should:
   - Have a length of 1 to 3 characters.
   - Not start with '0' unless it is '0' itself.
   - Be less than or equal to 255.
3. **Base Condition**: We only yield results once we’ve created four segments and used the entire input string.
4. **Recursive Function**: Define a recursive function that takes the current segment being filled, the string, and the current position in the string.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(2^n), where n is the length of the input string. This accounts for the potential valid splits across an exponential number of possibilities.
- **Space Complexity**: O(n), due to the usage of space for storing the resultant list of IP addresses.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    void backtrack(string& s, int start, vector<string>& result, vector<string>& current) {
        if (current.size() == 4) {
            if (start == s.size()) {
                result.push_back(current[0] + "." + current[1] + "." + current[2] + "." + current[3]);
            }
            return;
        }

        for (int length = 1; length <= 3; length++) {
            if (start + length > s.size()) break;

            string segment = s.substr(start, length);
            if ((length > 1 && segment[0] == '0') || (length == 3 && stoi(segment) > 255)) {
                continue;
            }

            current.push_back(segment);
            backtrack(s, start + length, result, current);
            current.pop_back();
        }
    }

    vector<string> restoreIpAddresses(string s) {
        vector<string> result;
        vector<string> current;
        backtrack(s, 0, result, current);
        return result;
    }
};
```
