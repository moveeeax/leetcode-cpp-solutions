# **Count and Say**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The "Count and Say" problem requires generating a sequence based on a particular naming convention. Starting with "1", each term is derived from the previous term by describing how many times each digit appears consecutively. For example:
- The first term is "1".
- The second term describes the first: "one 1" or "11".
- The third describes the second: "two 1s" or "21".
- The fourth describes the third: "one 2, then one 1" or "1211".

This leads us to the realization that each term is built by counting consecutive digits and stating the quantities followed by the digit itself. 

### **🚀 Approach**  
1. **Base Case**: Start with "1" as the first term when n = 1.
2. **Iterate** through numbers from 2 to n, building each subsequent term from the last one.
3. For each term, iterate through its digits:
   - Count how many times each digit appears consecutively.
   - Construct the next term by appending this count followed by the digit.
4. Return the nth term after constructing it.

This method relies on systematic counting and string construction, ensuring clarity in interpreting the count-and-say sequence.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(N * M), where N is the number of iterations for constructing terms, and M is the average length of the terms generated (the terms grow in length significantly).
- **Space Complexity**: O(M), where M is the space required to store the current term being generated.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    string countAndSay(int n) {
        if (n == 1) return "1";
        string result = "1";
        
        for (int i = 2; i <= n; ++i) {
            string current = "";
            char lastChar = result[0];
            int count = 0;
            
            for (char c : result) {
                if (c == lastChar) {
                    count++;  // Count how many times the last character appears
                } else {
                    current += to_string(count) + lastChar;  // Append count and character to the result
                    lastChar = c;  // Update the last character
                    count = 1;     // Reset count for the new character
                }
            }
            current += to_string(count) + lastChar;  // Handle the last group
            result = current;  // Update result for next iteration
        }
        
        return result;
    }
};  
```
