# **Length of Last Word**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to determine the length of the last word in a given string `s`, which may contain multiple words separated by spaces. A word is defined as a contiguous sequence of non-space characters. The last word is the segment of characters that appears after the last space in the string. If there are no words (i.e., if the string only contains spaces), the expected answer is `0`.  

### **🚀 Approach**  
To efficiently find the length of the last word, we can use a simple traversal approach. We will start from the end of the string and move backward until we find a character that is not a space, which would mark the start of the last word. We continue counting characters until we encounter another space or reach the start of the string. This method enables us to determine the length of the last word without needing to split the string into words or create additional data structures.  

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the length of the string `s`. In the worst case, we may need to traverse the entire string.
- **Space Complexity**: O(1), as we are only using a couple of variables to store the count and the current index.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int lengthOfLastWord(string s) {
        int length = 0;
        int index = s.size() - 1;

        // Skip trailing spaces
        while (index >= 0 && s[index] == ' ') {
            index--;
        }

        // Count the length of the last word
        while (index >= 0 && s[index] != ' ') {
            length++;
            index--;
        }

        return length;
    }
};  
```
