# **LeetCode Decode Ways**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to decode a string of digits where each digit (or combination of digits) can represent a letter in the alphabet (1 = A, 2 = B, ..., 26 = Z). Given a string `s` of digits, we need to determine the total number of ways to decode it. For example, the string "12" can be decoded as "AB" (1, 2) or "L" (12), resulting in 2 possible ways. We need to consider all possible combinations and ensure that we only count valid decodings.

### **🚀 Approach**  
The best approach to solve this problem is through dynamic programming. We can maintain an array `dp` where `dp[i]` represents the number of ways to decode the substring of the first `i` characters of `s`. 

1. **Initialization**: 
   - Start by setting `dp[0] = 1`, since there's one way to decode an empty string.
   - If the first character in `s` is not '0', then set `dp[1] = 1` (there's one way to decode it).

2. **Filling the DP Array**: 
   - Loop from the second character to the end of the string:
     - If the current character (`s[i-1]`) is not '0', add `dp[i-1]` to `dp[i]` (as this character can stand alone).
     - If the two characters together (`s[i-2]s[i-1]`) form a valid number between 10 and 26, add `dp[i-2]` to `dp[i]` (as this combination can be decoded as one letter).

3. **Final Result**: 
   - The result will be stored in `dp[n]`, where `n` is the length of the string `s`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) - We visit each character once.  
- **Space Complexity**: O(n) - We maintain an array of size `n+1`.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int numDecodings(string s) {
        if (s.empty() || s[0] == '0') return 0;
        
        int n = s.size();
        vector<int> dp(n + 1, 0);
        dp[0] = 1;  // base case: one way to decode an empty string
        dp[1] = 1;  // base case: valid first character
        
        for (int i = 2; i <= n; i++) {
            // Check single digit decoding
            if (s[i - 1] != '0') {
                dp[i] += dp[i - 1];
            }
            // Check two digit decoding
            int twoDigit = stoi(s.substr(i - 2, 2));
            if (twoDigit >= 10 && twoDigit <= 26) {
                dp[i] += dp[i - 2];
            }
        }
        
        return dp[n];
    }
};  
```
