# **LeetCode Word Search**

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem asks us to determine whether a given word can be found in a 2D board of characters. The word can be constructed from letters of sequentially adjacent cells, where two cells are adjacent if they share a side. Importantly, the same letter cell may not be used more than once for the same word. The task involves searching through the grid and keeping track of the letters that are already used as we form the word.

### **🚀 Approach**  
1. **Depth-First Search (DFS)**: We will employ a recursive DFS algorithm that explores all possible paths in the grid to form the given word. 
2. **Backtracking**: To avoid using the same cell multiple times, we will mark the current cell as visited (by changing its character to a special value) and backtrack to restore it afterward.
3. **Boundary Conditions**: Each time we move in a direction (up, down, left, right), we need to ensure that we remain within the grid's bounds and match the next expected character from the word.
4. **Early Termination**: If we find the word during our search, we can terminate early and return `true`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(M * N * 4^L), where M is the number of rows, N is the number of columns, and L is the length of the word. In the worst case, we explore all directions for each cell.
- **Space Complexity**: O(L), where L is the length of the word, for the recursive call stack in the DFS.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool exist(vector<vector<char>>& board, string word) {
        int rows = board.size();
        int cols = board[0].size();
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                if (backtrack(board, word, 0, i, j)) {
                    return true;
                }
            }
        }
        return false;
    }
    
    bool backtrack(vector<vector<char>>& board, string& word, int index, int x, int y) {
        if (index == word.size()) {
            return true;  // Found the word
        }
        if (x < 0 || x >= board.size() || y < 0 || y >= board[0].size() || board[x][y] != word[index]) {
            return false;  // Out of bounds or character does not match
        }
        
        char temp = board[x][y];
        board[x][y] = '#';  // Mark cell as visited

        // Explore all possible directions
        bool found = backtrack(board, word, index + 1, x + 1, y) ||
                     backtrack(board, word, index + 1, x - 1, y) ||
                     backtrack(board, word, index + 1, x, y + 1) ||
                     backtrack(board, word, index + 1, x, y - 1);
        
        board[x][y] = temp;  // Restore cell
        return found;
    }
};  
```
