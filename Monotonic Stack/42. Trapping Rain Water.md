# **Trapping Rain Water**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem is to determine how much rainwater can be trapped between the bars of varying heights, after it rains. Each bar's height is represented in an array, and we need to find spaces between bars that can hold water. To visualize this, imagine the bars as vertical heights where the water can be held in the valleys formed by lower bars between taller bars.

### **🚀 Approach**  
To solve this problem efficiently:
1. **Two-Pointer Technique**: We will use two pointers (left and right) that start at the beginning and the end of the array, respectively. 
2. **Tracking Water Levels**: We will maintain two variables to keep track of the maximum height encountered so far from both sides (`left_max` and `right_max`).
3. **Calculating Trapped Water**: As we move towards the center:
   - If `left_max` is less than `right_max`, it means that the left side limits how much water can be trapped at the left pointer position. We calculate the trapped water at the left pointer and move it inward.
   - If `right_max` is less than or equal to `left_max`, we do the same for the right pointer.

This method runs in linear time and uses constant space.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n)  
- **Space Complexity**: O(1)

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int trap(vector<int>& height) {
        if(height.empty()) return 0;
        
        int left = 0, right = height.size()-1;
        int left_max = 0, right_max = 0;
        int water = 0;

        while(left < right) {
            if(height[left] < height[right]) {
                if(height[left] >= left_max) {
                    left_max = height[left];
                } else {
                    water += left_max - height[left];
                }
                left++;
            } else {
                if(height[right] >= right_max) {
                    right_max = height[right];
                } else {
                    water += right_max - height[right];
                }
                right--;
            }
        }
        return water;
    }
};  
```
