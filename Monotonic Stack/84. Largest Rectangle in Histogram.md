# **Largest Rectangle in Histogram**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to find the area of the largest rectangle that can be formed within a histogram, represented by an array of heights of bars. Each bar's width is 1. The maximum area can be computed by examining potential rectangles that can be formed with each bar of the histogram as the smallest (limiting) height.

### **🚀 Approach**  
To efficiently find the largest rectangle, we can utilize a stack to keep track of the indices of the histogram bars. The main idea is to iterate through the heights, and whenever we find a bar that is shorter than the bar represented by the index at the top of the stack, we calculate the area of rectangles using the height of the bar at that index as the smallest height. 

1. When inspecting each bar:
   - If its height is greater than or equal to the height of the bar at the index on the top of the stack, push its index onto the stack.
   - If its height is less than that of the bar indexed by the top of the stack, pop from the stack, calculate the rectangle's area using the popped index height as the smallest height, and determine its width as the difference between the current index and the index at the new top of the stack minus one.
   
2. Continue this until all bars have been processed. Finally, ensure any remaining indices in the stack are processed to calculate potential areas.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of bars in the histogram since each bar is pushed and popped from the stack at most once.
- **Space Complexity**: O(n), in the worst case, for the stack in case the histogram is strictly increasing.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        stack<int> indices;
        int maxArea = 0;
        int n = heights.size();
        
        for (int i = 0; i < n; i++) {
            while (!indices.empty() && heights[indices.top()] > heights[i]) {
                int height = heights[indices.top()];
                indices.pop();
                int width = indices.empty() ? i : i - indices.top() - 1;
                maxArea = max(maxArea, height * width);
            }
            indices.push(i);
        }
        
        while (!indices.empty()) {
            int height = heights[indices.top()];
            indices.pop();
            int width = indices.empty() ? n : n - indices.top() - 1;
            maxArea = max(maxArea, height * width);
        }
        
        return maxArea;
    }
};  
```
