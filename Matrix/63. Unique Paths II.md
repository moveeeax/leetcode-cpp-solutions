# **LeetCode Unique Paths II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem presents a grid in which we need to find the number of unique paths from the top-left corner to the bottom-right corner. However, there are obstacles in the grid represented by `1`s, and we cannot traverse through these cells. The cells that are free to move through are represented by `0`s. Our task is to compute all possible unique paths when considering these obstacles.

### **🚀 Approach**  
To solve this problem, we can use dynamic programming. We'll create a 2D dp array where `dp[i][j]` will represent the number of unique paths to cell `(i, j)`. The basic idea is:
1. If the current cell is an obstacle (i.e., `grid[i][j] == 1`), set `dp[i][j] = 0`, meaning no paths can pass through this cell.
2. If it's the top-left cell `(0, 0)`, initialize it to `1` if it’s not an obstacle (if `grid[0][0] == 0`).
3. For each cell in the grid, if it's not an obstacle, calculate the number of unique paths to this cell as the sum of unique paths to the cell directly above it and the cell directly to the left:  
   - `dp[i][j] = dp[i-1][j] + dp[i][j-1]`
4. The result will be found in `dp[m-1][n-1]` where `m` is the number of rows, and `n` is the number of columns.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(m * n), where m is the number of rows and n is the number of columns in the grid. This is because we traverse every cell exactly once.
- **Space Complexity**: O(m * n) for the dp array used to store the counts of unique paths. We can also optimize the space complexity to O(n) if we only keep the current and previous rows.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int uniquePathsWithObstacles(vector<vector<int>>& obstacleGrid) {
        int m = obstacleGrid.size();
        int n = obstacleGrid[0].size();

        // If the starting or ending cell is an obstacle, there are no unique paths
        if (obstacleGrid[0][0] == 1 || obstacleGrid[m-1][n-1] == 1)
            return 0;
        
        // Initialize dp array
        vector<vector<int>> dp(m, vector<int>(n, 0));
        dp[0][0] = 1;  // Starting point

        // Fill the dp array
        for (int i = 0; i < m; ++i) {
            for (int j = 0; j < n; ++j) {
                // If current cell is an obstacle
                if (obstacleGrid[i][j] == 1) {
                    dp[i][j] = 0;
                } else {
                    // Sum paths from top and left cell
                    if (i > 0) {
                        dp[i][j] += dp[i-1][j];
                    }
                    if (j > 0) {
                        dp[i][j] += dp[i][j-1];
                    }
                }
            }
        }
        
        return dp[m-1][n-1];  // Return paths to bottom-right corner
    }
};  
```
