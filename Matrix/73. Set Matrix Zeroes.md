# **Set Matrix Zeroes**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem asks us to modify a given m x n matrix by setting entire rows and columns to zero if any element in that row or column is zero. This must be done in-place, meaning we cannot use extra space proportional to the size of the matrix (i.e., we cannot use another matrix).

### **🚀 Approach**  
A straightforward approach would be to iterate through the entire matrix and mark positions that should be set to zero. However, we can achieve this more efficiently by using the first row and the first column of the matrix itself to keep track of whether a row or column should be zeroed.

1. **Variable Flags**: First, determine if the first row and first column should be zeroed. We'll use two boolean flags for this (`row_zero` for the first row and `col_zero` for the first column).
  
2. **Mark Zeros**: Traverse the rest of the matrix (excluding the first row and column), and if a zero is found at position (i, j), we mark the start of the respective row and column (i.e., set the corresponding elements in the first row and first column to zero).

3. **Set Zeros**: Finally, iterate through the matrix again. For each cell (i, j) (excluding the first row and first column), if the first row at (0, j) or the first column at (i, 0) is zero, set the cell to zero.

4. **Handle First Row and First Column**: Lastly, if `row_zero` is true, set the entire first row to zero, and similarly for `col_zero` with the first column.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(m * n), where m is the number of rows and n is the number of columns. We traverse the matrix a constant number of times.
- **Space Complexity**: O(1), as we are using the input matrix itself for storage.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size();
        bool row_zero = false;
        bool col_zero = false;

        // Check if first row needs to be zeroed
        for (int j = 0; j < n; j++) {
            if (matrix[0][j] == 0) {
                row_zero = true;
                break;
            }
        }

        // Check if first column needs to be zeroed
        for (int i = 0; i < m; i++) {
            if (matrix[i][0] == 0) {
                col_zero = true;
                break;
            }
        }

        // Use the first row and first column to mark zeroes
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }

        // Set matrix cells to zero based on marks
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
        }

        // Handle first row
        if (row_zero) {
            for (int j = 0; j < n; j++) {
                matrix[0][j] = 0;
            }
        }

        // Handle first column
        if (col_zero) {
            for (int i = 0; i < m; i++) {
                matrix[i][0] = 0;
            }
        }
    }
};
```  
