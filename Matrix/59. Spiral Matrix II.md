# **Spiral Matrix II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to generate an `n x n` matrix filled with numbers from `1` to `n^2` in a spiral order. The spiral filling begins from the top-left corner of the matrix and proceeds right, downward, left, and then upward in a repeating fashion until all numbers are added to the matrix. 

### **🚀 Approach**  
We can accomplish this using a directed approach to traverse the matrix. We maintain boundaries (top, bottom, left, right) and incrementally fill the matrix based on our spiral movement. Here are the steps:
1. We start by initializing an `n x n` matrix with zeros.
2. Define boundaries: `top`, `bottom`, `left`, and `right` to mark the current spiral round.
3. Use a loop to fill numbers in a spiral manner:
   - Move from left to right across the top boundary and then increment the `top` boundary.
   - Move from top to bottom down the right boundary and then decrement the `right` boundary.
   - Move from right to left across the bottom boundary and then decrement the `bottom` boundary.
   - Move from bottom to top up the left boundary if still within limits and then increment the `left` boundary.
4. Continue this until the number of filled elements reaches `n^2`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n^2), since we fill each cell exactly once.  
- **Space Complexity**: O(1), we only utilize a constant amount of extra space regardless of input size as the matrix is created and returned directly.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<int>> generateMatrix(int n) {
        vector<vector<int>> matrix(n, vector<int>(n, 0));
        int left = 0, right = n - 1, top = 0, bottom = n - 1;
        int num = 1;

        while (left <= right && top <= bottom) {
            // Fill top row
            for (int i = left; i <= right; ++i) {
                matrix[top][i] = num++;
            }
            top++;

            // Fill right column
            for (int i = top; i <= bottom; ++i) {
                matrix[i][right] = num++;
            }
            right--;

            if (top <= bottom) {
                // Fill bottom row
                for (int i = right; i >= left; --i) {
                    matrix[bottom][i] = num++;
                }
                bottom--;
            }

            if (left <= right) {
                // Fill left column
                for (int i = bottom; i >= top; --i) {
                    matrix[i][left] = num++;
                }
                left++;
            }
        }
        return matrix;
    }
};  
```
