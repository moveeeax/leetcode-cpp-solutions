# **LeetCode Minimum Path Sum**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
We need to find the path from the top-left corner to the bottom-right corner of a grid containing non-negative integers, where the path can only move down or right. The goal is to minimize the sum of the numbers along the path. This problem can be approached using dynamic programming. 

### **🚀 Approach**  
We'll create a dynamic programming table `dp` where `dp[i][j]` represents the minimum path sum to reach the cell `(i, j)`. Here are the steps for the approach: 
1. **Initialization:** Start by setting `dp[0][0]` to `grid[0][0]` since that's the starting point.
2. **First Row and First Column:** Fill in the first row and the first column since they can only be reached from one direction (either from the left for the first row or from above for the first column).
3. **Filling the DP Table:** For each remaining cell, calculate the minimum path sum to it by taking the minimum of coming from the top or from the left and adding the current cell's value.
4. **Result:** The result will be found at `dp[m-1][n-1]`, where `m` is the number of rows and `n` is the number of columns in the grid.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(m * n), where m is the number of rows and n is the number of columns in the grid. We process each cell once.
- **Space Complexity**: O(m * n) for the DP table, but we can optimize this to O(n) by using only a single row to store the results of the previous row.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int minPathSum(vector<vector<int>>& grid) {
        if (grid.empty() || grid[0].empty()) return 0;
        int m = grid.size(), n = grid[0].size();
        
        vector<int> dp(n, 0);
        
        for (int i = 0; i < m; ++i) {
            for (int j = 0; j < n; ++j) {
                if (i == 0 && j == 0) {
                    dp[j] = grid[i][j];  // Start point
                } else if (i == 0) {
                    dp[j] = dp[j - 1] + grid[i][j];  // First row
                } else if (j == 0) {
                    dp[j] = dp[j] + grid[i][j];  // First column
                } else {
                    dp[j] = min(dp[j], dp[j - 1]) + grid[i][j];  // Min of top or left
                }
            }
        }
        
        return dp[n - 1];  // Bottom-right corner
    }
};  
```
