# **Reverse Nodes in k-Group**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The task is to reverse nodes in a linked list in groups of size `k`. If the number of nodes is less than `k`, we leave them as they are. The challenge is to achieve this in-place without altering the order of the nodes outside the reversed sections.

### **🚀 Approach**  
1. **Count Nodes**: First, traverse the linked list to count its total number of nodes. This helps us determine how many groups of `k` nodes can be reversed.
2. **Reverse in Groups**: Next, we use a loop to handle each group of `k` nodes:
   - For each group, perform the reversal of nodes. We need to maintain the pointers to ensure the rest of the list remains connected correctly.
3. **Handle the Remaining Nodes**: If there are fewer than `k` nodes remaining after the last complete group, keep them as they are without reversing.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) - We traverse the list a couple of times, but each node is processed a limited number of times.
- **Space Complexity**: O(1) - The reversal is done in-place with only a few pointers used for navigation.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    ListNode* reverseKGroup(ListNode* head, int k) {
        ListNode dummy(0);
        dummy.next = head;
        ListNode* prevGroupEnd = &dummy;
        
        while (true) {
            ListNode* kth = prevGroupEnd;
            for (int i = 0; i < k; ++i) {
                kth = kth->next;
                if (!kth) return dummy.next;  // Less than k nodes remaining
            }
            ListNode* groupStart = prevGroupEnd->next;
            ListNode* groupEnd = kth;
            ListNode* prev = groupEnd->next;
            ListNode* curr = groupStart;
            
            // Reverse the group
            while (curr != groupEnd) {
                ListNode* next = curr->next;
                curr->next = prev;
                prev = curr;
                curr = next;
            }
            // Adjust the links after reversing
            prevGroupEnd->next = groupEnd;
            groupStart->next = curr;
            prevGroupEnd = groupStart;  // Move prevGroupEnd to the end of the reversed group
        }
        return dummy.next;
    }
};
```

// Function to create a linked list from an array
ListNode* createList(std::vector<int> vals) {
    ListNode* head = nullptr;
    ListNode* tail = nullptr;
    for (int val : vals) {
        ListNode* node = new ListNode(val);
        if (!head) {
            head = node;
            tail = head;
        } else {
            tail->next = node;
            tail = tail->next;
        }
    }
    return head;
}

// Function to print the linked list
void printList(ListNode* head) {
    while (head) {
        std::cout << head->val << " -> ";
        head = head->next;
    }
    std::cout << "nullptr" << std::endl;
}

int main() {
    Solution solution;
    ListNode* head = createList({1, 2, 3, 4, 5});
    int k = 2;
    ListNode* newHead = solution.reverseKGroup(head, k);
    printList(newHead);  // Output should be the modified linked list
    return 0;
}
