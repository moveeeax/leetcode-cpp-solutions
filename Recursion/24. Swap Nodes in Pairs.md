# **LeetCode Swap Nodes in Pairs**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to swap every two adjacent nodes in a linked list. For a linked list: 1 -> 2 -> 3 -> 4, the resulting linked list after swapping should be: 2 -> 1 -> 4 -> 3. 

To achieve this, we need a method to traverse the linked list, swapping pairs of nodes as we go.

### **🚀 Approach**  
1. **Initialization**: We start by creating a dummy node that points to the head of the list. This dummy node helps manage edge cases (like when the list has less than two nodes) easily.
2. **Iterate**: Use a pointer to traverse the linked list in pairs:
   - For each pair of nodes (first and second), perform the swap.
   - Adjust the `next` pointers to ensure the links between swapped nodes and the rest of the list are maintained.
3. **Continue**: Move to the next pair of nodes and repeat until the end of the list is reached.
4. **Return**: Finally, return the next of the dummy node which points to the new head of the modified list.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the linked list, as we traverse the list once.  
- **Space Complexity**: O(1), since we are only using a few pointers for manipulation and not using any additional data structures that grow with the input size.

## **2️⃣ LeetCode Submission**  
```cpp
struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(NULL) {}
};

class Solution {
public:
    ListNode* swapPairs(ListNode* head) {
        ListNode dummy(0);
        dummy.next = head;
        ListNode* prev = &dummy;

        while (prev->next && prev->next->next) {
            ListNode* first = prev->next;
            ListNode* second = first->next;

            // Swapping
            first->next = second->next;
            second->next = first;
            prev->next = second;

            // Move to the next pair
            prev = first;
        }
        
        return dummy.next;
    }
};
```

#include <iostream>
using namespace std;

struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(NULL) {}
};

class Solution {
public:
    ListNode* swapPairs(ListNode* head) {
        ListNode dummy(0);
        dummy.next = head;
        ListNode* prev = &dummy;

        while (prev->next && prev->next->next) {
            ListNode* first = prev->next;
            ListNode* second = first->next;

            // Swapping
            first->next = second->next;
            second->next = first;
            prev->next = second;

            // Move to the next pair
            prev = first;
        }
        
        return dummy.next;
    }
};

int main() {
    Solution solution;

    // Creating a test linked list: 1 -> 2 -> 3 -> 4
    ListNode* head = new ListNode(1);
    head->next = new ListNode(2);
    head->next->next = new ListNode(3);
    head->next->next->next = new ListNode(4);

    // Swapping pairs
    ListNode* newHead = solution.swapPairs(head);

    // Printing the result
    ListNode* node = newHead;
    while (node) {
        cout << node->val << " ";
        node = node->next;
    }

    return 0;
}
