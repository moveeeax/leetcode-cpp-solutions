# **LeetCode Group Anagrams**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The task is to group a list of strings into anagrams. An anagram is a word formed by rearranging the letters of another word, for example, "eat" and "tea". This means we need to find all strings that, when sorted, result in the same sequence of characters. The positioning of letters doesn't matter; what matters is the frequency and occurrence of each character.

### **🚀 Approach**  
To solve this problem:
1. **Sorting Approach**: We can sort each string, and use the sorted string as a key in a hashmap (or dictionary). All original strings that turn into the same sorted string are grouped together.  
2. **Using a Map**: We will iterate over each string, sort it, and then append the original string to the list in the hashmap keyed by the sorted string.
3. **Final Grouping**: The result will be the values of the hashmap, which are lists of grouped anagrams.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n * k log k), where `n` is the number of strings, and `k` is the maximum length of a string. Sorting each string takes O(k log k).
- **Space Complexity**: O(n * k) to store the grouped anagrams.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        unordered_map<string, vector<string>> anagrams;
        for (string str : strs) {
            string sortedStr = str;
            sort(sortedStr.begin(), sortedStr.end());
            anagrams[sortedStr].push_back(str);
        }
        vector<vector<string>> result;
        for (auto& pair : anagrams) {
            result.push_back(pair.second);
        }
        return result;
    }
};  
```

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <unordered_map>
#include <algorithm>

using namespace std;

class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        unordered_map<string, vector<string>> anagrams;
        for (string str : strs) {
            string sortedStr = str;
            sort(sortedStr.begin(), sortedStr.end());
            anagrams[sortedStr].push_back(str);
        }
        vector<vector<string>> result;
        for (auto& pair : anagrams) {
            result.push_back(pair.second);
        }
        return result;
    }
};

int main() {
    Solution solution;
    vector<string> input = {"eat", "tea", "tan", "ate", "nat", "bat"};
    vector<vector<string>> output = solution.groupAnagrams(input);
    
    for (const auto& group : output) {
        cout << "[ ";
        for (const auto& str : group) {
            cout << str << " ";
        }
        cout << "] ";
    }
    return 0;
}
```
