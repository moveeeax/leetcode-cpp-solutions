# **LeetCode Merge Intervals**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to merge overlapping intervals from a given list of intervals. Each interval is represented by a pair of integers, where the first integer is the start and the second integer is the end. If two intervals overlap, we need to merge them to form a single interval. 

For example, given the intervals `[1,3]` and `[2,4]`, they overlap since the end of the first interval (3) is greater than the start of the second interval (2). Therefore, they should be merged into `[1,4]`.

### **🚀 Approach**  
1. **Sorting the Intervals**: Start by sorting the intervals based on their starting times. This helps in easily identifying overlaps since if intervals overlap, they will be adjacent in the sorted array.
2. **Merging**: Iterate through the sorted intervals and keep track of the current interval being constructed. If the next interval overlaps with this current interval, we update the end of the current interval to the maximum end of the two overlapping intervals. If there is no overlap, we will add the current interval to the result and move on to the next interval.
  
This algorithm uses a greedy approach where by sorting and then merging consecutively, we ensure all overlaps are handled efficiently.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n log n) due to sorting the intervals, where n is the number of intervals.
- **Space Complexity**: O(n) for storing the merged intervals in the result.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        if (intervals.empty()) return {};
        
        // Sort intervals by the starting times
        sort(intervals.begin(), intervals.end());
        
        vector<vector<int>> merged;
        vector<int> current = intervals[0];
        
        for (const auto& interval : intervals) {
            // If there's an overlap, merge intervals
            if (interval[0] <= current[1]) {
                current[1] = max(current[1], interval[1]); // Update the end
            } else {
                // No overlap, add the current interval to result
                merged.push_back(current);
                current = interval; // Move to the next interval
            }
        }
        
        merged.push_back(current); // Add the last merged interval
        return merged;
    }
};  
```
