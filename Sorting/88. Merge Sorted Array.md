# **LeetCode Merge Sorted Array**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires merging two sorted arrays into one sorted array. We are provided with two input arrays, where the first array has enough space allocated to hold the combined size of both arrays. The objective is to efficiently combine both arrays while maintaining sorted order.

### **🚀 Approach**  
1. **Understanding Input**: You have two arrays: `nums1` (which will contain the merged result) and `nums2`. The length of `nums1` is initially `m + n`, where the first `m` elements are sorted from `nums1` and the last `n` remaining spots are to be filled with elements from `nums2`.
  
2. **Two-pointer Technique**: We'll utilize two pointers, one for each array (`i` for `nums1` and `j` for `nums2`), and a third pointer `k` to keep track of the position in `nums1` where the next largest element should be placed.

3. **Merge Process**:
   - Start from the end of both sorted arrays, comparing the last elements.
   - Place the larger element at the end of `nums1` (position `m + n - 1`), and move the respective pointer.
   - Repeat until all elements from `nums2` are merged into `nums1`.

4. **Completion**: If there are remaining elements in `nums2` after exiting the loop, copy them over to `nums1` since `nums1` may already have the correct sorted order in its right position.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(m + n), where `m` is the number of elements in `nums1` and `n` is the number of elements in `nums2`. This is due to the single pass through both arrays.
- **Space Complexity**: O(1). We are merging in place and not using any additional data structures.

## **2️⃣ LeetCode Submission**  
```cpp
class MergeSortedArray {
public:
    void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) {
        int i = m - 1; // pointer for nums1
        int j = n - 1; // pointer for nums2
        int k = m + n - 1; // pointer for the last position in nums1

        while (i >= 0 && j >= 0) {
            if (nums1[i] > nums2[j]) {
                nums1[k--] = nums1[i--];
            } else {
                nums1[k--] = nums2[j--];
            }
        }
        
        // If there are remaining elements in nums2, move them
        while (j >= 0) {
            nums1[k--] = nums2[j--];
        }
    }
};
```
