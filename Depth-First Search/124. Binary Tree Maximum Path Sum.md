# **Binary Tree Maximum Path Sum**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem asks us to find the maximum sum of all possible paths in a binary tree. A path can start and end at any node in the tree and can travel through parent-child relationships. The challenge is to ensure that we consider only valid paths, which means we will need to search for the best sums recursively while properly managing the contributions at each node.

### **🚀 Approach**  
The approach involves recursively calculating the maximum path sums from the bottom of the tree upwards (post-order traversal). At each node, we consider two scenarios:
1. The maximum path that includes both subtrees (left and right) and the current node.
2. The maximum path that can continue to its parent from either the left or right subtree.

For every node, we calculate the maximum contributions from its children and decide if including both children and the node itself could yield a better path sum. The final result is maintained globally (as a class member) to track the highest sum encountered throughout the tree traversal.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the tree. Each node is visited once.  
- **Space Complexity**: O(h), where h is the height of the tree, due to recursive stack space. In the worst case (skewed tree), this can be O(n).

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int maxPathSum(TreeNode* root) {
        int maxSum = INT_MIN;
        maxPathSumHelper(root, maxSum);
        return maxSum;
    }

    int maxPathSumHelper(TreeNode* node, int& maxSum) {
        if (!node) return 0;

        // Recursive calls for left and right children
        int leftMax = max(maxPathSumHelper(node->left, maxSum), 0);
        int rightMax = max(maxPathSumHelper(node->right, maxSum), 0);

        // Current max including the node and both children
        int currentMax = node->val + leftMax + rightMax;
        maxSum = max(maxSum, currentMax);

        // Return max path sum passing either left or right child
        return node->val + max(leftMax, rightMax);
    }
};
```

```cpp
#include <iostream>
#include <climits>
using namespace std;

// Definition for a binary tree node.
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
};

class Solution {
public:
    int maxPathSum(TreeNode* root) {
        int maxSum = INT_MIN;
        maxPathSumHelper(root, maxSum);
        return maxSum;
    }

    int maxPathSumHelper(TreeNode* node, int& maxSum) {
        if (!node) return 0;

        int leftMax = max(maxPathSumHelper(node->left, maxSum), 0);
        int rightMax = max(maxPathSumHelper(node->right, maxSum), 0);

        int currentMax = node->val + leftMax + rightMax;
        maxSum = max(maxSum, currentMax);

        return node->val + max(leftMax, rightMax);
    }
};

int main() {
    Solution solution;

    // Create a sample tree: 
    //        -10
    //       /    \
    //      9      20
    //            /  \
    //           15   7
    TreeNode *root = new TreeNode(-10);
    root->left = new TreeNode(9);
    root->right = new TreeNode(20);
    root->right->left = new TreeNode(15);
    root->right->right = new TreeNode(7);

    // Calculate maximum path sum
    int result = solution.maxPathSum(root);
    cout << "Maximum Path Sum: " << result << endl; // Expected Output: 42

    return 0;
}
```
