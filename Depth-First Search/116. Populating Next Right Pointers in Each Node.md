# **Populating Next Right Pointers in Each Node**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to connect all nodes at the same level in a perfect binary tree with their next right pointers, which can be set to `NULL` if there is no next right node. A perfect binary tree is where every internal node has two children and all leaf nodes are at the same level.

We need to traverse the tree level by level and set the `next` pointer for every node to its immediate right node on the same level. This should be done efficiently without using extra space for storing the nodes.

### **🚀 Approach**  
We can utilize a level-order traversal approach using helper pointers to avoid using additional space for storing nodes. The idea is as follows:

1. Begin at the root of the tree and set the current level pointer to the root.
2. For each level, set the `next` pointers for all nodes:
   - Start with the leftmost node at the current level and use another pointer to traverse through the nodes from left to right.
   - As we connect the current node to its next right node, move to the next node at the current level, and repeat until all nodes at that level are processed.
3. Continue this until all levels of the tree are processed.

The implementation will involve updating the `next` pointers for all nodes in the tree structure directly.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the tree. Each node is visited exactly once.
- **Space Complexity**: O(1), since we are not using any extra space that grows with the input size; we are modifying the tree in place.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    void connect(Node* root) {
        if (!root) return;

        Node* current = root;
        while (current->left) {
            Node* nextLevel = current->left;

            Node* node = current;
            while (node) {
                node->left->next = node->right;
                if (node->next) {
                    node->right->next = node->next->left;
                }
                node = node->next;
            }
            current = nextLevel;
        }
    }
};
```

            Node* nextLevel = current->left;

            Node* node = current;
            while (node) {
                node->left->next = node->right;

            }
            current = nextLevel;
        }
    }
};

int main() {
