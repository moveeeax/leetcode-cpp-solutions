# **LeetCode Path Sum**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to determine whether the sum of the values along any path from the root node to a leaf node in a binary tree equals a specified target sum. A "path" is defined as a sequence of nodes starting at the root and proceeding downward to any leaf node, without traversing upwards. A leaf node is a node that does not have any children. Our goal is to check if there exists at least one such path where the sum of the values along the path matches the target value.

### **🚀 Approach**  
We will utilize a depth-first search (DFS) approach to explore all paths in the binary tree. Starting from the root, we will traverse left and right children recursively, maintaining a running total of the current path sum. At each node, if we reach a leaf node, we will check if the current path sum matches the target sum. Here’s a step-by-step breakdown of the solution:

1. **Base Case**: If the current node is null, we return false as there is no path through this node.
2. If the current node is a leaf (i.e., both left and right children are null):
   - We check if the sum of the current path equals the target sum. If it does, return true.
3. **Recursive Calls**: Otherwise, we recursively call the function for the left and right children of the current node, subtracting the value of the current node from the target sum.
4. Finally, return true if either of the recursive calls returns true, indicating that a valid path has been found.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the binary tree, since we may visit each node exactly once.
- **Space Complexity**: O(h), where h is the height of the binary tree, which is the maximum depth of the recursion stack.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool hasPathSum(TreeNode* root, int targetSum) {
        if (!root) return false;

        // Check if we reached a leaf node
        if (!root->left && !root->right) {
            return root->val == targetSum;
        }

        // Subtract current node's value from the target sum and recurse on left and right child
        targetSum -= root->val;
        return hasPathSum(root->left, targetSum) || hasPathSum(root->right, targetSum);
    }
};
```

```cpp
#include <iostream>
using namespace std;

// Definition for a binary tree node.
struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
};

class Solution {
public:
    bool hasPathSum(TreeNode* root, int targetSum) {
        if (!root) return false;

        if (!root->left && !root->right) {
            return root->val == targetSum;
        }

        targetSum -= root->val;
        return hasPathSum(root->left, targetSum) || hasPathSum(root->right, targetSum);
    }
};

int main() {
    Solution sol;
    
    // Constructing the binary tree as an example
    TreeNode* root = new TreeNode(5);
    root->left = new TreeNode(4);
    root->right = new TreeNode(8);
    root->left->left = new TreeNode(11);
    root->left->left->left = new TreeNode(7);
    root->left->left->right = new TreeNode(2);
    root->right->left = new TreeNode(13);
    root->right->right = new TreeNode(4);
    root->right->right->right = new TreeNode(1);

    int targetSum = 22;
    cout << (sol.hasPathSum(root, targetSum) ? "True" : "False") << endl;

    return 0;
}
```
