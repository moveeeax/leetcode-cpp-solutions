# **LeetCode Maximum Depth of Binary Tree**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires calculating the maximum depth of a binary tree. The depth of a binary tree is defined as the number of nodes along the longest path from the root node down to the farthest leaf node. If the tree is empty, the depth is zero. This can be visualized as counting how many nodes are present from the root to the deepest leaf.

To solve this problem, we can use a recursive approach. The base case for our recursion will be when the current node is `null`, in which case we will return a depth of `0`. Otherwise, we determine the maximum depth by comparing the depths of the left and right subtrees and adding one for the current node.

### **🚀 Approach**  
1. **Base Case**: If the current node is `null`, return depth `0`.
2. **Recursive Case**: If the current node is not `null`, recursively compute the depth of the left subtree and the right subtree.
3. **Combine Results**: The maximum depth from the current node will be `1 + max(depth of left subtree, depth of right subtree)`.
4. **Return the result**.

By following this systematic, recursive strategy, we efficiently calculate the maximum depth of the tree.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the binary tree. We need to visit each node once.
- **Space Complexity**: O(h), where h is the height of the tree. This accounts for the space used in the call stack due to recursion.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if (!root) return 0;
        return 1 + max(maxDepth(root->left), maxDepth(root->right));
    }
};
```

```cpp
#include <iostream>
using namespace std;

// Definition for a binary tree node.
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    int maxDepth(TreeNode* root) {
        if (!root) return 0;
        return 1 + max(maxDepth(root->left), maxDepth(root->right));
    }
};

int main() {
    // Create the tree:
    //      3
    //     / \
    //    9  20
    //       /  \
    //      15   7
    TreeNode* root = new TreeNode(3);
    root->left = new TreeNode(9);
    root->right = new TreeNode(20);
    root->right->left = new TreeNode(15);
    root->right->right = new TreeNode(7);
    
    Solution solution;
    cout << "Maximum depth of binary tree: " << solution.maxDepth(root) << endl;

    // Clean up memory (delete the nodes created)
    delete root->right->right;
    delete root->right->left;
    delete root->right;
    delete root->left;
    delete root;

    return 0;
}
```
