# **LeetCode Validate Binary Search Tree**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
To determine if a binary tree is a valid Binary Search Tree (BST), we must ensure every node follows the BST properties. In a valid BST:  
1. The left subtree of a node contains only nodes with keys less than the node’s key.
2. The right subtree of a node contains only nodes with keys greater than the node’s key.
3. Both the left and right subtrees must also be BSTs.  

### **🚀 Approach**  
We can approach this problem using recursive depth-first traversal. We will maintain an allowable range for each node based on its position in the BST:  
1. We begin with the full range of valid values for each node (i.e., negative infinity to positive infinity).
2. For each node visited, we check whether its value lies within the given range.
3. Recursively check the left child (new upper limit) and the right child (new lower limit) by adjusting the allowed range based on the value of the current node.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the tree, because we visit each node once.  
- **Space Complexity**: O(h), where h is the height of the tree, due to recursive stack space. In the worst case of an unbalanced tree, this can reach O(n).

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool isValidBST(TreeNode* root) {
        return validate(root, LONG_MIN, LONG_MAX);
    }
    
    bool validate(TreeNode* node, long lower, long upper) {
        if (!node) return true; // An empty node is a valid BST
        if (node->val <= lower || node->val >= upper) return false; // Violate BST property
        
        // Recursively validate left and right subtrees
        return validate(node->left, lower, node->val) && validate(node->right, node->val, upper);
    }
};
```

```cpp
#include <iostream>
#include <limits.h>

using namespace std;

// Definition for a binary tree node.
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
};

class Solution {
public:
    bool isValidBST(TreeNode* root) {
        return validate(root, LONG_MIN, LONG_MAX);
    }
    
    bool validate(TreeNode* node, long lower, long upper) {
        if (!node) return true; // An empty node is a valid BST
        if (node->val <= lower || node->val >= upper) return false; // Violate BST property
        
        // Recursively validate left and right subtrees
        return validate(node->left, lower, node->val) && validate(node->right, node->val, upper);
    }
};

int main() {
    Solution solution;

    // Example Test Case
    TreeNode* root = new TreeNode(2);
    root->left = new TreeNode(1);
    root->right = new TreeNode(3);
    
    cout << (solution.isValidBST(root) ? "True" : "False") << endl; // Expected: True

    // Clean up memory (deleting nodes)
    delete root->left;
    delete root->right;
    delete root;

    return 0;
}  
```
