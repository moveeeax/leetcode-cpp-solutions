# **LeetCode Surrounded Regions**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem is centered around identifying and marking "surrounded regions" in a 2D board of 'X's and 'O's. The objective is to replace 'O's that are completely surrounded by 'X's with 'X's, while 'O's that are on the borders or connected to the borders via other 'O's should remain unchanged. The challenge lies in correctly determining which 'O's can be considered surrounded by 'X's and thus should be converted.

### **🚀 Approach**  
1. **Identify Non-Surrounded 'O's**: First, we must locate all the 'O's that cannot be surrounded. This includes any 'O's on the border of the board. 
2. **DFS/BFS Traversal**: Starting from all border 'O's, we will use Depth-First Search (DFS) or Breadth-First Search (BFS) to traverse all connected 'O's. Mark these 'O's as non-surrounded (we can change them to a placeholder, like 'E').
3. **Replace**: After marking, the next step will be to traverse the board again:
    - Convert all 'O's that remain unmarked to 'X', as they are surrounded.
    - Convert the marked 'E's back to 'O's since they are not surrounded.
4. **Output the Modified Board**: Finally, return or print the modified board.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(m * n), where m is the number of rows and n is the number of columns in the board. This is because we might need to traverse every cell in the worst case.
- **Space Complexity**: O(m * n) in the worst case for the stack space if DFS or the queue space if BFS is used.

## **2️⃣ LeetCode Submission**  
```cpp
class SurroundedRegions {
public:
    void solve(vector<vector<char>>& board) {
        if (board.empty()) return;

        int rows = board.size();
        int cols = board[0].size();

        // Use DFS to mark non-surrounded 'O's
        for (int i = 0; i < rows; i++) {
            if (board[i][0] == 'O') dfs(board, i, 0);
            if (board[i][cols - 1] == 'O') dfs(board, i, cols - 1);
        }
        for (int j = 0; j < cols; j++) {
            if (board[0][j] == 'O') dfs(board, 0, j);
            if (board[rows - 1][j] == 'O') dfs(board, rows - 1, j);
        }

        // Flip the surrounded 'O's and revert the non-surrounded 'E's
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                if (board[i][j] == 'O') {
                    board[i][j] = 'X';
                }
                else if (board[i][j] == 'E') {
                    board[i][j] = 'O';
                }
            }
        }
    }

private:
    void dfs(vector<vector<char>>& board, int i, int j) {
        if (i < 0 || i >= board.size() || j < 0 || j >= board[0].size() || board[i][j] != 'O') {
            return;
        }
        board[i][j] = 'E'; // Mark as non-surrounded
        dfs(board, i + 1, j);
        dfs(board, i - 1, j);
        dfs(board, i, j + 1);
        dfs(board, i, j - 1);
    }
};
```
