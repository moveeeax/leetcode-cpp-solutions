# **LeetCode Same Tree**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem is about checking whether two binary trees are identical. Two binary trees are considered identical if they have the same structure and the same node values at each corresponding position. We can visualize binary trees with nodes containing values, where each node has left and right pointers leading to further nodes. The simplest case is when both trees are `NULL`, in which case we can say they are identical.

### **🚀 Approach**  
To determine if the two binary trees are the same, we can use a recursive approach. The recursion will inspect each corresponding node in both trees to ensure they match:

1. **Base Case**:
   - If both nodes are `NULL`, return `true`.
   - If one node is `NULL` and the other is not, return `false`.
   
2. **Value Check**:
   - If the values of both nodes are not equal, return `false`.

3. **Recursion**:
   - Recursively check the left subtree and the right subtree of both nodes. If both subtrees are identical, then the trees are identical.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) - where n is the number of nodes in the trees, since we potentially need to check each node.
- **Space Complexity**: O(h) - where h is the height of the tree due to recursion stack (in the worst case, it could be O(n) for an unbalanced tree).

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool isSameTree(TreeNode* p, TreeNode* q) {
        // If both nodes are null, trees are identical at this node
        if (!p && !q) return true;
        // If one is null and the other is not, trees are not identical
        if (!p || !q) return false;
        // Check the value of the current nodes and recurse for left and right children
        return (p->val == q->val) && 
               isSameTree(p->left, q->left) && 
               isSameTree(p->right, q->right);
    }
};
```

## **📝 Full Working Code (Local Execution)**  
```cpp
#include <iostream>

struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
};

class Solution {
public:
    bool isSameTree(TreeNode* p, TreeNode* q) {
        if (!p && !q) return true;
        if (!p || !q) return false;
        return (p->val == q->val) && 
               isSameTree(p->left, q->left) && 
               isSameTree(p->right, q->right);
    }
};

int main() {
    Solution solution;
    // Create test trees and call solution.isSameTree(...)
    TreeNode *tree1 = new TreeNode(1);
    tree1->left = new TreeNode(2);
    tree1->right = new TreeNode(3);

    TreeNode *tree2 = new TreeNode(1);
    tree2->left = new TreeNode(2);
    tree2->right = new TreeNode(3);
    
    bool result = solution.isSameTree(tree1, tree2);
    
    std::cout << "Are the two trees the same? " << (result ? "Yes" : "No") << std::endl;

    // Clean up
    delete tree1->left;
    delete tree1->right;
    delete tree1;

    delete tree2->left;
    delete tree2->right;
    delete tree2;

    return 0;
}
```
