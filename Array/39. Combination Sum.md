# **LeetCode Combination Sum**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to find all unique combinations of numbers from a given array that can sum up to a target value. We can reuse any number as many times as needed to form these combinations. The key here is managing the combination generation effectively and ensuring that each combination is stored uniquely, even if the same numbers can generate multiple permutations.

### **🚀 Approach**  
To solve this problem, we can use a backtracking approach. The steps involved are as follows:
1. **Backtrack Function**: We'll implement a recursive function that explores potential combinations. This function will take the current combination being formed, the remaining target value, and the starting index to handle duplicates.
2. **Base Case**: If the remaining target becomes zero, we have found a valid combination. Add this combination to the result list.
3. **Exploration**: Iterate over the candidates, allowing for reuse by calling the backtrack function with the same index, allowing infinite instances of each candidate.
4. **Preventing Duplicates**: To avoid duplicate combinations, carefully manage the index while exploring candidates.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(2^(n + t/m)), where `n` is the length of the candidates, `t` is the target sum, and `m` is the minimum of the candidates. The exponential factor arises from the combinatorial nature of forming sums.
- **Space Complexity**: O(t/m), which arises from the space used by the recursion stack and storage of results.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    void backtrack(vector<vector<int>>& result, vector<int>& current, vector<int>& candidates, int target, int start) {
        if (target == 0) {
            result.push_back(current);
            return;
        }
        for (int i = start; i < candidates.size(); i++) {
            if (candidates[i] > target) break; // Early stopping if the candidate exceeds target
            current.push_back(candidates[i]);
            backtrack(result, current, candidates, target - candidates[i], i); // i instead of i + 1 to reuse the same element
            current.pop_back(); // backtrack
        }
    }
    
    vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
        vector<vector<int>> result;
        vector<int> current;
        sort(candidates.begin(), candidates.end()); // Sort for early stopping
        backtrack(result, current, candidates, target, 0);
        return result;
    }
};  
```

        if (target == 0) {
            result.push_back(current);
            return;
        }

        }
    }
    
    vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
        vector<vector<int>> result;
        vector<int> current;
        sort(candidates.begin(), candidates.end()); // Sort for early stopping
        backtrack(result, current, candidates, target, 0);

int main() {
    Solution sol;
    vector<int> candidates = {2, 3, 6, 7};
    int target = 7;
