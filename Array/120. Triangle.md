# **LeetCode Triangle**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to find the minimum path sum from the top of a triangle to the base. Each step can only go to adjacent numbers on the row below. Given the nature of the problem, a dynamic programming approach is suitable. By breaking down the problem into smaller subproblems (the minimum path sums to each position on the triangle), we can build up the solution from the bottom to the top.

### **🚀 Approach**  
To solve this problem, we can use a bottom-up dynamic programming approach. This means:
1. Start from the second last row of the triangle and for each element, calculate the minimum path sum to that element by considering the minimum of the two adjacent elements in the row below.
2. Continue this process until we reach the top of the triangle.
3. The top element will hold the minimum path sum from top to bottom.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of elements in the triangle, since we visit each element once.
- **Space Complexity**: O(1), if we modify the triangle in place to save space.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int minimumTotal(vector<vector<int>>& triangle) {
        int n = triangle.size();
        for (int i = n - 2; i >= 0; --i) {
            for (int j = 0; j <= i; ++j) {
                triangle[i][j] += min(triangle[i + 1][j], triangle[i + 1][j + 1]);
            }
        }
        return triangle[0][0];
    }
};
```

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

class Solution {
public:
    int minimumTotal(vector<vector<int>>& triangle) {
        int n = triangle.size();
        for (int i = n - 2; i >= 0; --i) {
            for (int j = 0; j <= i; ++j) {
                triangle[i][j] += min(triangle[i + 1][j], triangle[i + 1][j + 1]);
            }
        }
        return triangle[0][0];
    }
};

int main() {
    Solution solution;
    vector<vector<int>> triangle = {
        {2},
        {3, 4},
        {6, 5, 7},
        {4, 1, 8, 3}
    };
    int result = solution.minimumTotal(triangle);
    cout << "Minimum path sum: " << result << endl;  // Expected output: 11
    return 0;
}
