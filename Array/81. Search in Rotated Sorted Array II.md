# **Search in Rotated Sorted Array II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to search for a target number in a rotated sorted array that may contain duplicates. Unlike a standard binary search in a strictly sorted array, we must consider the possibility of duplicates, which may influence our search strategy since the standard properties of ordering can be disrupted.

### **🚀 Approach**  
1. **Initialization**: Begin with two pointers, `left` and `right`, which will denote the starting and ending indices of the array, respectively.
2. **Binary Search Pattern**: While the `left` pointer is less than or equal to the `right` pointer:
   - Calculate the middle index `mid`.
   - If the middle element is equal to the target, return true.
   - If there are duplicates at the boundaries (i.e., `nums[left]` is equal to `nums[mid]` and `nums[mid]` is equal to `nums[right]`), we cannot determine which half is sorted, so we reduce our problem size by moving the `left` pointer to `left + 1` or `right` pointer to `right - 1` to skip over the duplicates.
   - If the left half is sorted:
     - Check if the target is within the left half ([`nums[left]`, `nums[mid]`]).
     - If it is, narrow the search to the left half by setting `right = mid - 1`. Otherwise, search in the right half.
   - If the right half is sorted:
     - Check if the target is within the right half ([`nums[mid]`, `nums[right]`]).
     - If it is, narrow the search to the right half by setting `left = mid + 1`. Otherwise, search in the left half.
3. **Termination**: If we exit the loop without finding the target, return false.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) in the worst case due to the potential for duplicates requiring us to traverse the entire array.
- **Space Complexity**: O(1) since we are utilizing a constant amount of extra space.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool search(vector<int>& nums, int target) {
        int left = 0, right = nums.size() - 1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if (nums[mid] == target) {
                return true;
            }
            
            if (nums[left] == nums[mid] && nums[mid] == nums[right]) {
                left++;
                right--;
            } 
            else if (nums[left] <= nums[mid]) {
                if (nums[left] <= target && target < nums[mid]) {
                    right = mid - 1;
                } else {
                    left = mid + 1;
                }
            } 
            else {
                if (nums[mid] < target && target <= nums[right]) {
                    left = mid + 1;
                } else {
                    right = mid - 1;
                }
            }
        }

        return false;
    }
};
```  
