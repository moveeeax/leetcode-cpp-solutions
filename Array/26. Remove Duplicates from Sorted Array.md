# **Remove Duplicates from Sorted Array**

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to modify a sorted array in such a way that all duplicate elements are removed, and only distinct elements are left in the array. The challenge is not only to remove the duplicates but also to ensure that the solution operates in-place, meaning we should use minimal additional space and modify the array itself.

The function should return the new length of the array after removing duplicates, while the first 'k' elements (where 'k' is the new length) should contain the unique values. The rest of the array can be ignored.

### **🚀 Approach**  
To achieve this, we can utilize a two-pointer technique:

1. **Initialize Two Pointers**: One pointer (`i`) will iterate through the array, and another pointer (`j`) will track the position where the next unique element should be placed.
2. **Iterate Through the Array**: As we iterate through the array with pointer `i`, we check if the current element is different from the last unique element found.
3. **Update Array**: When a unique element is found (i.e., `nums[i] != nums[j]`), we place it in the next position represented by `j` and increment `j`.
4. **Return the Length**: Finally, the return value will be `j`, which represents the count of unique elements.

This method runs in linear time and uses constant space, as we are not using extra data structures to store elements.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) where n is the number of elements in the array, since we traverse the array once.
- **Space Complexity**: O(1) because we are modifying the original array in-place without needing extra space proportional to the input size.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        if (nums.empty()) return 0; // Handle empty case

        int j = 0; // Pointer for the location of unique elements
        for (int i = 1; i < nums.size(); i++) {
            if (nums[i] != nums[j]) {
                j++; // Move to next position for unique element
                nums[j] = nums[i]; // Update with the unique value
            }
        }
        return j + 1; // Return number of unique elements
    }
};
```

```cpp
#include <iostream>
#include <vector>

using namespace std;

class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        if (nums.empty()) return 0;

        int j = 0;
        for (int i = 1; i < nums.size(); i++) {
            if (nums[i] != nums[j]) {
                j++;
                nums[j] = nums[i];
            }
        }
        return j + 1;
    }
};

int main() {
    Solution solution;
    vector<int> nums = {0,0,1,1,1,2,2,3,3,4};
    int newLength = solution.removeDuplicates(nums);
    
    cout << "New length of the array: " << newLength << endl;
    cout << "Array after removing duplicates: ";
    for (int i = 0; i < newLength; i++) {
        cout << nums[i] << " ";
    }
    cout << endl;

    return 0;
}
```
