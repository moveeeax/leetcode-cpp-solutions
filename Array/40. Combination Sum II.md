# **Combination Sum II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem asks us to find all unique combinations of a list of numbers that sum up to a specific target. Each number in the input list can only be used once in each combination. Given that there may be duplicates in the input list, we have to ensure that our combinations are unique. This adds complexity because we need to avoid creating duplicates during our backtracking process.

### **🚀 Approach**  
1. **Sorting**: Start by sorting the input list. This helps us to skip duplicates easily. For example, if we encounter the same number consecutively, we can choose to only use the first instance in that position and skip over the others.
  
2. **Backtracking**: Use a recursive backtracking function that explores all combinations:
   - If the current combination's sum equals the target, store it as a valid combination.
   - If the current sum exceeds the target, terminate that path.
   - Loop through the sorted list of candidates, adding each number to the current combination and recursively calling the function to explore further.
   - On returning from the recursion, remove the last added number to backtrack.

3. **Skip Duplicates**: While iterating through the candidates, if the current candidate is the same as the previous one, skip it to ensure that the same combination is not counted again.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(2^n), where n is the number of candidates. This accounts for all combinations being generated.
- **Space Complexity**: O(n), where n is the depth of the recursion stack due to storage of the current combination.

## **2️⃣ LeetCode Submission**  
```cpp
class CombinationSumII {
public:
    vector<vector<int>> combinationSum2(vector<int>& candidates, int target) {
        vector<vector<int>> result;
        vector<int> combination;
        sort(candidates.begin(), candidates.end());
        backtrack(candidates, target, 0, combination, result);
        return result;
    }

private:
    void backtrack(vector<int>& candidates, int target, int start, vector<int>& combination, vector<vector<int>>& result) {
        if (target == 0) {
            result.push_back(combination);
            return;
        }
        if (target < 0) {
            return;
        }
        for (int i = start; i < candidates.size(); i++) {
            if (i > start && candidates[i] == candidates[i - 1]) continue;  // Skip duplicates
            
            combination.push_back(candidates[i]);
            backtrack(candidates, target - candidates[i], i + 1, combination, result);  // Move to the next index
            combination.pop_back();  // Backtrack
        }
    }
};
```
