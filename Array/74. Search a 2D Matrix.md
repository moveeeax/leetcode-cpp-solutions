# **Search a 2D Matrix**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to determine whether a target value exists in a 2D matrix where each row is sorted in ascending order and the first integer of each row is greater than the last integer of the previous row. This property allows us to treat the 2D matrix as a flattened sorted array.

### **🚀 Approach**  
To efficiently find the target in this matrix, we can use binary search. By treating the matrix as a single-dimensional array, we can compute the indices for the rows and columns from the 1D index.

1. Calculate the total number of elements in the matrix as `m * n`, where `m` is the number of rows and `n` is the number of columns.
2. Initialize two pointers, `left` set to `0` and `right` set to `m * n - 1`, which represent the bounds of our binary search.
3. Perform a binary search:
   - Compute the middle index, `mid`.
   - Convert the `mid` index back to the row index and column index in the 2D matrix:
     - Row: `mid / n`
     - Column: `mid % n`
   - Compare the value at this (row, column) with the target:
     - If the value is equal to the target, return `true`.
     - If the value is less than the target, move the left pointer to `mid + 1`.
     - If the value is greater than the target, move the right pointer to `mid - 1`.
4. If the loop ends without finding the target, return `false`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(log(m * n)), since we are performing a binary search over the entire number of elements.
- **Space Complexity**: O(1), as we are using only a constant amount of extra space.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        if (matrix.empty() || matrix[0].empty()) return false;
        
        int m = matrix.size();
        int n = matrix[0].size();
        int left = 0, right = m * n - 1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            int mid_value = matrix[mid / n][mid % n];
            
            if (mid_value == target) {
                return true;
            } else if (mid_value < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        
        return false;
    }
};
```

        int m = matrix.size();
