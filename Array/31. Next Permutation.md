# **Next Permutation**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem is to find the next lexicographical permutation of an array of integers. The essence of the problem lies in rearranging the numbers to create the smallest possible next permutation. If no such permutation exists (for example, when the array is sorted in descending order), we need to return the lowest possible order (i.e., sorted in ascending order).

For example, given the array `[1, 2, 3]`, the next permutation would be `[1, 3, 2]`. If we have `[3, 2, 1]`, there is no higher permutation, and we should return `[1, 2, 3]`.

### **🚀 Approach**  
1. **Identify pivot**: Traverse the array from the end to find the largest index `i` such that `nums[i] < nums[i + 1]`. This identifies where the current permutation can be modified to create the next higher permutation.
  
2. **Find swap candidate**: If such an index `i` exists, find the largest index `j` greater than `i` such that `nums[j] > nums[i]`. This allows us to swap the ith value with a larger value that will give us the next permutation.

3. **Swap**: Swap the values at indices `i` and `j`.

4. **Reverse**: Finally, reverse the sequence from `i + 1` to the end of the array to ensure we have the smallest possible order after our change.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the length of the array. We traverse the array a few times, which is linear in terms of time.
- **Space Complexity**: O(1), as we are making modifications in place without using any additional data structures that scale with input size.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    void nextPermutation(vector<int>& nums) {
        int n = nums.size();
        int i = n - 2;

        // Step 1: Find the first decreasing element from the end
        while (i >= 0 && nums[i] >= nums[i + 1]) {
            i--;
        }
        
        // Step 2: If the whole array is non-increasing, reverse it
        if (i >= 0) {
            int j = n - 1;
            // Step 3: Find the number just larger than nums[i]
            while (j > i && nums[j] <= nums[i]) {
                j--;
            }
            // Swap the two numbers
            swap(nums[i], nums[j]);
        }
        // Step 4: Reverse the elements from i + 1 to end of the array
        reverse(nums.begin() + i + 1, nums.end());
    }
};  
```

#include <iostream>
#include <vector>
#include <algorithm>

class Solution {
public:
    void nextPermutation(std::vector<int>& nums) {
        int n = nums.size();
        int i = n - 2;

        while (i >= 0 && nums[i] >= nums[i + 1]) {
            i--;
        }
        
        if (i >= 0) {
            int j = n - 1;
            while (j > i && nums[j] <= nums[i]) {
                j--;
            }
            std::swap(nums[i], nums[j]);
        }
        std::reverse(nums.begin() + i + 1, nums.end());
    }
};

int main() {
    Solution solution;
    std::vector<int> nums = {1, 2, 3};
    solution.nextPermutation(nums);
    for(int num : nums) {
        std::cout << num << " ";
    }
    std::cout << std::endl;

    nums = {3, 2, 1};
    solution.nextPermutation(nums);
    for(int num : nums) {
        std::cout << num << " ";
    }
    std::cout << std::endl;

    return 0;
}
