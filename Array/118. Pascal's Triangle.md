# **LeetCode Pascal's Triangle**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem asks us to generate Pascal's Triangle, which is a triangular array of the binomial coefficients. Each number is the sum of the two numbers directly above it in the previous row. The first row is considered to be row 0 and contains just the number 1.  

For example, the first 5 rows of Pascal's Triangle are:
```
Row 0:     1
Row 1:    1 1
Row 2:   1 2 1
Row 3:  1 3 3 1
Row 4: 1 4 6 4 1
```

### **🚀 Approach**  
To solve this problem, we will iteratively build the rows of Pascal's Triangle. For every new row, we first append 1 (as every row starts with 1). Then, we calculate each new element by summing the two elements above it from the previous row. Finally, we append another 1 at the end for the completed row. 

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n^2), where n is the number of rows. This complexity arises because we have to generate each of the n rows and each row has an increasing number of elements to compute.  
- **Space Complexity**: O(n), where n is the number of rows. We need to store all rows in a list.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<int>> generate(int numRows) {
        vector<vector<int>> triangle(numRows);
        for (int i = 0; i < numRows; ++i) {
            triangle[i].resize(i + 1); // Resize for the current row
            triangle[i][0] = triangle[i][i] = 1; // The first and last elements are always 1

            for (int j = 1; j < i; ++j) {
                triangle[i][j] = triangle[i - 1][j - 1] + triangle[i - 1][j]; // Fill in the middle elements
            }
        }
        return triangle;
    }
};  
```

            triangle[i].resize(i + 1); // Resize for the current row

int main() {
    Solution sol;
    int numRows = 5; // Example input
    vector<vector<int>> result = sol.generate(numRows);
