# **Best Time to Buy and Sell Stock**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to determine the maximum profit from buying and selling a stock given the prices on different days. We are allowed to buy on one day and sell on a later day, but only once. The goal is to find the best day to buy that would result in the highest possible profit when selling afterward. 

### **🚀 Approach**  
To solve this problem, we will utilize a single-pass algorithm, maintaining two variables:
- `minPrice`: this represents the lowest price encountered so far as we iterate over the array of stock prices.
- `maxProfit`: this stores the maximum profit calculated as we consider the current day's price. 

The steps to implement the solution are:
1. Initialize `minPrice` to a very high value (we can use `INT_MAX`).
2. Initialize `maxProfit` to 0.
3. Iterate through each price in the list:
   - Update `minPrice` if the current price is lower than `minPrice`.
   - Calculate the profit by subtracting `minPrice` from the current price.
   - Update `maxProfit` if the calculated profit is greater than `maxProfit`.
4. After going through all the prices, `maxProfit` will hold the maximum profit possible.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) — We only make a single pass through the list of prices.
- **Space Complexity**: O(1) — We are using only a constant amount of space for `minPrice` and `maxProfit`.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int minPrice = INT_MAX;
        int maxProfit = 0;
        
        for (int price : prices) {
            if (price < minPrice) {
                minPrice = price; // Update the minimum price
            } else {
                maxProfit = max(maxProfit, price - minPrice); // Calculate profit and update maxProfit
            }
        }
        
        return maxProfit; // Return the maximum profit found
    }
};
```
