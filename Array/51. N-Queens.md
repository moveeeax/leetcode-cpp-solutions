# **N-Queens**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The N-Queens problem is a classic combinatorial problem where the task is to place N queens on an N × N chessboard such that no two queens threaten each other. A queen can move any number of squares along the row, column, or diagonal, thus the placement of each queen must be such that no two queens share the same row, column, or diagonal. 

### **🚀 Approach**  
To solve this problem, we'll use a backtracking algorithm, which is a recursive method for solving constraint satisfaction problems. We can represent the board as a set of positions where queens can be placed. The goal is to explore potential placements of queens, and to only proceed when it's valid to add a queen. If we reach the end of our recursive exploration with valid placements, we copy the state of the board as one of the solutions.

Key steps in the approach:
1. **Recursive backtracking**: For each row, we'll attempt to place a queen in each column and then recursively call the function to place queens in subsequent rows.
2. **Valid check**: For each column, and each combination of the row and diagonals, we will check if the placement is valid.
3. **Store solutions**: When a successful configuration of queens is found for all rows, we store this configuration as one of the solutions.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(N!), as we generate all possible arrangements of queens.
- **Space Complexity**: O(N), for storing the current placements of queens (in addition to recursive call stack).

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<string>> solveNQueens(int n) {
        vector<vector<string>> results;
        vector<string> board(n, string(n, '.'));
        vector<int> cols(n, 0), diag1(2 * n - 1, 0), diag2(2 * n - 1, 0);
        
        backtrack(results, board, 0, cols, diag1, diag2, n);
        return results;
    }

    void backtrack(vector<vector<string>>& results, vector<string>& board, int row,
                   vector<int>& cols, vector<int>& diag1, vector<int>& diag2, int n) {
        if (row == n) {
            results.push_back(board);
            return;
        }
        
        for (int col = 0; col < n; ++col) {
            if (cols[col] || diag1[row + col] || diag2[row - col + n - 1]) {
                continue; // If the column or the diagonals are already occupied
            }
            
            cols[col] = diag1[row + col] = diag2[row - col + n - 1] = 1; // Mark
            board[row][col] = 'Q'; // Place queen
            
            backtrack(results, board, row + 1, cols, diag1, diag2, n); // Recur
            
            cols[col] = diag1[row + col] = diag2[row - col + n - 1] = 0; // Unmark
            board[row][col] = '.'; // Remove queen
        }
    }
};
```  
