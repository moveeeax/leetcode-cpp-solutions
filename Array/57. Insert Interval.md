# **Insert Interval**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem is about inserting a new interval into a list of non-overlapping intervals while maintaining their order and ensuring that the resulting intervals still do not overlap. The key challenge lies in determining where the new interval fits in the sorted list and how it may merge with existing intervals if they overlap.  

To solve the problem, we need to traverse the list of existing intervals and handle three possibilities:
1. The new interval is completely before the current interval, which means we can simply add it to the result.
2. The new interval overlaps with the current interval, which implies we need to merge them.
3. The new interval is completely after the current one, in which case we continue to add the current interval to the result.

### **🚀 Approach**  
1. Create a result vector to store the merged intervals.
2. Iterate through the list of existing intervals using an index.
3. Check if the new interval should be added before the current interval or if it overlaps:
   - If the new interval ends before the current one starts, add the new interval to the result and continue.
   - If the new interval overlaps, merge it with the current interval by updating the end of the new interval.
4. If the new interval ends after the current interval starts, add the current interval to the result.
5. After processing all intervals, ensure to add the new interval if it has not yet been added.
6. Return the result containing all intervals.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of existing intervals, as we potentially examine each interval once.
- **Space Complexity**: O(n) in the worst case for the output list of merged intervals.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<vector<int>> insert(vector<vector<int>>& intervals, vector<int>& newInterval) {
        vector<vector<int>> result;
        int i = 0, n = intervals.size();
        
        // Add all intervals before the newInterval
        while (i < n && intervals[i][1] < newInterval[0]) {
            result.push_back(intervals[i]);
            i++;
        }
        
        // Merge the newInterval with overlapping intervals
        while (i < n && intervals[i][0] <= newInterval[1]) {
            newInterval[0] = min(newInterval[0], intervals[i][0]);
            newInterval[1] = max(newInterval[1], intervals[i][1]);
            i++;
        }
        
        result.push_back(newInterval); // Add the merged newInterval
        
        // Add all the remaining intervals
        while (i < n) {
            result.push_back(intervals[i]);
            i++;
        }
        
        return result;
    }
};  
```

#include <iostream>
#include <vector>
using namespace std;

class Solution {
public:
    vector<vector<int>> insert(vector<vector<int>>& intervals, vector<int>& newInterval) {
        vector<vector<int>> result;
        int i = 0, n = intervals.size();

        while (i < n && intervals[i][1] < newInterval[0]) {
            result.push_back(intervals[i]);
            i++;
        }

        while (i < n && intervals[i][0] <= newInterval[1]) {
            newInterval[0] = min(newInterval[0], intervals[i][0]);
            newInterval[1] = max(newInterval[1], intervals[i][1]);
            i++;
        }

        result.push_back(newInterval);
        
        while (i < n) {
            result.push_back(intervals[i]);
            i++;
        }

        return result;
    }
};

int main() {
    Solution sol;
    vector<vector<int>> intervals = {{1,3},{6,9}};
    vector<int> newInterval = {2,5};
    
    vector<vector<int>> merged = sol.insert(intervals, newInterval);
    
    for (const auto& interval : merged) {
        cout << "[" << interval[0] << "," << interval[1] << "] ";
    }
    cout << endl;

    return 0;
}
