# **Find First and Last Position of Element in Sorted Array**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to find the first and last positions of a specific target element within a sorted array. Given that the array is sorted, we can utilize binary search, which allows us to efficiently locate the desired indices instead of checking each element sequentially. 

### **🚀 Approach**  
1. **Binary Search**: Use binary search to find the first occurrence of the target. This involves adjusting the search range based on the current mid-point:
   - If the middle element is less than the target, adjust the left boundary to `mid + 1`.
   - If the middle element is greater than the target, adjust the right boundary to `mid - 1`.
   - If the middle element equals the target, record the index and continue searching towards the left side of the array to ensure it’s the first occurrence.

2. **Finding the Last Position**: Apply a similar binary search to find the last occurrence. When a match is found, move the search to the right side to check for further occurrences.

3. **Result Compilation**: If the target is found, return the indices of the first and last occurrence. If not found, return `[-1, -1]`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(log n) for each of the two binary search operations, resulting in O(log n) overall.
- **Space Complexity**: O(1), since we are using a constant amount of space.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<int> searchRange(vector<int>& nums, int target) {
        int left = searchLeft(nums, target);
        int right = searchRight(nums, target);
        return {left, right};
    }
    
    int searchLeft(vector<int>& nums, int target) {
        int low = 0, high = nums.size() - 1, result = -1;
        while (low <= high) {
            int mid = low + (high - low) / 2;
            if (nums[mid] == target) {
                result = mid; // Found target, record the position
                high = mid - 1; // Move left to find first occurrence
            } else if (nums[mid] < target) {
                low = mid + 1;
            } else {
                high = mid - 1;
            }
        }
        return result;
    }
    
    int searchRight(vector<int>& nums, int target) {
        int low = 0, high = nums.size() - 1, result = -1;
        while (low <= high) {
            int mid = low + (high - low) / 2;
            if (nums[mid] == target) {
                result = mid; // Found target, record the position
                low = mid + 1; // Move right to find last occurrence
            } else if (nums[mid] < target) {
                low = mid + 1;
            } else {
                high = mid - 1;
            }
        }
        return result;
    }
};
```
