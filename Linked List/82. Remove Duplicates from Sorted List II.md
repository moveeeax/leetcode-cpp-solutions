# **Remove Duplicates from Sorted List II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to remove all nodes from a sorted linked list that contain duplicates, leaving only distinct numbers behind. We are presented with a linked list where each element is in sorted order, and we need to modify the list such that all elements with duplicate values are removed entirely.  

To approach this problem, we need to traverse the list and keep track of any values that have duplicates. Once we identify the duplicates, we will adjust the pointers in our list to skip over these nodes.  

### **🚀 Approach**  
1. **Two Pointers Technique**: We'll use a dummy node to facilitate the removal process without edge-case handling for the head node. One pointer, `prev`, will point to the last distinct node we have seen, and another pointer, `cur`, will traverse through the linked list.
2. **Duplicate Detection**: As we traverse, we check if the current node has duplicates (i.e., if its value equals the next node's value). If duplicates are found, we continue to advance the `cur` pointer until we find a non-duplicate value.
3. **Linking**: Once we've found a distinct value, we link it to the previous distinct node (pointed by `prev`). If we encounter duplicates, we simply skip all nodes with that value.
4. **Return the Head**: Finally, we return the next node of the dummy as the modified head of the list.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the linked list. We only traverse the list once.
- **Space Complexity**: O(1), as we are using a constant amount of space regardless of the input size.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    ListNode* deleteDuplicates(ListNode* head) {
        ListNode dummy(0);
        dummy.next = head;
        ListNode* prev = &dummy;
        ListNode* cur = head;
        
        while (cur) {
            bool hasDuplicates = false;

            while (cur->next && cur->val == cur->next->val) {
                hasDuplicates = true;
                cur = cur->next;
            }

            if (hasDuplicates) {
                // Skip all duplicates
                prev->next = cur->next;
            } else {
                prev = prev->next;
            }
            
            cur = cur->next;
        }
        
        return dummy.next;
    }
};
```

#include <iostream>

struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(NULL) {}
};

class Solution {
public:
    ListNode* deleteDuplicates(ListNode* head) {
        ListNode dummy(0);
        dummy.next = head;
        ListNode* prev = &dummy;
        ListNode* cur = head;
        
        while (cur) {
            bool hasDuplicates = false;

            while (cur->next && cur->val == cur->next->val) {
                hasDuplicates = true;
                cur = cur->next;
            }

            if (hasDuplicates) {
                prev->next = cur->next;
            } else {
                prev = prev->next;
            }
            
            cur = cur->next;
        }
        
        return dummy.next;
    }
};

int main() {
    Solution solution;
    
    // Create linked list: 1 -> 2 -> 3 -> 3 -> 4 -> 4 -> 5
    ListNode *node1 = new ListNode(1);
    ListNode *node2 = new ListNode(2);
    ListNode *node3 = new ListNode(3);
    ListNode *node4 = new ListNode(3);
    ListNode *node5 = new ListNode(4);
    ListNode *node6 = new ListNode(4);
    ListNode *node7 = new ListNode(5);
    
    node1->next = node2;
    node2->next = node3;
    node3->next = node4;
    node4->next = node5;
    node5->next = node6;
    node6->next = node7;

    ListNode* result = solution.deleteDuplicates(node1);

    // Print the resulting list
    while (result) {
        std::cout << result->val << " ";
        result = result->next;
    }
    return 0;
}
