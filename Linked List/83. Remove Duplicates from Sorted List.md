# **Remove Duplicates from Sorted List**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to remove duplicates from a sorted linked list, ensuring that each value appears only once in the resulting list. This is a classic linked list manipulation problem where we want to maintain the integrity of the list while eliminating redundancy.  

### **🚀 Approach**  
To effectively remove the duplicates from the sorted linked list, we can use a single pass approach since the LinkedList is already sorted. We'll define a pointer to traverse the linked list, and for each node, we'll compare it with the next node. If the values are the same, we will skip the next node by adjusting the pointers. If they are different, we will just move forward in the list. This way, we ensure that we visit each node only once, maintaining O(n) time complexity.  

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of nodes in the linked list.  
- **Space Complexity**: O(1), since we are not using any additional data structures that grow with input size.

## **2️⃣ LeetCode Submission**

struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(nullptr) {}
};

class Solution {
public:
    ListNode* deleteDuplicates(ListNode* head) {
        ListNode* current = head;
        while (current != nullptr && current->next != nullptr) {
            if (current->val == current->next->val) {
                current->next = current->next->next;
            } else {
                current = current->next;
            }
        }
        return head;
    }
};
```

#include <iostream>

struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(nullptr) {}
};

class Solution {
public:
    ListNode* deleteDuplicates(ListNode* head) {
        ListNode* current = head;
        while (current != nullptr && current->next != nullptr) {
            if (current->val == current->next->val) {
                current->next = current->next->next;
            } else {
                current = current->next;
            }
        }
        return head;
    }
};

void printList(ListNode* node) {
    while (node != nullptr) {
        std::cout << node->val << " ";
        node = node->next;
    }
    std::cout << std::endl;
}

int main() {
    Solution solution;

    // Creating a sample linked list: 1 -> 1 -> 2 -> 3 -> 3
    ListNode* head = new ListNode(1);
    head->next = new ListNode(1);
    head->next->next = new ListNode(2);
    head->next->next->next = new ListNode(3);
    head->next->next->next->next = new ListNode(3);

    std::cout << "Original list: ";
    printList(head);

    head = solution.deleteDuplicates(head);

    std::cout << "List after removing duplicates: ";
    printList(head);

    // Clean up memory (delete nodes)
    while (head != nullptr) {
        ListNode* temp = head;
        head = head->next;
        delete temp;
    }

    return 0;
}  
```
