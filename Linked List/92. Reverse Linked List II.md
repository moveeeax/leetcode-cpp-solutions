# **Reverse Linked List II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The task is to reverse a part of a given singly linked list, specifically from position `m` to position `n`. The positions are 1-based indices, and our goal is to return the head of the modified list after reversing the specified section. This problem comes down to manipulating pointers in the linked list to achieve the desired reverse effect.

### **🚀 Approach**  
To solve this problem, we can break it down into several key steps:

1. **Traverse to the starting point**: We need to locate the node from which we will start reversing (i.e., position `m`), as well as maintain a pointer to the node just before it.

2. **Reverse the sublist**: Once we reach the `m`-th node, we'll reverse the nodes from this point to the `n`-th node. This involves manipulating the `next` pointers of the nodes in this range.

3. **Reconnect the reversed sublist**: After the sublist is reversed, we need to correctly link the head of the reversed section with the part before `m` and the part after `n`.

4. **Return the modified list head**: If `m` is 1, the new head will be the `n`-th node after reversing; otherwise, the head remains unchanged.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the length of the linked list, because we traverse the list at most twice.
- **Space Complexity**: O(1), since we only use a constant amount of additional space.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    ListNode* reverseBetween(ListNode* head, int m, int n) {
        if (!head || m == n) return head;

        ListNode dummy(0);
        dummy.next = head;
        ListNode* pre = &dummy;

        // Step 1: Reach the node just before the position m
        for (int i = 1; i < m; ++i)
            pre = pre->next;

        // Step 2: Reverse the sublist
        ListNode* curr = pre->next; // Starting node to be reversed
        ListNode* next = nullptr;
        for (int i = 0; i < n - m; ++i) {
            next = curr->next; // Remember the next node
            curr->next = next->next; // Reverse the link
            next->next = pre->next; // Move `next` to the front of the sublist
            pre->next = next; // Connect with the part before m
        }

        return dummy.next; // return the new head of the list
    }
};
```

```cpp
#include <iostream>

struct ListNode {
    int val;
    ListNode* next;
    ListNode(int x) : val(x), next(nullptr) {}
};

class Solution {
public:
    ListNode* reverseBetween(ListNode* head, int m, int n) {
        if (!head || m == n) return head;

        ListNode dummy(0);
        dummy.next = head;
        ListNode* pre = &dummy;

        for (int i = 1; i < m; ++i)
            pre = pre->next;

        ListNode* curr = pre->next;
        ListNode* next = nullptr;
        for (int i = 0; i < n - m; ++i) {
            next = curr->next;
            curr->next = next->next;
            next->next = pre->next;
            pre->next = next;
        }

        return dummy.next;
    }
};

int main() {
    Solution solution;
    
    // Create a linked list: 1 -> 2 -> 3 -> 4 -> 5
    ListNode* head = new ListNode(1);
    head->next = new ListNode(2);
    head->next->next = new ListNode(3);
    head->next->next->next = new ListNode(4);
    head->next->next->next->next = new ListNode(5);

    // Reverse from position 2 to 4
    head = solution.reverseBetween(head, 2, 4);

    // Output the modified list
    ListNode* current = head;
    while (current) {
        std::cout << current->val << " ";
        current = current->next;
    }
    std::cout << std::endl;

    // Clean up the linked list
    current = head;
    while (current) {
        ListNode* temp = current;
        current = current->next;
        delete temp;
    }

    return 0;
}
```
