# **Text Justification**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem is about justifying text in such a way that each line has a maximum width specified by `maxWidth`. We have to distribute the words within this width, fulfilling specific formatting conditions (like aligning text to both edges of the line). This involves ensuring that gaps between words are well-distributed so that the lines look visually appealing, especially when compared to a left-aligned text. Our goal is to take a list of words and format them according to the given constraints.

### **🚀 Approach**  
1. **Initialization**: Start by initializing an empty list that will hold the results and variables to track the current line of words and their total length.
2. **Word Addition**: Loop through the words, checking if adding the next word (plus an optional space if it's not the first word) would exceed `maxWidth`. If yes, justify the current line and reset the current line accumulator for the next set of words.
3. **Justifying Text**:
   - If there is only one word in the line, it stays left-aligned.
   - For multiple words, calculate the total number of spaces needed to fill the line to `maxWidth`.
   - Evenly distribute spaces between the words. If there's any leftover space, distribute it from left to right.
4. **Finalization**: After looping through all words, handle the last line. It should be left-justified and filled with spaces on the right.
5. **Output**: Return the list of justified lines.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the total number of characters in all the words combined since we touch each word and its characters at least once.
- **Space Complexity**: O(m), where m is the number of lines produced, as we store each line in the result array.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<string> fullJustify(vector<string>& words, int maxWidth) {
        vector<string> result;
        vector<string> currentLine;
        int numLetters = 0;

        for (const auto& word : words) {
            if (numLetters + word.size() + currentLine.size() > maxWidth) {
                for (int i = 0; i < maxWidth - numLetters; ++i) {
                    currentLine[i % (currentLine.size() - 1 ?: 1)] += " ";
                }
                result.push_back(join(currentLine));
                currentLine.clear();
                numLetters = 0;
            }
            currentLine.push_back(word);
            numLetters += word.size();
        }

        // Handle the last line
        string lastLine = join(currentLine, ' '); 
        lastLine.append(maxWidth - lastLine.size(), ' ');
        result.push_back(lastLine);

        return result;
    }

private:
    string join(const vector<string>& words, char delim = ' ') {
        string result;
        for (const auto& word : words) {
            if (!result.empty())
                result += delim;
            result += word;
        }
        return result;
    }
};
```  

```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

class Solution {
public:
    vector<string> fullJustify(vector<string>& words, int maxWidth) {
        vector<string> result;
        vector<string> currentLine;
        int numLetters = 0;

        for (const auto& word : words) {
            if (numLetters + word.size() + currentLine.size() > maxWidth) {
                for (int i = 0; i < maxWidth - numLetters; ++i) {
                    currentLine[i % (currentLine.size() - 1 ?: 1)] += " ";
                }
                result.push_back(join(currentLine));
                currentLine.clear();
                numLetters = 0;
            }
            currentLine.push_back(word);
            numLetters += word.size();
        }

        // Handle the last line
        string lastLine = join(currentLine, ' '); 
        lastLine.append(maxWidth - lastLine.size(), ' ');
        result.push_back(lastLine);

        return result;
    }

private:
    string join(const vector<string>& words, char delim = ' ') {
        string result;
        for (const auto& word : words) {
            if (!result.empty())
                result += delim;
            result += word;
        }
        return result;
    }
};

int main() {
    Solution solution;
    vector<string> words = {"This", "is", "an", "example", "of", "text", "justification."};
```  
