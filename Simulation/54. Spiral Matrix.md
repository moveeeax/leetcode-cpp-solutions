# **LeetCode Spiral Matrix**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem at hand is to traverse a 2D matrix in a spiral order. This means starting from the top-left element and moving to the right across the first row, then down the last column, then moving to the left across the last row, and finally going up the first column. This process continues inward until every element in the matrix has been visited.

### **🚀 Approach**  
To achieve spiral traversal, we will define boundaries for our movement: `top`, `bottom`, `left`, and `right`. Initially, `top` begins at 0, `bottom` at the last row index, `left` at 0, and `right` at the last column index. The procedure can be broken down as follows:

1. Traverse from left to right along the `top` boundary, then increase the `top` boundary downwards.
2. Traverse from top to bottom along the `right` boundary, then decrease the `right` boundary inwards.
3. If there are still rows left, traverse from right to left along the `bottom` boundary, then decrease the `bottom` boundary upwards.
4. If there are still columns left, traverse from bottom to top along the `left` boundary, then increase the `left` boundary rightwards.
5. Repeat steps 1 to 4 until all boundaries overlap.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(m * n), where m is the number of rows and n is the number of columns in the matrix. Each element is processed once.
- **Space Complexity**: O(1) if we consider the output space as external, and we are using only a constant amount of space for the boundaries and the result.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        vector<int> result;
        if (matrix.empty()) return result;

        int top = 0, bottom = matrix.size() - 1;
        int left = 0, right = matrix[0].size() - 1;

        while (top <= bottom && left <= right) {
            // Traverse from left to right along the top row
            for (int i = left; i <= right; i++) {
                result.push_back(matrix[top][i]);
            }
            top++;

            // Traverse from top to bottom along the right column
            for (int i = top; i <= bottom; i++) {
                result.push_back(matrix[i][right]);
            }
            right--;

            if (top <= bottom) {
                // Traverse from right to left along the bottom row
                for (int i = right; i >= left; i--) {
                    result.push_back(matrix[bottom][i]);
                }
                bottom--;
            }

            if (left <= right) {
                // Traverse from bottom to top along the left column
                for (int i = bottom; i >= top; i--) {
                    result.push_back(matrix[i][left]);
                }
                left++;
            }
        }

        return result;
    }
};
```  

```cpp
#include <iostream>
#include <vector>

using namespace std;

class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        vector<int> result;
        if (matrix.empty()) return result;

        int top = 0, bottom = matrix.size() - 1;
        int left = 0, right = matrix[0].size() - 1;

        while (top <= bottom && left <= right) {
            for (int i = left; i <= right; i++) {
                result.push_back(matrix[top][i]);
            }
            top++;

            for (int i = top; i <= bottom; i++) {
                result.push_back(matrix[i][right]);
            }
            right--;

            if (top <= bottom) {
                for (int i = right; i >= left; i--) {
                    result.push_back(matrix[bottom][i]);
                }
                bottom--;
            }

            if (left <= right) {
                for (int i = bottom; i >= top; i--) {
                    result.push_back(matrix[i][left]);
                }
                left++;
            }
        }

        return result;
    }
};

int main() {
    Solution solution;
    vector<vector<int>> matrix = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };
    vector<int> result = solution.spiralOrder(matrix);
    
    for(int num : result) {

    }

}
```  
