# **Pascal's Triangle II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to return a specific row from Pascal's Triangle, based on a given index. Pascal's Triangle is structured such that each element is the sum of the two elements directly above it. Each row corresponds to the binomial coefficients, which can also be calculated using the formula C(n, k) = n! / (k! * (n - k)!). For example, given index `3`, we want to return the fourth row: `[1, 3, 3, 1]`.

### **🚀 Approach**  
To determine the specific row in Pascal's Triangle, we can utilize an efficient approach that avoids constructing the entire triangle. Rather than computing the factorial for each element, we can build the row iteratively using the relationship between the coefficients.

1. **Initialization**: Start with the first element of the row as `1`.
2. **Iterate**: For each subsequent element in the row, use the previous element multiplied by the ratio of `(n - k + 1) / k`, where `n` is the row index and `k` is the current index of the element being computed.
3. **Result**: Continue until we've built the entire row and return it.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n) - We iterate through the row for `n` elements.
- **Space Complexity**: O(n) - We store `n` values for the row output.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    vector<int> getRow(int rowIndex) {
        vector<int> row(rowIndex + 1, 1);
        for (int k = 1; k < rowIndex; ++k) {
            row[k] = (long)row[k - 1] * (rowIndex - k + 1) / k;
        }
        return row;
    }
};
```
