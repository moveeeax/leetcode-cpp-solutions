# **LeetCode Maximum Subarray**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem is about finding the contiguous subarray within a one-dimensional array of numbers (which may be positive, negative, or zero) that has the largest sum. A straightforward way to think about this is that we want to find the maximum sum of consecutive elements in the array, and the subarray has to be at least one element long.

### **🚀 Approach**  
The most efficient way to solve this problem is by using Kadane's Algorithm, which allows us to traverse the array in a single pass while maintaining the maximum sum of the contiguous subarray found so far. 

Here's the step-by-step breakdown of the approach:
1. Initialize two variables: `max_sum` to track the largest sum found and `current_sum` to track the sum of the current subarray.
2. Iterate through each element in the array:
   - For each element, add it to `current_sum`.
   - If `current_sum` exceeds `max_sum`, update `max_sum`.
   - If `current_sum` drops below zero, reset it to zero since a negative sum can't contribute positively to subsequent subarrays.
3. After finishing the loop, `max_sum` will hold the maximum sum of any contiguous subarray.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of elements in the array—each element is visited once.  
- **Space Complexity**: O(1), as we are using a constant amount of space.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int max_sum = nums[0];
        int current_sum = nums[0];
        
        for (int i = 1; i < nums.size(); i++) {
            current_sum = max(nums[i], current_sum + nums[i]);
            max_sum = max(max_sum, current_sum);
        }
        
        return max_sum;
    }
};
```

        int current_sum = nums[0];
        
        for (int i = 1; i < nums.size(); i++) {
            current_sum = max(nums[i], current_sum + nums[i]);
