# **LeetCode Interleaving String**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem asks whether one string (`s3`) is formed by interleaving two other strings (`s1` and `s2`). Interleaving means that the characters from `s1` and `s2` can be combined in such a way that preserves the order of characters in each original string. For example, if `s1 = "aabcc"` and `s2 = "dbbca"`, the string `s3 = "aadbbcbcac"` is a valid interleaving because you can take characters from both `s1` and `s2` while maintaining their respective orders.

### **🚀 Approach**  
The optimal solution to this problem can be achieved using dynamic programming. We will create a 2D boolean DP table, where `dp[i][j]` will be `true` if the substring formed by the first `i` characters of `s1` and the first `j` characters of `s2` can form the substring of length `i+j` in `s3`.  

Here is how to approach the problem:

1. **Table Initialization**: Create a 2D array `dp` of size `(len(s1)+1) x (len(s2)+1)`, initialized to `false`. The dimensions account for the option of taking no characters from either string.
   
2. **Base Case**: Set `dp[0][0]` to `true` because two empty strings can form an empty string.

3. **Filling the Table**:
   - Iterate through each character in `s1` and `s2`.
   - For each pair of indices `(i, j)`, check:
     - If the character in `s3` matches the current character in `s1`, update `dp[i][j]` based on `dp[i-1][j]`.
     - If the character in `s3` matches the current character in `s2`, update `dp[i][j]` based on `dp[i][j-1]`.

4. **Final Check**: The answer will be found in `dp[len(s1)][len(s2)]`, which tells whether the full `s3` can be formed by interleaving the full `s1` and `s2`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(m * n), where m and n are the lengths of `s1` and `s2`, respectively. This is due to traversing the `dp` table.
- **Space Complexity**: O(m * n), as we need a 2D table to store intermediate results.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool isInterleave(string s1, string s2, string s3) {
        int m = s1.size(), n = s2.size();
        if (m + n != s3.size()) return false;

        vector<vector<bool>> dp(m + 1, vector<bool>(n + 1, false));
        dp[0][0] = true;

        for (int i = 0; i <= m; ++i) {
            for (int j = 0; j <= n; ++j) {
                if (i > 0 && s1[i - 1] == s3[i + j - 1]) {
                    dp[i][j] = dp[i][j] || dp[i - 1][j];
                }
                if (j > 0 && s2[j - 1] == s3[i + j - 1]) {
                    dp[i][j] = dp[i][j] || dp[i][j - 1];
                }
            }
        }

        return dp[m][n];
    }
};  
```
