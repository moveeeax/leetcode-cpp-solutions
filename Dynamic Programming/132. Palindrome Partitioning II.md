# **Palindrome Partitioning II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem is about finding the minimum number of cuts needed to partition a string such that every substring is a palindrome. A palindrome is a string that reads the same forwards and backwards. The approach here requires understanding these properties and efficiently determining the minimum cuts. 

The solution involves:
1. Checking if substrings of the input string are palindromes.
2. Using dynamic programming to store results of the substring checks and cuts needed to find the optimal result.

### **🚀 Approach**  
1. **Palindrome Check**: First, we need to identify all palindromic substrings of the input string. This can be efficiently done using a 2D boolean array where `isPalindrome[i][j]` is true if the substring from index `i` to `j` is a palindrome.
  
2. **Dynamic Programming to Calculate Cuts**: Create a `dp` array where `dp[i]` represents the minimum cuts needed for the substring that ends at index `i`. 
   - If the substring from the start to `i` is a palindrome, no cuts are needed (`dp[i] = 0`).
   - Otherwise, iterate through all positions before `i` to find the number of cuts required based on previously computed palindromic substrings.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n^2), where n is the length of the string, due to the nested loops for palindrome checking and calculating cuts.  
- **Space Complexity**: O(n^2) for the `isPalindrome` array. The `dp` array takes O(n) space.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int minCut(string s) {
        int n = s.size();
        vector<vector<bool>> isPalindrome(n, vector<bool>(n, false));
        vector<int> dp(n);
        
        for (int i = 0; i < n; ++i) {
            int minCuts = i; // Max cuts is i, as we could cut every character
            for (int j = 0; j <= i; ++j) {
                if (s[j] == s[i] && (i - j < 2 || isPalindrome[j + 1][i - 1])) {
                    isPalindrome[j][i] = true; // Found a palindrome from j to i
                    minCuts = j == 0 ? 0 : min(minCuts, dp[j - 1] + 1); // No cut if we include this palindrome
                }
            }
            dp[i] = minCuts;
        }
        return dp[n - 1];
    }
};  
```
