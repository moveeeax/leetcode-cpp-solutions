# **LeetCode Edit Distance**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem is to find the minimum number of operations required to convert one string (word1) into another string (word2). The operations allowed are:
1. Insert a character.
2. Delete a character.
3. Replace a character.

This is a classic dynamic programming problem where we will use a matrix to keep track of the minimum edit distance at each subproblem. The goal is to fill out this matrix based on the given operations.

### **🚀 Approach**  
1. **Dynamic Programming Table Initialization**: We create a 2D vector `dp` where `dp[i][j]` holds the minimum edit distance between the first `i` characters of `word1` and the first `j` characters of `word2`.
2. **Base Cases**:
    - If `word1` is empty, the only option is to insert all characters of `word2` (i.e., `dp[0][j] = j`).
    - If `word2` is empty, the only option is to delete all characters of `word1` (i.e., `dp[i][0] = i`).
3. **Filling the DP Table**: 
    - For each character position in both strings, if the characters are the same, `dp[i][j] = dp[i-1][j-1]`.
    - If the characters are different, we take the minimum of the operations (insert, delete, replace) plus one:
      - Insert: `dp[i][j-1] + 1`
      - Delete: `dp[i-1][j] + 1`
      - Replace: `dp[i-1][j-1] + 1`
4. **Result**: The value at `dp[m][n]` will give us the minimum edit distance where `m` and `n` are the lengths of `word1` and `word2`, respectively.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(m * n), where m is the length of `word1` and n is the length of `word2`.  
- **Space Complexity**: O(m * n), for the DP table.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int minDistance(string word1, string word2) {
        int m = word1.size(), n = word2.size();
        vector<vector<int>> dp(m + 1, vector<int>(n + 1));

        for (int i = 0; i <= m; ++i) {
            for (int j = 0; j <= n; ++j) {
                if (i == 0) {
                    dp[i][j] = j; // Insertions
                } else if (j == 0) {
                    dp[i][j] = i; // Deletions
                } else if (word1[i - 1] == word2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1]; // No operation needed
                } else {
                    dp[i][j] = min({dp[i - 1][j] + 1, // Deletion
                                    dp[i][j - 1] + 1, // Insertion
                                    dp[i - 1][j - 1] + 1}); // Replacement
                }
            }
        }
        return dp[m][n];
    }
};  
```
