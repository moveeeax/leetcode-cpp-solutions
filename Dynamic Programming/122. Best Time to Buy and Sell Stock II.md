# **Best Time to Buy and Sell Stock II**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
In this problem, we want to maximize our profit from buying and selling stock given a series of prices for each day. Unlike a typical stock problem where we can only buy and sell once, in this case, we are allowed to buy and sell multiple times. Therefore, our goal is to accumulate profit from all the upward trends in the stock prices. If the price on the next day is greater than the current day, we can take advantage of that difference by "buying" on the current day and "selling" on the next day.  

### **🚀 Approach**  
To solve the problem, we can follow a simple iterative approach:  
1. **Iterate through the list of prices**: We'll check if the price tomorrow is greater than today.  
2. **Accumulate profits**: If it is, we can add the difference (tomorrow's price - today's price) to our total profit.  
3. **Continue this till the end of the list**: This way, we'll capture all profitable transactions in a single pass through the prices list.  

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n), where n is the number of days (length of the prices array). We make a single pass through the array to compute the profit.  
- **Space Complexity**: O(1), as we only need a constant amount of additional space for variables to store the profit and indices.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int profit = 0;
        for (int i = 1; i < prices.size(); ++i) {
            if (prices[i] > prices[i - 1]) {
                profit += prices[i] - prices[i - 1];
            }
        }
        return profit;
    }
};
```
