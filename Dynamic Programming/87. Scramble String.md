# **LeetCode Scramble String**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to determine whether one string can be transformed into another string through a series of scrambles. A string is a scramble of another string if it can be formed by recursively splitting the string into two non-empty substrings, swapping those substrings, and continuing to do this on each of the resulting substrings.  

For example, consider the strings "great" and "rgeat". We can split "great" into "gr" and "eat", swap those to get "eatgr". Then we can split "eat" into "e" and "at", and again swap these to yield "rgeat".

The challenge is to verify whether two given strings are scrambles of each other, taking into account all valid recursive splits and swaps.

### **🚀 Approach**  
1. **Base Cases**:  
   - If the two strings are equal, return `true`.
   - If their lengths differ, return `false`.
   - If they have different character counts, return `false`.

2. **Recursive Checking**:  
   - For each possible split point `i`, check if:
     - The left part of `s1` is a scramble of the left part of `s2` AND 
     - The right part of `s1` is a scramble of the right part of `s2`
   - OR, if:
     - The left part of `s1` is a scramble of the right part of `s2` AND 
     - The right part of `s1` is a scramble of the left part of `s2`  
   - This means that we need to check both possible interleavings at each level of recursion.

3. **Memoization**:  
   - To optimize the solution and reduce the computational time, a cache should be maintained to store results of previously computed substrings.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(n^4), where n is the length of the strings. Due to the recursive nature and having substring checks.
- **Space Complexity**: O(n^3) for the recursive call stack and memoization storage.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool isScramble(string s1, string s2) {
        if (s1 == s2) return true;
        if (s1.length() != s2.length()) return false;

        int n = s1.length();
        vector<vector<vector<int>>> memo(n, vector<vector<int>>(n, vector<int>(n + 1, -1)));
        return scrambleCheck(s1, s2, 0, 0, n, memo);
    }
    
    bool scrambleCheck(string &s1, string &s2, int i, int j, int len, vector<vector<vector<int>>> &memo) {
        if (memo[i][j][len] != -1) return memo[i][j][len];
        
        string s1_sub = s1.substr(i, len);
        string s2_sub = s2.substr(j, len);
        
        if (s1_sub == s2_sub) {
            memo[i][j][len] = true;
            return true;
        }
        
        if (sorted(s1_sub) != sorted(s2_sub)) {
            memo[i][j][len] = false;
            return false;
        }
        
        for (int k = 1; k < len; ++k) {
            if ((scrambleCheck(s1, s2, i, j, k, memo) && scrambleCheck(s1, s2, i + k, j + k, len - k, memo)) ||
                (scrambleCheck(s1, s2, i, j + len - k, k, memo) && scrambleCheck(s1, s2, i + k, j, len - k, memo))) {
                memo[i][j][len] = true;
                return true;
            }
        }
        
        memo[i][j][len] = false;
        return false;
    }
    
    string sorted(string str) {
        sort(str.begin(), str.end());
        return str;
    }
};
```
