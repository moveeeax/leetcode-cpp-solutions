# **LeetCode Wildcard Matching**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to determine if a given string `s` can be matched by a pattern `p` containing wildcard characters. The character `'*'` can represent any sequence of characters (including an empty sequence), while the character `'?'` can match any single character. To solve this, we can leverage dynamic programming to track the matching state between the string and the pattern.

### **🚀 Approach**  
1. **Dynamic Programming Table Initialization**: We'll use a 2D boolean array `dp` where `dp[i][j]` will be true if the first `i` characters of string `s` can be matched with the first `j` characters of the pattern `p`.
2. **Base Case**: Start with `dp[0][0] = true` since an empty string matches an empty pattern. If `p` starts with multiple `'*'`, prepare the `dp` table for those.
3. **Filling the DP Table**:
   - If the current characters of `s` and `p` match or if `p` has `'?'`, set `dp[i][j]` based on `dp[i-1][j-1]`.
   - If `p[j-1]` is `'*'`, it can either match an empty string (use the value from `dp[i][j-1]`) or expand to include the current character from `s` (use the value from `dp[i-1][j]`).
4. **Final Result**: The value at `dp[s.size()][p.size()]` will provide the final answer to whether `s` matches `p`.

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(m * n), where m is the length of the string and n is the length of the pattern.  
- **Space Complexity**: O(m * n) for the DP table.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    bool isMatch(string s, string p) {
        int m = s.size();
        int n = p.size();
        vector<vector<bool>> dp(m + 1, vector<bool>(n + 1, false));
        dp[0][0] = true;

        // Fill the first row for patterns starting with '*'
        for (int j = 1; j <= n; j++) {
            if (p[j - 1] == '*') 
                dp[0][j] = dp[0][j - 1];
        }

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (p[j - 1] == s[i - 1] || p[j - 1] == '?') {
                    dp[i][j] = dp[i - 1][j - 1];
                } else if (p[j - 1] == '*') {
                    dp[i][j] = dp[i][j - 1] || dp[i - 1][j];
                }
            }
        }

        return dp[m][n];
    }
};
```
