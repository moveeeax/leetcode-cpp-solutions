# **Search Insert Position**  

## **1️⃣ Problem Understanding**  
### **📌 Intuition**  
The problem requires us to find the position at which a target value should be inserted into a sorted array. Given that the array is sorted, we can efficiently determine the position using binary search rather than a linear search, achieving better performance. The end goal is to return the index at which the target would be located if it were inserted in order.

### **🚀 Approach**  
1. **Binary Search**: Utilize the binary search technique to halve the search space with each iteration. Initialize two pointers, `left` and `right`, to represent the lower and upper bounds of our search space, respectively.
2. **Finding the Target**: In each iteration, calculate the middle index. If the middle element is equal to the target, return that index. If the target is less than the middle element, adjust the `right` pointer to `mid - 1`; otherwise, adjust the `left` pointer to `mid + 1`.
3. **Insert Position**: If the target is not found in the array, the `left` pointer will eventually point to the index where it can be inserted while maintaining order. 

### **⏱️ Complexity Analysis**  
- **Time Complexity**: O(log n) due to the binary search method reducing the search space by half each time.
- **Space Complexity**: O(1) since we are using a constant amount of additional space for pointers.

## **2️⃣ LeetCode Submission**  
```cpp
class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int left = 0, right = nums.size() - 1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] == target) {
                return mid;
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        
        return left; // Insert position
    }
};
```

```cpp
#include <iostream>
#include <vector>
using namespace std;

class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int left = 0, right = nums.size() - 1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] == target) {
                return mid;
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        
        return left; // Insert position
    }
};

int main() {
    Solution solution;
    vector<int> nums = {1, 3, 5, 6};
    int target = 5;
    cout << solution.searchInsert(nums, target) << endl; // Output: 2
    
    target = 2;
    cout << solution.searchInsert(nums, target) << endl; // Output: 1
    
    target = 7;
    cout << solution.searchInsert(nums, target) << endl; // Output: 4
    
    target = 0;
    cout << solution.searchInsert(nums, target) << endl; // Output: 0

    return 0;
}
```
